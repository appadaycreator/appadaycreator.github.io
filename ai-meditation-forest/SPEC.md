# 仕様書

## v2.3.0（2026-06-13）
- **【M8】 ツールチップ・入力ガイダンス充実**: 瞑想設定画面の各入力欄（気分選択、時間選択、目的選択）に説明テキストと?アイコン+ツールチップを追加
  - 気分選択: 各気分の選択肢に title 属性を追加し、tooltip で説明
  - 時間選択: 初心者向け・推奨・継続者向けの説明テキストを表示
  - 目的選択: 各目的の詳細説明をツールチップで提供
- **【M10】 印刷・結果エクスポート機能**: セッション完了画面に 3つの新機能を実装
  - 📋 コピー: セッション結果をテキスト形式でクリップボードにコピー
  - 📥 CSV保存: 瞑想履歴を CSV ファイルとしてダウンロード（日時、気分、時間、目的、ポイント、完了後の気分）
  - 🖨 印刷: window.print()で印刷機能を呼び出し、結果を紙に出力可能

## v2.2.0（2026-06-10）
- **【M2】 input要素UX改善**: ダミーform要素（name="amfn"、"amfv"、"amfs"）に placeholder 属性とaria-label を追加し、アクセシビリティと入力補助を強化
- **【M4】 統計グラフ可視化**: Chart.js 3.9.1 を導入し、履歴画面に瞑想データの統計グラフを実装
  - 統計サマリー: 総セッション数、総ポイント、平均瞑想時間をカード表示
  - 過去7日間のセッション数（棒グラフ）
  - 気分別セッション分布（ドーナツグラフ）
  - ポイント推移（折れ線グラフ）
  - レスポンシブ対応（PC: 2列、モバイル: 1列表示）

## v2.1.0（2026-05-31）
- FontAwesome CDN（75KB）を削除し、全アイコンを絵文字に移行（パフォーマンス改善）
- ポイント計算の変数名タイポ（basePojnts → basePoints）を修正
- 時間選択ボタンを小画面（420px以下）でパディング縮小・収まるよう改善
- Escapeキーでサイドバーが閉じる動作を追加
- メニューボタンに aria-expanded / aria-controls 追加（アクセシビリティ向上）

- Tailwind CSS と Font Awesome を利用したシングルページアプリ
- Supabase を使ったデータ保存に対応（デモではダミーキーを設定）
- ログインボタンは廃止し、必要に応じて将来再導入予定
- PWA 対応でインストール可能
- 画面遷移は JavaScript で制御
- 共通の固定ヘッダーを実装し、モバイルではメニューボタンから各ページにアクセス可能
- 使い方、免責事項、プライバシーポリシー、FAQ の静的ページを追加し、文字色をより濃い灰色(text-gray-400)に調整
- LPページと利用規約ページを新設
- PCではヘッダーに「使い方」リンクのみを表示し、全てのメニューをフッターに配置
- サイドバーはモバイル専用で、右側に固定。透過しないデザインで最前面に表示され、外側をクリックすると閉じる
- 瞑想時間は1・5・10・15・20分から選択可能
- 履歴保存処理を見直し、ローカルストレージへの保存を確実にしました
- 音声ガイドのフェーズ切り替え処理を修正し、メッセージが1度のみ再生されるよう改善
- 言語切替機能を改善し、履歴などの表示も選択言語に合わせて翻訳
- 選択した言語に合わせて音声ガイドの読み上げ言語を自動切替
- モバイル向けにハンバーガーメニューのタッチ操作を追加
- セッション画面に選択した気分・目的・時間を表示
- 全ページに言語切替ボタンを配置し、PCではヘッダー、モバイルではサイドバーから切り替え可能
- 瞑想中の言語切り替え処理を改善し、音声とテキストの不整合を解消
- 瞑想セッション中は言語設定の変更を無効化し、一貫した体験を提供
- 瞑想再開機能を追加し、前回と同じ設定（気分・時間・目的）で瞑想を再開可能
- 前回のセッション設定をローカルストレージに自動保存し、ページリロード後も維持
