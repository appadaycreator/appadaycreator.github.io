# 子供のおこづかい管理アプリ【無料】記録・管理アプリ - 技術仕様書

## 概要

**サービス名**: Allowance Manager
**バージョン**: 2.5.0
**更新日**: 2026-06-11（自動改善 P1・P2 実装）
**URL**: https://appadaycreator.com/allowance-manager/

お小遣いの収支を記録してお金の使い方を学習。グラフで可視化。登録不要・完全無料でご利用いただけます。

## 主な機能

### v2.5.0 改善（2026-06-11追加・複数ユーザー・バックアップ・分析機能）
- **複数ユーザー管理機能（P1）**: 兄弟姉妹など複数の子どもの記録を個別に管理
  - ヘッダーのプロフィール選択で簡単に切り替え
  - 新しい記録を作成して追加可能
  - 各プロフィール独立のエントリ・目標設定・グラフを保有
- **データバックアップ・リストア機能（P1）**: JSON形式でのデータ保存・復元
  - 「💾 バックアップ」ボタン → JSON ファイルをダウンロード
  - 「📥 復元」ボタン → JSON ファイルを選択して復元
  - ブラウザキャッシュ削除時のデータロスト対策
- **データ分析・洞察メッセージの充実化（P2）**: より個別化された診断メッセージ
  - 記録件数に応じた段階的なアドバイス
  - 支出パターン認識（「支出が収入の◯%」など詳細化）
  - 親向け注意警告（支出 > 収入×1.2 の場合）
- **動的アドバイスの拡充（P2）**: ユーザー状態に基づいた個別メッセージ
  - 記録初期段階：「まず2週間記録しよう」
  - 支出多い時期：「使い方を相談しよう」
  - 目標達成中：「このペースを保とう」
- **テキスト最適化（P2）**: 重複セクション削除でページサイズ最適化
  - 「子どものお金教育」セクション（重複）を削除
  - ページサイズ約15% 削減

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 使い方

1. 記録したい項目を入力フォームに入力する
2. 「追加」または「記録」ボタンをクリックする
3. 記録一覧で過去のデータを確認する
4. グラフや集計で傾向を把握する
5. 継続的に記録して変化を追跡する

## よくある質問（FAQ）

**Q: 子供のおこづかい管理アプリは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。
