# 腰痛・肩こり原因チェッカー【無料】無料診断・アドバイス - 技術仕様書

## 概要

**サービス名**: Back Pain Checker
**バージョン**: 1.4.0
**更新日**: 2026-06-14
**URL**: https://appadaycreator.com/back-pain-checker/

生活習慣・仕事スタイル・姿勢から腰痛・肩こりの原因タイプを診断し、改善エクササイズと受診目安を提案。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: localStorage（診断履歴を最新10件保存）
- **永続化**: ブラウザのlocalStorage に保存（同一デバイス・同一ブラウザのみ）
- **プライバシー**: 入力データのサーバー送信なし・外部API通信なし

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、診断の説明を確認する
2. スタートボタンをクリックして診断を開始する
3. 表示される質問に順番に回答する（約1〜2分）
4. すべて回答すると診断結果が自動表示される
5. 結果のアドバイスを確認してSNSでシェアする

## よくある質問（FAQ）

**Q: 腰痛・肩こり原因チェッカーは無料で使えますか？**

はい、完全無料でご利用いただけます。登録・ログイン・アプリのインストールも不要です。

**Q: 何回でも診断できますか？**

はい、何度でもご利用いただけます。条件を変えて繰り返し診断してみてください。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてのデバイスに対応しています。

**Q: 診断結果は保存されますか？**

はい、ブラウザのlocalStorageに最新10件の診断結果が自動保存されます。同じブラウザ・同じデバイスからアクセスすると前回の診断結果が表示されます。ブラウザのキャッシュをクリアするとデータが削除されます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、個人情報・入力データのサーバー送信は行いません。

### v1.3.0（2026-06-10）
- **M10 印刷・エクスポート機能**: 結果画面に「印刷」「CSVダウンロード」ボタンを追加。診断結果をPrint/CSVで保存可能に。@media print で印刷時レイアウト最適化

- **M4 Chart.js グラフの完全実装**: 結果画面にスコア分布の横棒グラフ（4タイプ別色分け）を表示。スコア詳細（数値）も同時表示し、診断理由が視覚的に理解しやすく

### v1.2.2（2026-06-04）
- **PWA アイコン色の統一**: manifest.json のアイコンカラーを #6366F1（紫）から #EA580C（オレンジ）に変更。theme-color・ヘッダー背景色・ブランドカラーと完全に統一され、PWAのブランド表現が一貫性を持つように改善
