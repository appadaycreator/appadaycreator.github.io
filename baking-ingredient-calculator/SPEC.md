# お菓子・パン材料計算ツール【無料】簡単・無料計算 - 技術仕様書

## 概要

**サービス名**: Baking Ingredient Calculator
**バージョン**: 2.2.0
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/baking-ingredient-calculator/

レシピの分量を人数・型サイズに合わせて自動換算。グラム・カップ・cc相互変換も対応。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 入力フォームの各項目を確認する
2. 必要な数値・条件をすべて入力する
3. 計算ボタンをクリックして結果を取得する
4. 計算結果と詳細な内訳を確認する
5. 条件を変えて複数パターンを比較する

## よくある質問（FAQ）

**Q: お菓子・パン材料計算ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 計算結果は正確ですか？**

参考値としてご活用ください。重要な判断の際は専門家・公式機関にご確認ください。

**Q: 何回でも計算できますか？**

はい、回数制限なく何度でもご利用いただけます。条件を変えて比較にお使いください。

**Q: 入力したデータはどこかに保存されますか？**

いいえ。すべての計算はブラウザ内で処理され、データのサーバー送信は行いません。

**Q: スマートフォンで利用できますか？**

はい、PC・スマートフォン・タブレットすべてに対応しています。

## 更新履歴

### v2.3.0 (2026-06-10) - コード品質向上・重複関数統合版

#### 実装した主な改善（P1・P2 対応）

**P1: コード重複排除・統合**
1. 重複関数定義の削除
   - `calculate()` 関数 2 個 → 1 個に統合
   - `addIngredient()`, `removeIngredient()`, `resetForm()` 関数の重複排除
   - localStorage キー名を `bakingCalc_data` から `bakingData` に統一
   - 不要な `copyResults()` 関数を削除、`copyMarkdown()` に統合

2. JavaScript コードの整理
   - 古い `loadFromLocalStorage()` / `saveToLocalStorage()` 関数を削除
   - 計算ロジックを単一の `calculate()` 関数に集約
   - 約200行の不要なコードを削除

3. グラフ描画の安定性向上
   - `drawCharts()` 内で既存チャートインスタンスの確実な破棄
   - `resetForm()` で Chart インスタンスも明示的に破棄

**P2: ユーザーフィードバック改善**
4. Copy 機能の改善
   - `copyMarkdown()` で event.target ではなく `querySelector()` で適切にボタン取得
   - コピー成功メッセージの表示時間を 2000ms → 3000ms に延長
   - エラーメッセージも統一形式に修正

5. エラーハンドリング統一
   - `showError()` 関数で統一されたエラー表示
   - 計算エラー時の自動消去を 4000ms に統一

**技術仕様（変更点）**

修正・統合した JavaScript 関数：
- `calculate()`: 統合版（グラフ・エクスポート込み）
- `drawCharts()`: メモリリーク対策強化版
- `resetForm()`: グラフインスタンス破棄機能追加
- `copyMarkdown()`: イベント処理修正版

localStorage 仕様：
- キー名: `bakingData` (統一)
- データ構造: `{scalingRatio, originalPeople, targetPeople, ingredients}`
- 自動保存: 計算実行時、ページ再読み込み時に復元
