# バイク・原付購入維持費診断 | バイクの年間維持費を無料試算 | AppADayCreator - 技術仕様書

## 概要

**サービス名**: Bike Cost Advisor
**バージョン**: 1.5.1
**更新日**: 2026-06-20
**URL**: https://appadaycreator.com/bike-cost-advisor/

バイク・原付・スクーターの購入費用と年間維持費を7つの質問で無料試算。車種・用途・保険を考慮した総コストと最適な選び方をアドバイスします。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## 更新履歴

### v1.5.1 (2026-06-20)
- **UI改善 - 質問進行表示の統一（V2）**: テキスト「質問X/7」とプログレスバーが横並びで視線が分散していた問題を解消。テキストをバーの上部ラベルとして配置し、プログレスバーを全幅表示に変更（w-32→w-full）してより直感的な進捗把握を実現
- **UI改善 - ラジオ選択肢の視覚的分離（V1）**: 選択肢ラベルに `bg-gray-50` 背景色・`hover:bg-red-50` ホバー色を追加し、白背景からの視覚的分離を強化。スマホでの誤タップリスクを軽減
- **アクセシビリティ改善（M20）**: ラジオボタンの `aria-label` を全選択肢共通の「入力フォーム」から各選択肢テキストに変更し、スクリーンリーダー対応を改善

### v1.5.0 (2026-06-13)
- **M6: 目標設定・達成度トラッキング**: 
  - 年間維持費の目標値設定フォームを結果画面に追加
  - 目標値と実績のプログレスバーで達成度を可視化（100%基準）
  - localStorageに目標値を保存して次回訪問時に復元
  - 前回の診断結果との比較機能（改善度・増加度を動的表示）
  - 継続利用を促進するための「🎯 目標値設定・達成度トラッキング」セクション

### v1.4.0 (2026-06-10)
- **M4グラフ・ビジュアライゼーション（Chart.js実装）**: 
  - 年間維持費の内訳を円グラフ（doughnut）で表示
  - 初年度総費用 vs 翌年以降の年間維持費を棒グラフで表示
  - 5年間の累計コスト推移を折れ線グラフで表示
  - すべてのグラフは診断結果後に自動描画
  - モバイル・PC双方でレスポンシブに対応
- **UI/UX細部改善**: 
  - 各質問に「💡 ヒント文」を追加（免許・走行距離・駐車場など選択の指針を補助）
  - 質問下の説明文により入力時の迷いを軽減

### v1.3.0 (2026-06-10)
- **localStorage を利用したデータ保存・履歴機能（M3実装）**: 診断完了時に入力値と計算結果をブラウザに自動保存し、過去10件の診断履歴を表示・復元可能に
- **印刷・PDF保存機能（M10実装）**: 結果セクションに「📄 印刷・PDF保存」ボタンを追加、window.print()で印刷ダイアログ起動
- **結果テキストコピー機能**: 診断結果をテキスト形式でクリップボードにコピーする「📋 結果をコピー」ボタンを追加
- **履歴復元機能**: 過去の診断結果から入力値を復元して再診断できるように対応

### v1.2.0 (2026-06-06)
- **コスト内訳の動的表示**: 年間維持費の内訳（税金・保険・燃料・整備・駐車場）を選択内容に応じた実際の試算値で表示
- **5年間総コスト追加**: 購入費+年間維持費×5年の長期コストを結果に表示
- **予算比較フィードバック**: 選択した予算と試算購入費を比較し、予算超過警告または確認メッセージを表示
- **燃料費計算バグ修正**: 「200km未満」選択時に誤って1万円加算されていた不具合を修正（`0 || 1` → `?? 1`）
- **結果シェアボタン追加**: 実際のコスト数値を含むLINE/Xシェアボタンを結果セクションに追加
- **利用者の声修正**: バイクと無関係な「車の購入」の体験談をバイク関連に差し替え

### v1.1.0 (2026-06-04)
- **購入費用計算機能追加**: バイク種別と新車/中古の選択から想定購入費用を自動計算
- **初年度総費用の表示**: 購入費用 + 年間維持費を「初年度総費用」として表示
- **税金表記修正**: 「軽自動車税」→「軽二輪税」に訂正
- **関連商品リンク修正**: 楽天市場の検索キーワードを「自転車」→「バイク ヘルメット」に修正
