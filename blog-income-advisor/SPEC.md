# ブログ・SNS収益化適性診断｜性格・スキルから最適な収益化手段を診断 - 技術仕様書

## 概要

**サービス名**: Blog Income Advisor
**バージョン**: 1.3.0
**更新日**: 2026-06-20
**URL**: https://appadaycreator.com/blog-income-advisor/

性格・スキル・時間から最適なブログ・SNS収益化手段を診断。ブログ・YouTube・Instagram・Podcastなど8つの質問で最適な副業スタイルがわかる無料診断。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- Chart.js（v4.4.0）— グラフ・ビジュアライゼーション
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## 変更履歴

### v1.3.0 (2026-06-20)
- **収益バグ修正**: 診断結果後のアフィリエイトセクション（📚 収益化ノウハウを学ぶ）が `display:none` でハードコードされていたため非表示だった問題を修正（6本のA8アフィリエイトリンクが常に非表示だった）
- **V1対応**: quiz-btn にactive状態のシャドウ・スケール・選択時紫リングを追加し、モバイルでのタップフィードバックを視覚化

### v1.4.0 (2026-06-10)
- **M4施策実装**: Chart.js を使用した診断結果の可視化（ドーナッツゲージ・横棒グラフ）
  - ドーナッツチャート：最推奨手段のマッチ度を円形ゲージで表示（中央にパーセンテージ表示）
  - 横棒グラフ：TOP3の手段をマッチ度（%）で比較
  - レスポンシブ対応（モバイル・PC 両対応）
- グラフ表示時に古いチャートを自動削除（メモリリーク防止）

### v1.3.0 (2026-06-10)
- localStorage によるデータ保存・復元機能を実装（入力値と診断結果を自動保存）

- 結果表示UI を改善（コピー・CSV・印刷・LINE・X シェアボタンを整理）

- 結果カードに収益化開始期間（income）フィールドを追加表示
- スコア生数値表示をマッチ度パーセンテージバーに改善
- 結果ページにXシェアボタンを追加（LINE・コピーと並列表示）
