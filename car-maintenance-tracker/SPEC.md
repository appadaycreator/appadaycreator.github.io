# 車のメンテナンス時期チェッカー【無料】記録・管理アプリ - 技術仕様書

## 概要

**サービス名**: Car Maintenance Tracker
**バージョン**: 1.5.0
**更新日**: 2026-06-19
**URL**: https://appadaycreator.com/car-maintenance-tracker/

走行距離・購入年・最終点検日からオイル交換・タイヤ交換・車検の時期を診断。愛車の維持コストを把握。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 機能一覧（v1.2.0）

- **メンテナンス時期診断**: 現在の走行距離を入力すると、記録データをもとに各メンテナンスの状態を進捗バー＋色分け（良好🟢/要注意🟡/要交換🔴）で表示
- **標準間隔**: オイル交換5000km/6ヶ月、タイヤ交換30000km/3年、車検2年、バッテリー交換3年、洗車1ヶ月
- **日付指定記録**: 任意の実施日（デフォルト今日）でメンテナンスを記録
- **種類別フィルター**: 記録一覧をメンテナンス種類で絞り込み
- **CSVエクスポート**: 記録全件をBOM付きUTF-8のCSVでダウンロード
- **種類別サマリー**: 費用・件数を種類別テーブルで集計表示
- **自動サマリー表示**: ページリロード後も保存済みデータのサマリーを表示

## 使い方

1. 「メンテナンス時期診断」に現在の走行距離を入力して状態を確認する
2. 記録フォームで実施日・種類・走行距離・費用を入力する
3. 「記録を追加」ボタンで保存する
4. 種類別フィルターや記録一覧で過去のデータを確認する
5. CSVダウンロードでデータをバックアップする

## よくある質問（FAQ）

**Q: 車のメンテナンス時期チェッカーは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。
