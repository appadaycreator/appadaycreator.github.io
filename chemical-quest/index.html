化学を学ぼう

基礎から応用まで、楽しく化学を学習

⚛️

基礎化学

  • ⚛️ 原子 — 物質の最小単位。陽子・中性子・電子で構成
  • 🔗 分子 — 原子が結合したもの（H₂O = 水素2 + 酸素1）
  • 📋 元素 — 同じ原子番号を持つ原子の種類（118種類）
  • ⚖️ 原子量 — 炭素12を基準とした相対質量
🔗

化学結合

  • ⚡ イオン結合 — Na⁺とCl⁻が引き合う（食塩 NaCl）
  • 🤝 共有結合 — 電子を共有する（水 H₂O、二酸化炭素 CO₂）
  • 🔩 金属結合 — 自由電子が金属原子間を移動する
  • 💧 水素結合 — HとO/N/Fの間の弱い結合
🧪

化学反応

  • 🔥 燃焼 — CH₄ + 2O₂ → CO₂ + 2H₂O
  • ⚗️ 中和 — HCl + NaOH → NaCl + H₂O
  • 🌱 光合成 — 6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂
  • ⚖️ 質量保存則 — 反応前後の質量は変わらない

化学式パズルで学ぶ元素記号

🧪 ゲームの遊び方

元素記号（H・O・C・N等）を組み合わせて、水（H₂O）・二酸化炭素（CO₂）・アンモニア（NH₃）などの化学式を作るパズルゲームです。理科・化学の授業の予習復習に最適。中学生・高校生が楽しみながら元素記号と化学式を覚えられます。

📚 学習効果

ゲーム感覚で化学式を作ることで、元素の組み合わせパターンが自然と身につきます。H₂O、CO₂、NaCl、C₆H₁₂O₆（グルコース）など、日常生活でよく登場する化合物の化学式を楽しく学習できます。定期テスト対策にも活用できます。
