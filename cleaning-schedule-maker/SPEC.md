# 掃除スケジュール自動作成ツール【無料】自動作成ツール - 技術仕様書

## 概要

**サービス名**: Cleaning Schedule Maker
**バージョン**: 1.4.1
**更新日**: 2026-06-20（UI/UX改善・FAQセクション最適化）
**URL**: https://appadaycreator.com/cleaning-schedule-maker/

家の間取り・ライフスタイルから最適な掃除スケジュールを自動生成。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## 更新履歴

### v1.4.1 (2026-06-20)
- **UI改善**: グレーアウトボタン（寝室・玄関・ベランダ・書斎）に `disabled` 属性・ツールチップを追加
  - ホバー時に「このサービスでは未対応の部屋です」表示で、ユーザーの混乱を解消
  - disabled スタイル（opacity 0.5, cursor: not-allowed）を追加
- **フォーム改善**: select 要素のパディングを 8px → 6px 8px に縮小
  - モバイルでの表示領域を拡大し、「週に使える掃除時間の目安」フィールド全体が見えるように改善
- **SEO改善**: meta description を 160文字超から120-160文字に最適化
  - 「無料」「自動生成」「掃除計画」などのキーワード強化
- **コンテンツ整理**: ページ内の空のFAQセクションを削除（JSON-LD FAQPage は維持）

### v1.4.0 (2026-06-13)
- **M5機能**: 計算結果に基づいた動的アフィリエイト商品推薦を実装
  - `generateDynamicRecommendations()` 関数で、選択部屋・人数・ペット有無をもとに推薦商品をスコア化
  - 各商品に priority・rooms・hasPet・minMembers 属性を設定し、ユーザー入力に応じて上位3-4商品を推薦
  - 例: ペットあり → クリーニング手袋・除菌スプレー優先 / 時間超過 → 洗剤セット優先
- **M5機能**: A8.net アフィリエイト掃除用品6種類を推薦マスタ化（洗剤・手袋・雑巾・スプレー・ハンガー・フィルター）
- **M5機能**: マウスオーバーで推薦商品がハイライト表示、各リンクは a8mat パラメータで収益計測可能
- **GSC対策**: メタタイトル・description に主要キーワード（掃除スケジュール・自動作成・無料・間取り・ライフスタイル）追加
- **GSC対策**: FAQPage schema 5問、WebApplication・HowTo・BreadcrumbList schema を装備

### v1.3.0 (2026-06-10)
- **M3機能**: localStorage を使用した計算結果の自動保存機能を実装
- **M3機能**: 「前回のデータを復元」ボタンで前回の入力値を即座に復元
- **M3機能**: 「履歴を表示」ボタンでモーダル形式で過去10件の計算履歴を時系列表示
- **M3機能**: 「履歴をすべて削除」ボタンで不要な履歴を一括削除（確認ダイアログ付き）
- **M4機能**: Chart.js を導入し、計算結果を複数のグラフで可視化
- **M4機能**: 横棒グラフで「設定予算」と「実際に必要な週あたり分数」を比較
- **M4機能**: 進捗バーで予算達成率を視覚化（超過時は赤色警告表示）
- **M4機能**: ドーナツチャートで部屋別の掃除時間の割合を表示
- **M4機能**: 詳細カードで総タスク数・毎日/週1回/月1回/季節ごとのタスク分類を表示
- **M10機能**: 「CSVダウンロード」ボタンで計算結果をCSV形式でダウンロード（日付自動付与）
- **M10機能**: 「テキストコピー」ボタンでブラウザのクリップボードに結果をコピー
- **M10機能**: 「印刷する」ボタンで @media print CSS を活用した見栄えの良い印刷
- **UX改善**: 各操作後にトースト通知（3秒自動消去）で確認メッセージを表示
