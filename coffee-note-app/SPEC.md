## 機能仕様

### 1. コーヒー記録機能

#### 1.1 データ構造

```javascript
{
  id: number,                    // タイムスタンプベースのID
  beanName: string,              // 豆の名前（必須）
  origin: string,                // 産地
  roastLevel: string,            // 焙煎度
  brewingMethod: string,         // 抽出方法
  beanAmount: number,            // 豆の量(g)
  waterAmount: number,           // 水の量(ml)
  waterTemp: number,             // 水温(℃)
  extractionTime: number,        // 抽出時間(秒)
  grindSize: string,             // 挽き具合
  overallRating: number,         // 総合評価(1-5)
  aroma: number,                 // 香り(1-5)
  acidity: number,               // 酸味(1-5)
  bitterness: number,            // 苦味(1-5)
  sweetness: number,             // 甘味(1-5)
  body: number,                  // コク(1-5)
  flavorNotes: array,            // フレーバーノート
  notes: string,                 // メモ
  favorite: boolean,             // お気に入りフラグ
  date: string,                  // 記録日時(ISO形式)
  userId: string                 // ユーザーID
}
```

#### 1.2 入力検証 (M2: v1.16.0 改善版)

**リアルタイムバリデーション機能**:
- 記録日時: 未来の日時は選択不可、エラーメッセージ表示
- 豆の名前: 必須入力、入力エラーメッセージ表示
- 産地: 必須入力、2文字以上、エラーメッセージ表示
- 焙煎度: 必須選択、エラーメッセージ表示
- 抽出方法: 必須選択、エラーメッセージ表示
- 豆の量: 5～100g の範囲チェック、警告メッセージ表示
- 水の量: 50ml 以上の確認、黄金比の自動計算・表示
- 水温: 70℃～100℃ の範囲チェック、焙煎度別の推奨温度ヒント表示
- 抽出時間: 30秒～600秒 の範囲チェック、抽出方法別の目安表示
- 挽き具合: 必須選択、エラーメッセージ表示

**フォーム送信時のバリデーション**:
- 全ての必須フィールドの一括チェック
- バリデーション失敗時は詳細エラーメッセージを表示して保存を中止
- 抽出比率（豆：水）の自動計算・最適値判定表示

## データ管理

現行はすべて **ブラウザ localStorage** に保存。外部APIへの通信なし。

```javascript
// 保存先キー
localStorage.getItem('coffeeRecords')  // コーヒー記録（JSON配列）
localStorage.getItem('coffeeBeans')    // 豆情報（JSON配列）
localStorage.getItem('language')       // 言語設定（'ja'|'en'）
localStorage.getItem('fontSize')       // フォントサイズ（'sm'|'base'|'lg'）
```
