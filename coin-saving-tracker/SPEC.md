# 小銭貯金・貯金箱記録ツール【無料】記録・管理アプリ - 技術仕様書

## 概要

**サービス名**: Coin Saving Tracker
**バージョン**: 1.5.1
**更新日**: 2026-06-20
**URL**: https://appadaycreator.com/coin-saving-tracker/

小銭貯金の総額をリアルタイム計算。100円・500円貯金など目標設定も対応。複数の貯金記録を保存・管理可能。登録不要・完全無料でご利用いただけます。

## 機能一覧（v1.4.0 入力UX・ガイダンス改善）

- **リアルタイム計算**: 硬貨の枚数を入力すると、即座に合計金額が更新される
- **目標金額設定**: 目標額を設定して進捗バーで可視化
- **記録保存機能**: 記録名を入力して現在の状態をlocalStorageに保存。一覧から読込・削除が可能
- **入力値検証**: 負の数や不正な値を自動修正
- **モバイル最適化**: スマートフォン（480px以下）でコイングリッドを2列表示
- **SNSシェア機能**: Xシェア・LINEシェアに現在の貯金額と目標達成率を含めて共有
- **グラフ可視化（NEW v1.3）**: Chart.jsを使用した硬貨内訳ドーナツチャート・目標達成率ゲージで直感的に可視化
- **結果エクスポート機能（NEW v1.3）**: 
  - 📋 クリップボードコピー：結果をテキストでコピー
  - 📥 CSVダウンロード：硬貨内訳・合計・目標をCSV形式で保存
  - 🖨️ 印刷：window.print()で印刷対応
- **使い方セクション**: ツール固有の操作フローに合わせた内容
- **アクセシビリティ改善**: ボタン・グラフに適切な aria-label を設定
- **入力フィールドの UX 改善（v1.4）**: 
  - コイン入力フィールド（6個）に `placeholder="0"` を追加（何を入力すべきか明確に）
  - 全入力フィールド（コイン・目標金額・記録名）に `aria-label` を追加（スクリーンリーダー対応）
  - 全入力フィールドに `title` 属性でツールチップを設定（マウスホバー時に説明表示）
  - 各入力セクション下部にガイダンステキストを追加（何を入力すべきかを明確に説明）
- **クッキーバナーUX改善（NEW v1.5.1 P1施策）**:
  - 背景色を `#1e293b` （濃いグレー）から `#f8fafc` （淡色）に変更し、フッターとの情報階層を明確化
  - パディング削減（`1rem 1.5rem` → `0.75rem 1rem`）でバナー高さを60px以下に圧縮
  - ボタンテキストを「同意して閉じる」→「同意する」に短縮
  - `justify-content: center` で テキスト・ボタンを中央配置し視覚的距離を最小化
  - localStorage への `cookie-consent` フラグ保存を追加（再訪問時に自動非表示）
- **FAQ充実化（NEW v1.5.1 P1施策 - GSC優先対応）**:
  - 既存Q&A 3問を維持
  - 新たに 3 問追加（計 6 問）:
    1. 「硬貨の枚数を間違えて入力した場合、修正できますか？」→ 入力フィールドの上書き修正対応を説明
    2. 「複数の貯金箱を管理することはできますか？」→ 記録保存機能で複数管理可能と説明
    3. 「スマートフォンとパソコンでデータは同期されますか？」→ ローカルストレージの仕様を説明
    4. 「目標金額に達した場合、何か自動で起こりますか？」→ 達成メッセージ・リセット機能を説明
  - サービス固有の内容のみで、汎用文を排除
- **ツールチップ・入力ガイダンス充実（NEW v1.5 M8施策）**:
  - 各硬貨入力欄（500円～1円）に `.input-helper` で「◯◯玉の枚数」ガイダンステキスト追加
  - 目標金額欄に「貯めたい金額を円単位で入力（例: 10000）」ガイダンステキスト追加
  - 記録名欄に「例: 2026年6月の貯金箱、旅行資金」ガイダンステキスト追加
  - 各セクションに「?」アイコン（`.help-icon`）を配置し、ホバーでツールチップ表示（CSS`:hover::after`で実装）
  - ツールチップデザイン: 吹き出し型（#1f2937背景、白テキスト、矢印付き）
- **結果に連動したアフィリエイト商品推薦（NEW v1.5 M5施策 - リンク実装完了）**:
  - グラフセクション直後に `.affiliateRecommend` セクション追加（初期状態: `display:none`）
  - `updateAffiliateRecommend(total, goal)` 関数で、目標金額に基づいて推薦コンテンツを動的に分岐:
    - goal ≥ 50,000円: NISA・投資信託系（A8.net経由の資産運用サービスリンク）
    - goal 10,000～50,000円: 家計管理・節約系（ハピタス・カード関連A8.netリンク）
    - goal < 10,000円 かつ total > 0: ポイントサイト系（ハピタス A8.netリンク）
  - 推薦カード: クリック可能なリンク要素（`<a>`タグ）で実装、`target="_blank"` で別窓表示、`rel="nofollow noopener noreferrer"`で安全性確保
  - デザイン: ホバーエフェクト（opacity 0.8）、背景色分け、「詳しく見る →」テキストで操作性向上
  - アフィリエイト: A8.net px.a8.net リンク（既存の「ハピタス」「じゃらん」と同一パラメータ体系）を使用

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化
- **データ構造**: 記録は JSON 形式で保存（日時・金額・硬貨内訳）

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## よくある質問（FAQ）

**Q: 小銭貯金・貯金箱記録ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
