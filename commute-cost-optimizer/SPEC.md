# 通勤コスト最適化診断【無料】簡単・無料計算 - 技術仕様書

## 概要

**サービス名**: Commute Cost Optimizer
**バージョン**: 1.2.0
**更新日**: 2026-06-13
**URL**: https://appadaycreator.com/commute-cost-optimizer/

電車・バス・車・自転車の通勤コストを比較して最適な通勤方法を診断。登録不要・完全無料でご利用いただけます。

## v1.2.1 新機能 (2026-06-13)
- **M2 コア機能のUX改善（リアルタイムバリデーション対応）**: 入力時にリアルタイム入力バリデーション、ボーダーカラー変更で視覚的フィードバック、エラーメッセージを動的に表示、aria-invalid/role=alert でアクセシビリティ向上、スライダーとの双方向同期で直感的入力体験

## v1.2.0 新機能 (2026-06-13)
- **M2 コア機能のUX改善**: 入力フォームにplaceholder/ラベル充実化、バリデーション強化、エラーメッセージ表示で入力補助、範囲スライダー（距離・日数）で直感的な入力をサポート
- **M6 目標設定・達成度トラッキング**: 目標月間交通費の設定、達成度プログレスバー表示、目標達成時の自動検出機能
- **M7 比較・シミュレーション機能**: What-ifシミュレーション（距離±5km、出勤日±5日）を複数パターン表示

## v1.1.0 新機能 (2026-06-10)
- **M3 データ保存・履歴機能**: localStorage で計算履歴を自動保存（最新10件）、ページ読み込み時に前回データを復元
- **M4 グラフ可視化**: Chart.js で通勤方法別の月額コスト比較棒グラフ、最適度スコア（0-100）ゲージ表示
- **M10 印刷・エクスポート**: 結果をクリップボードにコピー、window.print() による印刷対応

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する
