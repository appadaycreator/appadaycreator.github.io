# 仮想通貨損益計算ツール【無料】簡単・無料計算 - 技術仕様書

## 概要

**サービス名**: Crypto Profit Calculator
**バージョン**: 1.4.0
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/crypto-profit-calculator/

購入価格・現在価格から仮想通貨の損益・税金概算を計算。複数通貨対応。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 新機能（v1.4.0）【2026-06-10 更新】

### P1 実装済み（改善施策 M4）
- **利益スコア分析パネルで複合ビジュアライゼーション【M4 拡張】**: 
  - **スコア表示（ゲージ）**: 利益効率を 0-100 スコアで表示。利益率 25% で満点化。視覚的に利益性を判定
  - **プログレスバー（利益率）**: 仮想通貨の購入単価に対する利益率を % 表示＆プログレスバー化。0% ～ 100% の直感的な進捗表示
  - **プログレスバー（手取り率）**: 税差引後の実質利益率を % 表示＆プログレスバー化。税負担の重さを視覚化
  - **ドーナツチャート（税シミュレーション）**: 税前利益と推定税額の比率をドーナツ図で表示。手取額と税額の構成比を一目瞭然
  - 外部 API なし・ブラウザ内で全て動作

## 旧機能（v1.3.0）【2026-06-10 実装】

### P1 実装済み（改善施策 M8）
- **入力フォームへのツールチップ充実【M8】**: 各入力欄（購入単価・現在単価・保有数量・課税所得）に？アイコンツールチップを追加。ホバーで詳細説明表示
- **入力ガイダンス拡張**: 課税所得欄に具体例「給与500万円＋暗号損益50万円＝550万円」を図解。ユーザーの迷い解消

## 旧機能（v1.2.0）

### P1 実装済み
- **複数通貨対応**: JPY・USD・EUR・GBP・CNY に対応。選択した通貨で結果を表示
- **詳細な損益内訳表示**: 購入時総額、現在時総額、税前利益、推定税額、税率、**手取額（実質利益）**を明細表示
- **入力値の自動保存**: localStorage を活用し、ページ再読み込み後も入力値を復元。複数パターン計算時の効率向上

### P2 実装済み
- **クリアボタン**: 入力値と計算結果をワンクリックでリセット
- **課税所得説明の改善**: ヘルプテキスト「給与・事業所得など他の所得を含めた合計」を追加して、ユーザーの入力精度向上

## 使い方

1. 入力フォームの各項目を確認する
2. 必要な数値・条件をすべて入力する
3. 計算ボタンをクリックして結果を取得する
4. 計算結果と詳細な内訳（購入時総額、現在時総額、税前利益、推定税額、**手取額**）を確認する
5. 条件を変えて複数パターンを比較する（入力値は自動保存されます）

## よくある質問（FAQ）

**Q: 仮想通貨損益計算ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 計算結果は正確ですか？**

参考値としてご活用ください。重要な判断の際は専門家・公式機関にご確認ください。

**Q: スマートフォンで利用できますか？**

はい、PC・スマートフォン・タブレットすべてに対応しています。
