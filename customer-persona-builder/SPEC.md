# 顧客ペルソナ作成ツール【無料】自動作成ツール - 技術仕様書

## 概要

**サービス名**: Customer Persona Builder
**バージョン**: 1.8.0
**更新日**: 2026-06-22
**URL**: https://appadaycreator.com/customer-persona-builder/

ビジネス内容を入力すると顧客ペルソナ（年齢・悩み・行動）を自動生成してマーケ戦略に活用。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- Chart.js 4.4.0（ペルソナスコアの可視化）
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 機能

- **ペルソナ自動生成**: 業種×ターゲット×価格帯の組み合わせから3つの顧客ペルソナを自動選択
- **ペルソナ表示項目**: 名前・年齢・性別・職業・年収・家族構成・悩み3点・情報収集方法・購買決定要因・代表発言
- **業種カテゴリ**: 美容・食品・IT・教育・健康・フィットネス・エンタメ・金融・不動産・その他（計10種）
- **ターゲット**: B2C（一般消費者）/ B2B（法人）
- **価格帯**: 低価格／中価格／高価格（価格帯によってペルソナ順を変更）
- **テキストコピー**: 生成ペルソナをクリップボードにコピー（SNS・社内共有向け）
- **スコア表示**: 年収スコア・ニーズスコア・情報感度スコアをプログレスバーで可視化
- **履歴管理**: 過去の生成結果を localStorage に自動保存・復元・削除機能
- **CSV エクスポート**: 生成ペルソナを CSV ファイルでダウンロード

## 使い方

1. 業種カテゴリ・ターゲット・価格帯を選択する
2. 商品・サービスの説明を任意で入力する
3. 「ペルソナを生成する」ボタンをクリックする
4. 生成された3つのペルソナを確認・活用する
5. 条件を変えて再生成し、複数パターンを比較する
