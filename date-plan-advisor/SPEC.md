## 更新履歴

### v1.4.0 (2026-06-13)
- **目標設定・達成度トラッキング（M6）**: 次のデートの目標マッチ度を設定可能に、前回診断スコアとの進捗を表示
- **比較・シミュレーション機能（M7）**: 別条件を選択してデートプラン候補を比較・シミュレーション機能を追加

### v1.3.0 (2026-06-10)
- **データ保存・履歴機能（M3）**: localStorage を使用した診断履歴の自動保存（最大10件）
- **マッチ度スコア表示（M4）**: 診断結果に「マッチ度スコア」ゲージを表示
- **CSV・テキストエクスポート（M10）**: 診断結果をCSV形式でダウンロード、テキストコピー可能
- **印刷機能（M10）**: window.print() で結果をPDF/紙に印刷可能
- **履歴削除機能**: ブラウザに保存された履歴を削除するボタンを追加

### v1.2.0 (2026-06-05)
- **戻るボタン実装**: 質問2以降「← 前の問題に戻る」ボタンを表示、前回の回答を保持
- **キーボード操作対応**: Enter キーで選択肢を確定、矢印キー（↑↓）で選択肢移動、タブ移動対応
- **シェアボタン移動**: JavaScript動的生成から HTML に直接組み込み（結果画面に常時表示）
- **絵文字付きコース表示**: 10種類の絵文字（🚶🍽️🎬⭐等）を各おすすめコースに付与
- **アクセシビリティ向上**: tabindex 設定、フォーカス管理、キーボードナビゲーション対応

### v1.1.0 (2026-06-04)
- **UX改善**: 初期画面の説明文拡張で利用内容を明確化
- **モバイル最適化**: オプションボタンの min-height を 48px → 56px に拡張（WCAG AAA対応）
- **シェア機能**: Twitter/LINE/URL コピー機能を結果画面に追加
- **視覚改善**: 各おすすめコースに絵文字を付与、背景色を段階的に変更
- **関連サービス導線**: 結果下部に他のカップルサービスへのリンク追加
- **ナビゲーション改善**: header を sticky化してモバイルスクロール時も常に表示

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、診断の説明を確認する
2. スタートボタンをクリックして診断を開始する
3. 表示される質問に順番に回答する（約1〜2分）
4. すべて回答すると診断結果が自動表示される
5. 結果のアドバイスを確認してSNSでシェアする

## よくある質問（FAQ）

**Q: デートプラン診断・提案は無料で使えますか？**

はい、完全無料でご利用いただけます。登録・ログイン・アプリのインストールも不要です。

**Q: 何回でも診断できますか？**

はい、何度でもご利用いただけます。条件を変えて繰り返し診断してみてください。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてのデバイスに対応しています。

**Q: 診断結果は保存されますか？**

結果はブラウザには保存されません。スクリーンショットやSNSシェアでご記録ください。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、個人情報・入力データのサーバー送信は行いません。
