このアプリケーションは、部屋情報を Supabase に保存し、その他のデータをブラウザのローカルストレージに保存するシングルページアプリです。ユーザー管理には Supabase 認証を利用します。

## 主な機能
- 部屋、アイテム、写真の登録と管理
- 進捗状況のチャート表示
- Supabase を利用した認証 (メール/パスワード)
- データのインポート・エクスポート
- 利用規約やプライバシーポリシー等の法的情報をフッターから閲覧可能

## 技術スタック
- Vue 3 (CDN 版)
- Supabase JS (UMD 版)
- Tailwind CSS
- Chart.js
