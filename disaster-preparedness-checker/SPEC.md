# 仕様書

このアプリは以下のファイルで構成されています。

- **index.html**: メイン画面。使い方ガイドを含む。
- **privacy.html**: プライバシーポリシーを掲載するページ。
- **disclaimer.html**: 免責事項を掲載するページ。
- **app.js**: アプリの主要ロジックをまとめたJavaScript。
- **manifest.json**: PWA用の設定ファイル。
- Google Tag Manager を用いたアクセス解析。
- X（旧Twitter）へのシェア機能。

診断結果が100点の場合は改善提案を表示せず、必要に応じて「追加の備え」として任意のアドバイスを提示します。

特別なサーバーやビルド工程は不要で、ブラウザで `index.html` を開くだけで利用できます。

## バージョン履歴

### v1.3.1 (2026-06-13)
- **M2：入力UI・バリデーション改善（フェーズ2実装）** - 基本情報入力フォームのUX強化
  - select要素のfirst optionに disabled属性を追加（「— 選択してください —」）
  - startCheckBtn クリック時に都道府県・住居タイプ・家族人数の入力検証を実装
  - 入力欄の change イベントでリアルタイムバリデーションを実行
  - 入力エラー時に validationErrors div を表示し、errorList に詳細メッセージを追記
  - エラー表示時に該当要素を自動スクロール（UX向上）
- **M6：目標設定・達成度トラッキング** - 防災準備の目標設定と進捗管理機能を追加
  - 結果画面に「総合スコアの目標」を0〜100点の範囲で設定できるスライダーを実装
  - 目標値をlocalStorageに自動保存
  - 前回診断との比較表示：スコア差分・改善率（%）・トレンド表示（📈📉➡️）
  - 初回診断時は「次回の診断で前回との比較が表示されます」というガイダンスを表示
- **M8：ツールチップ・入力ガイダンス充実** - 各入力欄に詳細なヘルプテキストとツールチップを追加
  - 都道府県選択：地域のハザードリスク説明
  - 住居タイプ選択：住居構造と避難方法の関連性を記載
  - 家族人数選択：備蓄量計算（水3L/人/日など）について記載
  - 年齢構成セクション：ガイダンステキスト「該当する項目をチェックしてください...」を追加
  - 各チェックボックスにtitle属性で「粉ミルク・おむつ備蓄が必要」などの具体的な備蓄品を記載
