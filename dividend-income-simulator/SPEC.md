# 配当収入シミュレーター【無料】簡単・無料計算 - 技術仕様書

## 概要

**サービス名**: Dividend Income Simulator
**バージョン**: 2.2.0（localStorage + Chart.js + 印刷機能）
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/dividend-income-simulator/

保有株・投資信託の配当利回りから年間配当収入を計算。FIRE達成額との比較も。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## 新機能（v2.0.0）

### 1. クイックプリセット機能
- 「日本高配当株」「VYM(米国)」「バランス型」のプリセットボタンを追加
- ワンクリックで推奨値を自動入力可能
- 初心者向けの参考値を表示（日本高配当株: 3.5%など）

### 2. リアルタイム自動計算
- フォーム入力時に500msの遅延後、自動的に計算を実行
- ボタンクリックなしで即座に結果を確認可能
- デバウンス処理により連続入力時の計算回数を最適化

### 3. コピー・シェア機能
- 計算結果をテキストでコピー
- Twitter（X）で結果を共有
- LINEで結果をシェア
- 各SNSに最適な文言を自動生成

### 4. 入力値バリデーション強化
- リアルタイムエラー表示（alert不要）
- 異常値に対する警告メッセージ
- アクセシビリティ対応（aria-invalid/aria-describedby）

### 5. FIRE達成度の視覚改善
- プログレスバーの色を段階的に変更
  - 0-30%: 赤（#EF4444）
  - 30-70%: 黄色（#FBBF24）
  - 70-100%: 緑（#16A34A）
- 達成状況が一目でわかるUI

### 6. 計算精度向上
- Math.round()で月額配当を四捨五入
- Math.ceil()で必要資産額を切り上げ
- より精密な資産計算が可能

## 改善（v2.1.0）

### 1. FIRE達成度メッセージの動的生成
- 達成率に応じた段階的なメッセージを表示
  - 0-20%: 🚀 まだ始まったばかり。長期的に積み立てを継続しましょう
  - 20-50%: 📈 順調に進んでいます。継続が成功の鍵です
  - 50-80%: 🎯 FIREまでもう少し。目標が見えてきました
  - 80-100%: 🏆 おめでとう！FIRE達成が目前です
  - 100%以上: ✨ FIRE達成！自由な人生が始まります
- ユーザーのモチベーション向上

### 2. 税金クイック選択ボタン
- 「🏦 一般口座」「🎁 NISA」「✏️ カスタム」ボタンで税率を自動入力
- 初心者でも簡単に正確な税率を選択可能
- ボタンのactive状態で選択中の口座区分を視認可能

### 3. 数値入力UIの最適化
- `inputmode="decimal"` 属性を全ての数値入力に追加
- スマートフォンで小数点キーボードが確実に表示される
- より快適なモバイル操作体験

### 4. ヒント文言の改善
- 「1000万円 = 1000」の説明を「1000万円 = 1000 と入力」に改善
- ユーザーが入力方法をより明確に理解可能

## 改善（v2.2.0）

### 1. localStorage による計算履歴・入力値保存機能（M3実装）
- 入力値（保有資産額、配当利回り、税率、FIRE目標）を自動保存
- ページ読込時に前回の入力値を自動復元
- 過去5件の計算結果を履歴として保存・表示
- 「履歴をクリア」ボタンで一括削除可能
- 履歴から選択して再計算可能（「復元」ボタン）
- ブラウザ localStorage で永続化

### 2. Chart.js による結果可視化（M4実装）
- Chart.js CDN を読込
- 2つのグラフを結果セクション内に自動描画
  - **配当額推移グラフ（折れ線グラフ）**: 資産額（50%～150%）に応じた月額配当の推移を表示
  - **FIRE達成度ゲージ（ドーナツグラフ）**: 達成率と未達成率を色分けして表示
    - 0-30%: 赤（低進捗）
    - 30-70%: 黄色（中程度）
    - 70-100%: 緑（高進捗）
- モバイル・PC共にレスポンシブ対応

### 3. 印刷・PDF ダウンロード機能（M10実装）
- 「🖨️ 印刷」ボタンを追加（結果セクションのシェアボタン付近）
- `window.print()` でブラウザの print ダイアログを実行
- 既存の `@media print` CSS で不要な UI（ボタン、広告等）を非表示
- ブラウザの「PDF として保存」機能で PDF ダウンロード対応
