# 犬・猫の年齢換算ツール【無料】簡単・無料計算 - 技術仕様書

## 概要

**サービス名**: Dog Age Calculator
**バージョン**: 1.2.0
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/dog-age-calculator/

犬種・猫の年齢を人間年齢に換算。ライフステージ別のケアアドバイスも提供。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 入力フォームの各項目を確認する
2. 必要な数値・条件をすべて入力する
3. 計算ボタンをクリックして結果を取得する
4. 計算結果と詳細な内訳を確認する
5. 条件を変えて複数パターンを比較する

## v1.1.0 改善内容（2026-06-03）

### 猫・犬の年齢換算ロジック精密化
- **猫**: AAHA（米国動物病院協会）推奨式に準拠
  - 生後1年で人間の18歳相当、2年で25歳、以降1年ごと+4歳
  - より学術的な計算式に更新
- **犬（サイズ別）**: より精密な換算公式
  - 小型犬: 1年目は10倍、2年目は8倍、3年目以降は5倍
  - 中型犬: 1年目は11倍、2年目は8.5倍、3年目以降は6倍
  - 大型犬: 1年目は12倍、2年目は9倍、3年目以降は7.5倍

### 月齢（月単位）入力サポート
- 「歳 / 月」選択セレクタを追加
- 子犬・子猫ユーザーが月単位で直感的に入力可能
- 例：生後6ヶ月の子猫を「月: 6」で入力

### ライフステージ別の詳細ケアアドバイス
- シンプルな1行アドバイスから、3項目の詳細情報に拡張
  - 💡 **栄養**: 年齢別の食事内容の目安
  - 💡 **運動**: ライフステージに応じた運動量の提案
  - 💡 **注意点**: 健康診断の頻度・気をつけるべき症状
- 結果表示エリアに「ケアアドバイス」セクションを追加

### エラー表示の改善
- alert() から インラインエラー表示に変更
- より詳細で分かりやすいエラーメッセージ
  - 体重なし: 「体重を入力してください」
  - 年齢なし: 「年齢を入力してください」
  - 両方なし: 「体重を入力してください / 年齢を入力してください」
  - 無効な値: 「体重と年齢は0より大きい値を入力してください」

## よくある質問（FAQ）

**Q: 犬・猫の年齢換算ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 計算結果は正確ですか？**

参考値としてご活用ください。重要な判断の際は専門家・公式機関にご確認ください。

**Q: 何回でも計算できますか？**

はい、回数制限なく何度でもご利用いただけます。条件を変えて比較にお使いください。

**Q: 入力したデータはどこかに保存されますか？**

いいえ。すべての計算はブラウザ内で処理され、データのサーバー送信は行いません。

**Q: スマートフォンで利用できますか？**

はい、PC・スマートフォン・タブレットすべてに対応しています。
