# 教育費 総額いくら？【無料計算】幼稚園〜大学まで進路別シミュレーター - 技術仕様書

## 概要

**サービス名**: Education Cost Simulator
**バージョン**: 1.2.9
**更新日**: 2026-06-21
**URL**: https://appadaycreator.com/education-cost-simulator/

幼稚園から大学まで私立・公立別に教育費を無料計算。習い事費用込みで総額が分かる。学資保険・資金計画に活用できます。登録不要。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. お子様の年齢を選択する
2. 小学校・中学校・高校・大学の公立/私立を選択する
3. 習い事があれば複数選択する
4. 「教育費を計算する」ボタンを押す
5. 総額・内訳グラフ・月々の積立目標額を確認する

## よくある質問（FAQ）

**Q: 教育費 総額いくら？は無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。
