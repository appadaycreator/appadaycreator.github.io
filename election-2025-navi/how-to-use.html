学習モジュール

▶学習の進め方

  • 「参院選とは？」から順番に学習
  • 各モジュール後にクイズで理解度チェック
  • 80%以上の正答率で次のモジュールに進行
  • 全モジュール完了でバッジ獲得

🏆ゲーミフィケーション

  • ⭐ 学習進捗に応じてポイント獲得
  • ⭐ 各モジュール完了で特別バッジ
  • ⭐ 友達との進捗比較機能
  • ⭐ 全国ランキング表示
🔍

政策マッチング

❓質問システム

  • ✓ 経済政策・社会保障・外交など10分野
  • ✓ 各分野5問、計50問の質問
  • ✓ 「賛成」「反対」「どちらでもない」で回答
  • ✓ 重要度も3段階で設定可能

📊結果分析

  • % 各政党・候補者との一致度を％で表示
  • % 分野別の詳細分析グラフ
  • % マッチング理由の詳細説明
  • % 結果のSNSシェア・保存機能
