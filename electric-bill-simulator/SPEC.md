# 電気代シミュレーター【無料】簡単・無料計算 - 技術仕様書

## 概要

**サービス名**: Electric Bill Simulator
**バージョン**: 2.3.0
**更新日**: 2026-06-13
**URL**: https://appadaycreator.com/electric-bill-simulator/

家電の消費電力・使用時間から電気代を計算。節電効果もシミュレーション。複数パターン比較、プリセット家電、季節別料金対応。登録不要・完全無料でご利用いただけます。

## 新機能（v2.0.0）

### 1. プリセット家電機能
- 11種類の一般的な家電を瞬時に選択可能
- カテゴリ分類：冷暖房、調理家電、生活家電、その他
- 選択時に消費電力と標準使用時間を自動入力

### 2. ユニット変換（W / kW）
- 消費電力の単位を W または kW で指定可能
- 自動変換で正確な計算を実現

### 3. 季節別料金対応
- 春秋：26円/kWh（標準単価）
- 夏：32円/kWh（高単価）
- 冬：29円/kWh（高単価）
- 季節選択時に電力単価が自動更新

### 4. 複数パターン比較機能
- 複数のシナリオを追加して並列表示
- パターン間の削減額・削減率を自動計算
- 最大5パターンまで比較可能

### 5. 月別コスト推移グラフ
- 12ヶ月間の予想コスト推移を Canvas で描画
- 視覚的に年間電気代の傾向を理解可能

### 6. スマートアドバイス機能
- 高消費電力検出：500W以上の場合、使用時間短縮を提案
- 長時間使用検出：8時間以上の場合、デバイス買い替え提案
- 高額電気代検出：月3000円以上の場合、プラン見直し提案

### 7. データの自動保存・復元
- localStorage による自動保存
- 「前回のデータ」ボタンで即座に復元
- 「クリア」ボタンで履歴削除

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: 「クリア」ボタンまたはブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）

## 使い方

1. 入力フォームの各項目を確認する
2. 必要な数値・条件をすべて入力する
3. 計算ボタンをクリックして結果を取得する
4. 計算結果と詳細な内訳を確認する

## よくある質問（FAQ）

**Q: 電気代シミュレーターは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 計算結果は正確ですか？**

参考値としてご活用ください。重要な判断の際は専門家・公式機関にご確認ください。

**Q: 何回でも計算できますか？**

はい、回数制限なく何度でもご利用いただけます。条件を変えて比較にお使いください。

**Q: 入力したデータはどこかに保存されますか？**

いいえ。すべての計算はブラウザ内で処理され、データのサーバー送信は行いません。

**Q: スマートフォンで利用できますか？**

はい、PC・スマートフォン・タブレットすべてに対応しています。
