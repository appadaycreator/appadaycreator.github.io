# スペルマスター | 英単語スペルゲーム - TOEIC・英検対応 - 技術仕様書

## 概要

**サービス名**: English Word Spelling Game
**バージョン**: 1.3.0
**更新日**: 2026-06-13
**URL**: https://appadaycreator.com/english-word-spelling-game/

英単語のスペルを入力するタイピングゲーム。TOEIC頻出・英検レベル別300単語。タイマーモードとノーマルモードで英語力アップ！

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- Chart.js 4.4.0（グラフ表示）
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

- **FAQPage スキーマ拡充**: 5問の FAQ を追加（難易度、採点方法、タイマー制限、オフライン対応など）

## 前バージョン（v1.2.0）

### M4: グラフ・スコア・ゲージで結果を可視化
- **正答率ゲージ**: ドーナツチャートで現在の正答率を視覚化
- **最近のプレイ履歴**: 折れ線グラフで正答率の推移を表示

## 使い方

1. 記録したい項目を入力フォームに入力する
2. 「追加」または「記録」ボタンをクリックする
3. 記録一覧で過去のデータを確認する
4. グラフや集計で傾向を把握する
5. 継続的に記録して変化を追跡する

## よくある質問（FAQ）

**Q: スペルマスター | 英単語スペルゲーム - TOEIC・英検対応は無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。
