# 食品賞味期限管理ツール【無料】記録・管理アプリ - 技術仕様書

## 概要

**サービス名**: Expiry Date Tracker
**バージョン**: 1.1.0
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/expiry-date-tracker/

食品の賞味期限を登録して期限切れを防止。冷蔵・冷凍・常温別に管理。登録不要・完全無料でご利用いただけます。

## 更新履歴

### v1.5.0 (2026-06-19)
- **コンテンツ品質修正**: 注意事項・参考リンクが住宅系（リフォーム・不動産・DIY）の別サービスコンテンツになっていた誤りを修正し、食品賞味期限管理に関連する内容（食中毒予防・冷蔵庫温度・開封済み食品の注意、消費者庁・農林水産省・厚生労働省・環境省の関連リンク）に置き換え

- **GSC対応**: H1タグにキーワード（冷蔵・冷凍・常温、一括管理）を追加し上位表示狙い

- **コンテンツの一貫性改善**: テンプレート転用による誤りを修正
  - 「こんな方・シーンで役立ちます」セクション → 食品管理関連に全面更新
  - よくある質問（詳細）の内容を賞味期限・食品保存に関するFAQに変更
  - ツール説明文を「食品賞味期限管理」に特化した内容に修正
  - 「住まい・暮らしツール」の表記を「食品賞味期限管理ツール」に統一
- **削除ダイアログの改善**: より親切な確認メッセージ
  - 削除時に食品名・賞味期限・保存場所を表示する詳細な確認ダイアログを実装

### v1.2.0 (2026-06-10)
- **アクセシビリティ強化**: スクリーンリーダー対応の改善
  - 全入力フィールドに aria-describedby を追加し、ガイダンステキストが説明として読み上げられるように改善
  - 食品名・賞味期限・数量・保存場所・メモ各フィールドの説明が関連付けられた
- **保存場所ツールチップのUX改善**: モバイル対応
  - ❓アイコンをクリック可能なボタンに変更し、モバイル・タッチデバイスでも動作
  - クリック時に保存場所ごとの保存期間目安を表示するヘルプボックスを開閉
  - キーボード操作（Enter・Space）にも対応
  - aria-expanded と aria-controls 属性を追加してアクセシビリティを向上

### v1.1.0 (2026-06-10)
- **グラフ・ビジュアライゼーション機能**: Chart.jsで保存場所別・期限状態別の円グラフを実装
  - 保存場所別分布（冷蔵・冷凮・常温）を可視化
  - 期限状態別分布（OK・3日以内・期限切れ）をリアルタイム更新
- **入力フォーム改善**: 各入力欄にガイダンステキストとツールチップを追加
  - 食品名: 例を示す説明文
  - 賞味期限: 入力方法のコツを記載
  - 数量: 単位の自由度を説明
  - 保存場所: ❓アイコンでツールチップ表示（保存日数の目安）
  - メモ: 任意入力を明記
- **エクスポート・印刷機能**: 3つの出力方法を実装
  - 📥 CSVダウンロード: 外部ツールで分析可能
  - 📋 テキストコピー: タブ区切り形式でクリップボード保存
  - 🖨️ 印刷ボタン: 現在の記録一覧をA4サイズで印刷
- **フォームバリデーション**: 入力値をリアルタイム検証
  - 食品名・賞味期限が空の場合、追加ボタンを無効化

## 使い方

1. 記録したい項目を入力フォームに入力する
2. 「追加」または「記録」ボタンをクリックする
3. 記録一覧で過去のデータを確認する
4. グラフや集計で傾向を把握する
5. 継続的に記録して変化を追跡する

## よくある質問（FAQ）

**Q: 食品賞味期限管理ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。
