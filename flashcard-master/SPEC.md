## v2.4 (2026-06-29) - V1〜V5 コンバージョン直結UI修正 + M2 Ctrl+Enter / aria-invalid対応
**実装内容**:
1. **[V1] CTAボタンをソリッド化**：ゴーストボタン廃止→`background:#5b4dd3;color:#fff;font-size:18px;padding:16px 36px`のソリッドボタンに変更。ファーストビューでのクリック認識率向上。
2. **[V2] 全0スタッツ非表示**：デッキ未作成時に`stats-grid`を`display:none`にして「誰も使っていない」印象を排除。
3. **[V3] モバイルヘッダー1行化**：`@media(max-width:640px){#app-title{font-size:12px;white-space:nowrap;overflow:hidden;text-overflow:ellipsis}}`で4行折り返しを解消。
4. **[V4] 未取得バッジ非表示**：取得済みバッジのみ表示、未取得はレンダリングしない。未取得0件時は誘導メッセージを表示。
5. **[V5] 社会的証明テキスト追加**：CTAボタン直下に「完全ブラウザ動作 · データはあなたの端末にのみ保存 · 英検・TOEIC・資格試験対応」を追記。
6. **[M2] Ctrl+Enter ショートカット**：問題・答えのtextareaからCtrl+Enter(⌘+Enter)でカード追加可能に。aria-invalid属性をshowFieldError/clearFieldErrorで管理。

## v2.2 (2026-06-22) - M2段階2：入力UI・バリデーション最適化 + スクショ問題全解決
**実装内容**:
1. **[V1] トースト通知最小化**：エラーボックスを`padding:8px 12px`・`font-size:0.75rem`に縮小。max-width を 320px → 280px に。モバイルでは最大幅を280pxに制限。エラーが主要コンテンツをブロックしない設計に改善。
2. **[V4] アフィリエイトバナー低視認化**：固定下部バナーの背景色を`bg-white` → `bg-gray-50`に変更。グラデーション＆カラフルなボタンを撤廃し、テキストリンク化（`text-xs text-gray-600`）。スポンサー表記を削除し、プライバシー同意ボタンとしての視認性を低下。
3. **[V5] モバイルテキスト折り返し改善**：トーストに`word-wrap:break-word`・`overflow-wrap:break-word`・`white-space:normal`・`line-height:1.2`を追加。モバイルでエラーテキストが1文字単位で改行されない対応。
4. **[M2] フォーム入力UI統一**：全フォーム（新規デッキ・デッキ編集・カード追加・カード編集）のラベル構造を統一。文字数カウント表示をラベル右に移動（`flex justify-between`）。ヒントテキストを削除して最小化。placeholder テキストから文字数制限記載を削除して簡潔化。

## v1.0
- デッキを作成し、カードを追加して学習できます。
- 学習進捗や連続学習日数を表示します。
- JSON/CSV ファイルからカードをインポートできます。
- クリップボードの JSON データを貼り付けてカードをインポートできます。
- トップ画面から習得済みカードの一覧を確認できるボタンを追加しました。
- ヘッダーのボタン群が段落ちしないようスタイルを調整しました。
- カード一覧で上下の矢印ボタンによりカードの表示順を変更できます。
- `usage.html` に操作手順をまとめています。
