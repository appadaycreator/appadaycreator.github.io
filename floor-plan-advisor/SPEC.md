## 概要

**サービス名**: Floor Plan Advisor
**バージョン**: 2.2.0
**更新日**: 2026-06-14
**URL**: https://appadaycreator.com/floor-plan-advisor/

家族構成・ライフスタイル・予算から最適な間取りタイプ（1K〜4LDK以上）と注意点を診断。登録不要・完全無料でご利用いただけます。

## v2.2.0 変更内容（2026-06-14）

### 新機能（M6・M7施策）
- **M6: 目標設定・達成度トラッキング**: 希望する間取りタイプを目標として設定。診断結果が目標と一致した回数を3回まで追跡。達成度をプログレスバーで可視化。localStorage で永続化
- **M7: What-ifシミュレーション機能**: 1つの入力条件を変更して別のシナリオを即時シミュレーション。家族構成・テレワーク頻度・収納こだわり・予算を変更対象に設定可能。「現在の診断」「シミュレーション結果」を比較表示。ブラウザ内で動作（サーバー通信なし）

### UI・ユーザー体験の向上
- 目標セクションと What-if シミュレーションセクションを結果画面に統合
- 再診断・条件変更による継続利用を促進
- シミュレーション実行時に複数パターンの間取り可能性を可視化

## 診断ロジック仕様

### 入力（6問）

| Q | 内容 | インデックス |
|---|------|------------|
| Q1 | 家族構成 | 0=一人 / 1=2人 / 2=3人 / 3=4人以上 |
| Q2 | テレワーク頻度 | 0=出勤 / 1=週1-2 / 2=週3-4 / 3=フル |
| Q3 | 趣味・ライフスタイル | 0=シンプル / 1=料理重視 / 2=趣味部屋 / 3=来客多い |
| Q4 | 収納こだわり | 0=気にしない / 1=普通 / 2=たっぷり / 3=WIC必須 |
| Q5 | 将来予定 | 0=変わらない / 1=子増 / 2=同居 / 3=縮小 |
| Q6 | 月額予算 | 0=〜8万 / 1=8-12万 / 2=12-18万 / 3=18万+ |

### 判定ロジック

1. **基本スコア**: 家族構成インデックス（0〜3）
2. **将来加点**: 子ども増・同居予定で +1
3. **拡張ニーズ加点**: 趣味部屋/来客・収納こだわり・テレワーク多めの合算が2以上で +1
4. **一人テレワーク補正**: 一人暮らし＋テレワーク多め/趣味部屋 → 最低1LDKへ
5. **縮小予定**: futureが縮小なら -1
6. **スコアマッピング**: 0=1K, 1=1LDK, 2=2LDK, 3=3LDK, 4+=4LDK
7. **予算注釈**: 2LDK×〜8万円 / 3LDK×〜12万円 / 4LDK×〜18万円 で注釈表示

### 出力（間取りタイプ）

| キー | 間取り | 面積目安 |
|------|--------|--------|
| 1K | 1K・1R | 〜30㎡ |
| 1LDK | 1LDK・2K | 30〜50㎡ |
| 2LDK | 2LDK | 50〜70㎡ |
| 3LDK | 3LDK | 70〜90㎡ |
| 4LDK | 4LDK以上 | 90㎡〜 |

各タイプ: タイトル / サブタイトル / 概要説明 / こんな人に向いている（3点）/ 注意点（2点）/ 内見チェックポイント（3点）/ 参考予算ラベル

## 使い方

1. ページを開き、「診断スタート」ボタンをクリックする
2. 家族構成・テレワーク頻度・ライフスタイル・収納・将来予定・予算の6問に回答する
3. すべて回答すると推奨間取りタイプが自動表示される
4. 向いている点・注意点・内見チェックポイントを確認する
5. 結果をSNSでシェアしたり、条件を変えて再診断したりする

## よくある質問（FAQ）

**Q: 間取り選び診断は無料で使えますか？**

はい、完全無料でご利用いただけます。登録・ログイン・アプリのインストールも不要です。

**Q: 診断結果の精度はどのくらいですか？**

家族構成・テレワーク頻度・趣味・収納こだわり・将来予定・予算の6軸で総合判定します。あくまで参考目安としてご利用ください。

**Q: 途中で回答を間違えたら修正できますか？**

はい、各質問画面の「← 戻る」ボタンで前の質問に戻って回答し直せます。

**Q: 診断結果は保存されますか？**

結果はブラウザには保存されません。スクリーンショットやSNSシェアでご記録ください。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、個人情報・入力データのサーバー送信は行いません。
