  - 「CSV でダウンロード」ボタン（診断結果をCSV形式で出力）
  - 「テキストをコピー」ボタン（結果サマリーをクリップボードへ）
  - 「PDF で印刷」ボタン（window.print()で印刷UI呼び出し）
- Print CSS 更新: 歴史・エクスポートボタンを非表示、スコア・グラフは表示
