# 家具配置アドバイザー【無料】無料診断・アドバイス - 技術仕様書

## 概要

**サービス名**: Furniture Layout Advisor
**バージョン**: 1.3.0
**更新日**: 2026-06-13
**URL**: https://appadaycreator.com/furniture-layout-advisor/

部屋のサイズと家具サイズを入力して最適な配置を提案。動線・採光も考慮。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元・診断結果の自動保存
- **クリア**: ブラウザのサイトデータ削除で初期化
- **履歴機能**: 過去3件の診断結果を保存・表示

## 新機能（バージョン 1.4.0）

### V1: CTA ボタン視認性向上
- 「今すぐ診断」ボタンのサイズ拡大（padding 15px→18px、font-size 17px→18px、font-weight 800→900）
- min-height 56px で最小タッチ領域確保（WCAG 2.1 AAA 対応）
- シャドウ効果強化でボタンの視覚的優位性向上

### V2: モバイル応答性改善
- 小型スクリーン（480px以下）で CTA ボタンを全幅表示（flex-direction: column）
- 英雄画像の高さを最適化（max-height: 50vh）で縦長歪みを防止

### V3: 広告領域の視覚的区別
- AdSense 広告に背景色（#fef3c7）と枠線（#D97706）を追加
- 「📢 スポンサード」ラベルで広告領域を明確化

### M8: 入力ガイダンス充実
- パターン比較フォーム全入力項目に aria-label 属性を追加
- スクリーンリーダー対応を強化し、ユーザーガイドを詳細化

## 新機能（バージョン 1.3.0）

### M7: 比較・シミュレーション機能
- 複数の部屋サイズ・ライフスタイル条件を組み合わせてパターンを作成
- 作成したパターンを「パターン比較・シミュレーション」セクションに保存
- テーブル形式で複数パターンの診断結果を横並びで比較表示
- 各パターンの最適な家具配置タイプ・スコアを一覧表示
- 「もし〜だったら」シミュレーションで最適なレイアウト案を探索可能
- localStorage で比較パターンを永続化

## 新機能（バージョン 1.2.0）

### M6: 目標設定・達成度トラッキング機能
- 診断結果後、4つの配置スタイル（ミニマル・くつろぎ・仕事対応・収納重視）から目標を選択可能
- 設定した目標を localStorage に保存
- 結果画面に「目標達成度」をプログレスバー＋パーセンテージで表示
- 次回診断時に目標に対する達成度を自動計算・表示
- 「目標をリセット」ボタンで目標を変更可能
- ユーザーの継続利用と改善意欲を促進

## 既存機能（バージョン 1.1.0）

### M3: localStorage を使用した診断結果の自動保存機能
- 診断完了後、結果と質問への回答を自動保存
- 結果画面に「診断履歴（直近3件）」を表示
- 過去の診断結果をクリックして詳細を再表示可能

### M4: Chart.js を使用した結果の可視化
- 4つのタイプスコア（ミニマル・くつろぎ・仕事対応・収納重視）を円グラフで表示
- スコア分布を直感的に可視化
- レスポンシブデザインでモバイルにも対応

### M10: 印刷・結果エクスポート機能
- 「結果を印刷」ボタン: window.print() で印刷用フォーマット表示
- 「結果をダウンロード」ボタン: テキストファイルとしてダウンロード可能
- 診断日時・タイプ・説明・アドバイスを含めて出力

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、診断の説明を確認する
2. スタートボタンをクリックして診断を開始する
3. 表示される質問に順番に回答する（約1〜2分）
4. すべて回答すると診断結果が自動表示される
5. 結果のアドバイスを確認してSNSでシェアする

## よくある質問（FAQ）

**Q: 家具配置アドバイザーは無料で使えますか？**

はい、完全無料でご利用いただけます。登録・ログイン・アプリのインストールも不要です。

**Q: 何回でも診断できますか？**

はい、何度でもご利用いただけます。条件を変えて繰り返し診断してみてください。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてのデバイスに対応しています。

**Q: 診断結果は保存されますか？**

はい、診断結果はブラウザの localStorage に自動保存されます。結果画面に「診断履歴」として過去3件の診断が表示されます。スクリーンショット・SNSシェア・ダウンロード機能でも結果を記録できます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、個人情報・入力データのサーバー送信は行いません。
