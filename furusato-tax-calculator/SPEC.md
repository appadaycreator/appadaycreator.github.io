# 仕様書（ふるさと納税限度額計算ツール）

## 目的
ユーザーが年収や家族構成を入力するだけで、ふるさと納税の控除上限額を簡単に把握できるようにする。

## 対象ユーザー
- ふるさと納税をこれから始めたい人
- 今年の控除上限額を手早く知りたい人

## 画面構成
### 1. 診断ページ（SPA）
- ルート: `/`
- 役割: 入力フォームと診断結果の表示

### 2. LP（ランディングページ）
- ルート: `/lp.html`
- 役割: サービス紹介と診断ページへの導線

## ルーティング仕様
- GitHub Pages の静的ホスティングを前提に、LP から診断ページへは `/` に遷移する。
- `index.html` を直接指定しないことで、SPA ルートの `/` が正しく表示される。

## 非機能要件
- 静的ホスティング環境（GitHub Pages）で動作すること
- ブラウザのみで完結し、サーバーサイド処理を必要としない
