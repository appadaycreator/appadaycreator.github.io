# ゲームプレイ時間管理ツール【無料】記録・管理アプリ - 技術仕様書

## 概要

**サービス名**: Game Time Tracker
**バージョン**: 1.1.0
**更新日**: 2026-05-29
**URL**: https://appadaycreator.com/game-time-tracker/

ゲームタイトル別のプレイ時間を記録して月間・年間の統計を可視化。登録不要・完全無料でご利用いただけます。

## 機能一覧（v1.2.0）

| 機能 | 説明 |
|------|------|
| タイマー計測 | ゲームタイトルを入力してHH:MM:SS形式でリアルタイム計測 |
| セッション自動保存 | 停止時にlocalStorageへ自動保存（10秒未満は保存しない） |
| 今月の統計 | 月間総プレイ時間・セッション数・最多プレイゲームを表示 |
| ゲーム別集計 | 全期間/今月/今年でタイトル別累計時間をグラフ表示 |
| セッション履歴 | 直近20件の履歴と個別削除機能 |
| 一括削除 | 全データをワンクリックで削除 |
| オートコンプリート | 過去入力したゲームタイトルをdatalistで補完 |
| ダークモード | OSの設定を自動検出、手動トグル可能（設定を保存） |
| データエクスポート | CSV/JSON形式でセッション履歴をダウンロード可能 |

## データ管理

- **ストレージキー**: `game_time_tracker_v1`
- **保存形式**: JSON配列（セッションごとにid/title/startTime/endTime/durationを保持）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: 「すべて削除」ボタンまたはブラウザのサイトデータ削除で初期化

## データ構造

```json
[
  {
    "id": "1716123456789",
    "title": "モンスターハンターワイルズ",
    "startTime": "2026-05-29T21:00:00.000Z",
    "endTime": "2026-05-29T22:35:00.000Z",
    "duration": 5700
  }
]
```

## 使い方

1. 「ゲームタイトル」欄にプレイするゲーム名を入力する
2. 「計測開始」ボタンを押してタイマーをスタートする
3. ゲームを終えたら「計測停止」ボタンでセッションを保存する
4. 「ゲーム別プレイ時間」で累計時間をグラフで確認する
5. 「今月の統計」で月間の傾向を把握し、ゲームバランスを見直す

## よくある質問（FAQ）

**Q: ゲームのプレイ時間データはどこに保存されますか？**
すべてのデータはお使いのブラウザのlocalStorageに保存されます。サーバーには一切送信されません。

**Q: 記録したセッションを削除できますか？**
はい、各セッションの「削除」ボタンから個別に削除できます。全データは「すべて削除」ボタンで一括削除可能です。

**Q: 複数デバイスでデータを同期できますか？**
いいえ、localStorageはデバイス・ブラウザごとに独立しています。同期には対応していません。

**Q: スマートフォンでも使えますか？**
はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: プレイ時間の記録件数に制限はありますか？**
ブラウザのlocalStorage容量（通常5MB）の範囲内であれば制限なく記録できます。
