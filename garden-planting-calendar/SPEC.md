# 家庭菜園・植付けカレンダー【無料】無料Webツール - 技術仕様書

## 概要

**サービス名**: Garden Planting Calendar
**バージョン**: 1.2.0
**更新日**: 2026-06-14
**URL**: https://appadaycreator.com/garden-planting-calendar/

地域・栽培したい野菜から最適な植付け時期と育て方スケジュールを自動生成。登録不要・完全無料でご利用いただけます。

## 更新履歴

### v1.2.0 (2026-06-10)
- **M3実装**: localStorage を活用した検索履歴機能を追加
  - ユーザーの過去20件の検索を自動保存
  - 過去の結果をワンクリックで復元可能
  - 履歴クリア機能を実装
- **M4実装**: Chart.js を使用した年間タイムライン可視化を追加
  - 月別の植付け・育成・収穫期間を視覚的に表示
  - レスポンシブ対応のバーチャート
- **M10実装**: 印刷・エクスポート機能を追加
  - 🖨️ 印刷ボタン（window.print()）
  - 📋 コピーボタン（Clipboard API）
  - 💾 CSV ダウンロード機能
- **M6実装**: 目標設定・達成度トラッキング機能を追加
  - 栽培目標の設定と進捗管理（localStorage保存）
  - 達成率の自動計算と進捗バー表示
  - 完了した目標の履歴保存と前回最高達成率との比較表示

### v1.1.1 (2026-06-04)
- **アクセシビリティ改善**: 野菜削除ボタンに `aria-label` 属性を追加し、スクリーンリーダー対応を向上

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## よくある質問（FAQ）

**Q: 家庭菜園・植付けカレンダーは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: 結果を保存・共有できますか？**

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
