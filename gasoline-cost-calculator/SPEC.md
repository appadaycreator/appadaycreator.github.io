# ガソリン代計算【無料】給油費用・燃費・距離から自動計算 - 技術仕様書

## 概要

**サービス名**: Gasoline Cost Calculator
**バージョン**: 1.3.0（2026-06-10 UX・機能改善）
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/gasoline-cost-calculator/

走行距離・燃費・ガソリン価格を入力して給油費用を無料計算。高速代・旅行費用の見積もりにも。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 記録したい項目を入力フォームに入力する
2. 「追加」または「記録」ボタンをクリックする
3. 記録一覧で過去のデータを確認する
4. グラフや集計で傾向を把握する
5. 継続的に記録して変化を追跡する

## 最新の改善内容（v1.3.0）

### UX改善・機能追加
- **入力フィールドへの Placeholder 統一化**: すべての入力欄（月間走行距離・年間走行距離・燃費・ガソリン単価など）に具体的な入力例を表示
- **ツールチップ・ガイダンステキスト追加**: 各入力欄の label 横に `?` アイコンを配置し、ホバーで説明文を表示（M8 対応）
- **印刷・エクスポート機能**: 計算結果を「印刷」「コピー」「CSVダウンロード」の3つの方法で保存可能（M10 対応）
  - 「印刷」→ window.print() で印刷用ビューを表示
  - 「コピー」→ Clipboard API でテキスト形式のコピー
  - 「ダウンロード」→ CSV ファイルをローカルに保存

## 最新の改善内容（v1.2.0）

- **チャート見出し色バグ修正**: `color:#a0c4ff`（白背景で見えない）→ `color:#0369a1` に修正（月別推定・車種比較の両チャート）

- **プリセット価格ボタンで自動再計算**: 計算結果表示中にプリセット価格を変更すると即座に再計算

## よくある質問（FAQ）

**Q: ガソリン代計算は無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。

**Q: 計算履歴は何件まで保存されますか？**

最新10件のみ保存されます。11件目を追加すると、最も古いデータが自動削除されます。
