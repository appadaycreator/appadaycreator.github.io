使い方ガイド

1. 新しい贈り物を記録する

  1. ホームページの「新しい贈り物を追加」ボタンをクリックします
  2. 贈り先の名前を入力します
  3. 贈り物の名前を入力します
  4. 贈った日を選択します（デフォルトは今日の日付）
  5. 思い出やメモがあれば記入します
  6. 画像があればアップロードします（オプション）
  7. 「保存」ボタンをクリックして完了です

2. 贈り物リストを確認する

ホームページに記録した贈り物が一覧表示されます。最新の贈り物から順番に表示され、以下の情報が含まれます：

  • 贈り物の名前
  • 贈り先の名前
  • 贈った日
  • 思い出・メモ
  • 画像（アップロードした場合）

3. 贈り物を削除する

各贈り物カードの「削除」ボタンをクリックすると、その贈り物を削除できます。削除前に確認ダイアログが表示されます。

4. データの管理

データのエクスポート

記録したデータをJSONファイルとしてエクスポートできます。これにより、データのバックアップや他のデバイスへの移行が可能です。

データのインポート

エクスポートしたJSONファイルをインポートすることで、データを復元できます。

5. オフライン機能

Gift MemoryはPWA（Progressive Web App）として動作するため、以下の機能が利用できます：

  • オフラインでの使用
  • ホーム画面への追加
  • プッシュ通知（対応ブラウザのみ）

6. プライバシーについて

Gift Memoryは完全にローカルで動作するアプリケーションです。記録したデータはお使いのブラウザのIndexedDBに保存され、外部サーバーに送信されることはありません。

7. 対応ブラウザ

以下のブラウザで動作します：

  • Google Chrome（推奨）
  • Microsoft Edge
  • Firefox
  • Safari

8. トラブルシューティング

データが表示されない場合

ブラウザの設定でIndexedDBが無効になっている可能性があります。設定を確認してください。

画像が表示されない場合

画像ファイルのサイズが大きすぎる可能性があります。5MB以下の画像ファイルを使用してください。

アプリが動作しない場合

ブラウザを最新版に更新するか、キャッシュをクリアしてください。
