  • ✓ テスト通知: 設定画面からテスト通知を送信可能

技術的な実装:

  • • useNotifications.js: 通知ロジックの管理（許可リクエスト、スケジュール計算、通知送信）
  • • NotificationSettings.vue: 通知設定画面（オン/オフ、時刻設定、テスト送信）
  • • NotificationBanner.vue: 未設定ユーザーへの誘導バナー
  • • Service Worker: バックグラウンド通知の送信とクリックハンドリング
  • • localStorage: 通知設定と最終チェック日時の保存

カレンダービュー機能

水やりスケジュールを月間カレンダー形式で視覚的に管理。複数の植物の水やり予定を一覧表示し、過去の記録も確認できます。

主な機能:

  • ✓ 水やり予定マーク: 各日付に水やりが必要な植物の数をバッジで表示（期限切れは赤、通常は青）
  • ✓ 複数植物の一覧表示: 1日に複数の植物の水やり予定がある場合も、全て表示
  • ✓ 過去記録の確認: 前月・次月の移動ボタンで過去の水やり履歴を確認可能
  • ✓ 今日の予定表示: カレンダー下部に今日の水やり予定植物を一覧表示
  • ✓ 月間統計: 今月の水やり予定数、期限切れ数、管理中の植物数を表示
  • ✓ 植物詳細への遷移: 植物カードをクリックで詳細ページへ遷移

技術的な実装:

  • • useCalendar.js Composable: カレンダーロジック、水やり予定計算、月間統計
  • • CalendarView.vue: 月間カレンダーUI、前月/次月ナビゲーション、今日ボタン
  • • date-fns活用: startOfMonth, endOfMonth, eachDayOfInterval等でカレンダー生成
  • • 水やり予定計算: last_watered_at + watering_frequency で次回水やり日を算出
  • • スヌーズ対応: snoozed_untilが設定されている植物を除外して表示
  • • レスポンシブ対応: モバイル・デスクトップ両方で最適な表示
  • • テストカバレッジ100%: 全ての関数を網羅的にテスト

📱 モバイル対応:

  • • 小さい画面でも見やすい日付セルデザイン
  • • タップしやすいボタンサイズ（最小44×44px）
  • • スワイプジェスチャーで前月/次月移動（予定）
  • • ボトムナビゲーションから簡単アクセス

  • • PlantEncyclopediaView.vue: 図鑑一覧画面、タブ切り替え、統計表示
  • • PlantEncyclopediaCard.vue: 植物カード表示、お気に入りボタン
