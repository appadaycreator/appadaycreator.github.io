# 健診結果の見方【無料】血液検査・健康診断の数値を解説 - 技術仕様書

## 概要

**サービス名**: Health Check Explainer
**バージョン**: 1.4.0 (2026-06-13)
**更新日**: 2026-06-13
**URL**: https://appadaycreator.com/health-check-explainer/

健康診断・血液検査の結果数値の意味を分かりやすく解説。気になる項目の改善方法も提案。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 新機能（v1.4.0）

### M7: 比較・シミュレーション機能
- **3パターンシミュレーション表示**: 現在値・改善後・未改善シナリオの健康スコア比較
- **スコア差分表示**: 改善シナリオとの比較により「あと何ポイント向上できるか」を明示
- **視覚的な3色分類**: 現在値（黄）・改善後（緑）・未改善（赤）で直感的に理解可能
- **改善動機の可視化**: 「改善したらスコアが XX ポイント上がる」という具体的な目標提示

### GSC 構造化データ・メタ情報の最適化
- **メタタイトル最適化**: 主要キーワード「血液検査」「数値解説」を前に配置
- **メタ description 強化**: 「基準値との比較・改善アドバイス」など検索意図に合わせた改善
- **既存スキーマ確認**: FAQPage・HowTo・WebApplication・Organization・BreadcrumbList スキーマが実装済み

## 新機能（v1.3.0）

### M4: Chart.js による結果ビジュアル強化
- **健康スコア（0-100）の表示**: ゲージメーター風のスコア表示で全体的な健康状態を一目で把握
- **相対比較チャート**: 複数の検査項目を横棒グラフで相対比較し、数値の高低を視覚的に理解
- **正規化された表示**: すべての項目を0-100%の範囲で正規化し、異なる基準値の項目も比較可能に

### M8: 入力フォームのツールチップ・ガイダンス充実
- **ツールチップアイコン（?）**: 各入力フィールド横に「?」アイコンを追加。ホバーで詳細説明を表示
- **詳細説明テキスト**: 各項目に「朝測定推奨」「空腹時測定」など実践的なガイダンス
- **単位明記**: placeholderに「例) 120 mmHg」と単位を明記して入力例を具体化
- **入力バリデーション**: 異常に高い／低い値の入力時に警告メッセージを表示

## 新機能（v1.2.0）

### 1. バグ修正: 計測履歴の20件キャップ
- `saveToHistory` 内の `const history` を `let history` に修正（再代入エラーで21件以上保存されていたバグ）

### 2. 計測履歴のインラインUI表示
- `alert()` による履歴表示を廃止
- フォーム上部の折りたたみ式パネルで履歴を一覧表示
- 「この値を再入力」ボタンで過去の測定値をフォームに復元可能

### 3. 数値ビジュアルゲージ（各結果カード）
- 各検査項目のカードに低値・正常・要注意・要改善の4段階を色帯で表示
- 現在値の位置を縦バーマーカーで表示
- 自分の数値が基準値のどこに位置するか一目で把握可能

### 4. 中性脂肪（TG）・γ-GTPの入力項目追加
- 中性脂肪（TG）: 正常 50-149 mg/dL
- γ-GTP: 正常 10-50 U/L
- 異常値に応じた肝臓ケアサプリの推奨リンクも動的表示

### 5. 結果表示後のスクロール位置修正
- 「数値を解説する」実行後に `window.scrollTo(0,0)` でトップへ戻す

## 新機能（v1.1.0）

### 1. スマートフォン最適化入力フロー
- 各フィールドに正常範囲を表示
- `inputmode` 属性で数値キーボード自動表示
- タッチターゲット 48px 以上確保

### 2. 計測履歴管理
- localStorage で最新 20 件の履歴を自動保存
- 「計測履歴を見る」ボタンで過去の測定値を確認
- 日付付きで履歴を管理

### 3. 結果のシェア機能
- 📋 **結果をコピー**: 診断結果をテキスト化してクリップボードにコピー
- 𝕏 **Twitter シェア**: 結果と URL を自動入力して共有
- **LINE シェア**: LINEで友人・医師と共有

### 4. 動的な推奨グッズ表示
- 異常値に基づいて Amazon リンクを動的に表示
  - 高血圧 → 血圧計を推奨
  - 高血糖 → 血糖測定器を推奨
  - 高コレステロール → DHA/EPA サプリを推奨
  - 肥満 → 体重管理グッズを推奨

### 5. 関連ツール・サービスの適切化
- 不適切な「老後資金」「ふるさと納税」「ペット選び」を削除
- 健診関連ツール「BMI 管理」「サプリメント診断」「メンタルヘルス」「睡眠管理」に統一

## 使い方

1. ページを開き、入力フォームの項目を確認する（正常範囲を参考に）
2. 必要な情報を入力する（自動数値キーボード表示）
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じて以下を活用:
   - 📋 結果をコピー（テキスト化）
   - 𝕏/LINE でシェア（SNS共有）
   - 📊 計測履歴から過去の測定値を確認
