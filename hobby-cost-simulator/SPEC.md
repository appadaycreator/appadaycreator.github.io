# 趣味別コスト・費用シミュレーター | 年間コストと初期費用を計算 - 技術仕様書

## 概要

**サービス名**: Hobby Cost Simulator
**バージョン**: 1.2.0
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/hobby-cost-simulator/

趣味の種類と頻度から年間コストと初期費用を計算。50種類以上の趣味に対応。月予算内で続けられる趣味の提案とコスト削減ポイントも紹介。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

### v1.3.0（2026-06-10）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## よくある質問（FAQ）

**Q: 趣味別コスト・費用シミュレーター | 年間コストと初期費用を計算は無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: 結果を保存・共有できますか？**

スクリーンショットでの保存、SNSシェア、または CSV ダウンロード・印刷機能からご共有いただけます。

**Q: 前回の入力値は保存されますか？**

はい。入力値とシミュレーション結果は自動的にブラウザに保存され、ページを再度訪問した際に復元されます。
