## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **履歴保存**: 最新10回分の診断結果を自動保存
- **永続化**: ページ読み込み時に自動復元（前回結果を復元可能）
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- **Chart.js** - グラフ表示とデータビジュアライゼーション
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 主な機能

### 診断機能
- 5問の診断質問で ホテル/Airbnb の適合度を算出
- スコアバーで横並び比較表示

### グラフビジュアライゼーション（v1.1.0+）
- **比較グラフ**: 現在の診断結果を棒グラフで表示
- **推移グラフ**: 複数回診断時に折れ線グラフで推移を表示

### 履歴管理（v1.1.0+）
- 診断結果を localStorage に自動保存（最新10回分）
- ページ再訪問時に「前回結果を復元」ボタンを表示
- 過去の診断結果を選択可能（日付表示付き）

### 目標設定・達成度トラッキング（v1.2.0+）
- ユーザーが「Airbnb向き」「ホテル向き」「バランス型」から目標を選択
- 毎回の診断で進捗をプログレスバーで表示
- 前回診断との改善度を自動計算・表示
- 目標達成時にバッジ表示

### 結果エクスポート（v1.2.0+）
- **プリント**: window.print() 対応で印刷ダイアログ表示
- **テキストコピー**: 診断結果をクリップボードにコピー
- **CSV ダウンロード**: 全診断履歴を CSV 形式でダウンロード

## 使い方

1. ページを開き、診断の説明を確認する
2. スタートボタンをクリックして診断を開始する
3. 表示される質問に順番に回答する（約1〜2分）
4. すべて回答すると診断結果が自動表示される
5. 結果のアドバイスを確認してSNSでシェアする

## よくある質問（FAQ）

**Q: ホテルvsAirbnb比較診断は無料で使えますか？**

はい、完全無料でご利用いただけます。登録・ログイン・アプリのインストールも不要です。

**Q: 何回でも診断できますか？**

はい、何度でもご利用いただけます。条件を変えて繰り返し診断してみてください。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてのデバイスに対応しています。

**Q: 診断結果は保存されますか？**

はい。最新10回の診断結果がブラウザに自動保存されます。プリント・テキストコピー・CSVダウンロードで共有・記録できます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、個人情報・入力データのサーバー送信は行いません。

**Q: 目標を設定するにはどうすればいいですか？**

診断結果ページで「🎯 目標を設定する」ボタンをクリックし、「Airbnb向き」「ホテル向き」「バランス型」から選択してください。その後の診断で自動的に進捗が追跡されます。

**Q: 診断履歴をダウンロードできますか？**

はい。結果ページの「⬇️ CSV ダウンロード」ボタンで全診断履歴を CSV 形式でエクスポートできます。
