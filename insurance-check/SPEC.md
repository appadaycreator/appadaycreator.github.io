## 1. ビジネス要件

### 1.1 サービス概要
保険見直し診断は、生命保険・医療保険・学資保険が現在の生活状況に合っているかを10問で診断し、過不足を判定する無料Webアプリ。

### 1.2 収益モデル
| チャネル | 単価 | 状態 |
|---------|------|------|
| Google AdSense | インプレッション課金 | 審査中 |
| A8.net 保険コンパス | 12,000円/相談 (EPC 1,057) | 審査中 |
| A8.net マネードクター | 11,279円/面談 (確定率67%) | 審査中 |
| A8.net 保険見直し本舗 | 15,384円/相談 (EPC 312) | 審査中 |
| Amazon アフィリエイト | 購入額の数% | 設置済み |

### 1.3 KPI目標（月次）
- 月間セッション数: 500以上
- 診断完了率: 70%以上
- アフィリエイトCVR: 2%以上
- 月間収益目標: 10,000円以上

### 1.4 ターゲットユーザー
- 30〜50代の既婚者・子育て世帯
- ライフイベント後（結婚・出産・住宅購入）の見直し検討者
- 保険料を減らしたいが何を基準に判断すれば良いか分からない人

## 3. 要件定義

### 3.1 診断フロー設計
```
質問1: 年齢 (20-29/30-39/40-49/50-59/60以上)
質問2: 家族構成 (独身/既婚・子なし/子あり)
質問3: 年収 (〜300/300-500/500-800/800以上 万円)
質問4: 現在加入している保険種類 (複数選択)
質問5: 月間保険料 (〜1/1-3/3-5/5-10/10以上 万円)
質問6: 持ち家か賃貸か
質問7: 大病・手術歴の有無
質問8: 子どもの学資準備状況
質問9: 老後資金準備状況
質問10: 保険の見直し経験
```

### 3.2 判定ロジック
- スコア算出: 各回答に重み付けポイント付与
- 判定区分:
  - 0〜30点: 不足（保険が足りない可能性）
  - 31〜70点: 適切
  - 71〜100点: 過剰加入（保険料削減余地あり）

### 3.3 SEO要件
- メタdescription: 150文字以内
- h1タグ: 1個（キーワード含む）
- FAQPage構造化データ: 4問以上
- AggregateRating: 評価数78以上
- canonical: 設定済み
- sitemap.xml: 設定済み
