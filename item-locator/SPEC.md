## 2. 機能詳細仕様

### 2.1 アイテム管理機能

#### 2.1.1 収納場所の階層構造
```
リビング
├── 本棚
│   ├── 1段目
│   ├── 2段目
│   └── 3段目
├── テレビ台
│   ├── 引き出し1
│   └── 引き出し2
└── 収納ボックス
    ├── 衣類
    └── 小物
```
収納場所は最大3階層まで管理でき、ツリー画面から詳細場所の編集・削除が可能です。
詳細場所は「詳細場所管理」ページで追加・並び替えができます。
各収納場所には任意でアイコンを設定でき、収納マップ上のボックスに表示されます。

#### 2.1.2 アイテム登録フロー
1. 親カテゴリ選択/作成
2. 詳細位置入力
3. アイテム情報入力
   - 必須項目：アイテム名、詳細位置
   - 任意項目：メモ
4. 登録完了
5. 収納マップから場所を選択した場合は、表示領域の「アイテム追加」ボタンで即座に登録フォームを表示

#### 2.1.3 データ検証
- アイテム名：1-50文字
- メモ：0-200文字
- 親カテゴリ：1-30文字
- 詳細位置：1-30文字

### 2.3 データ管理

#### 2.3.1 Supabase保存構造（正規化テーブル）
```javascript
// 3つの正規化されたテーブルを使用
{
  'rooms': [
    // 収納部屋情報
    { id: 'uuid', name: 'string', user_id: 'uuid', created_at: 'timestamp' }
  ],
  'locations': [
    // 収納場所情報
    { id: 'uuid', name: 'string', room_id: 'uuid', parent_id: 'uuid', user_id: 'uuid', created_at: 'timestamp' }
  ],
  'items': [
    // アイテム情報
    { id: 'uuid', name: 'string', room_id: 'uuid', location_id: 'uuid', user_id: 'uuid', created_at: 'timestamp' }
  ]
}
```

#### 2.3.2 データバックアップ
- JSON形式でのエクスポート

#### 2.3.3 Supabase接続手順（新設計）
1. [Supabase](https://supabase.com/)でプロジェクトを作成し、URLとAnonキーを取得します。
2. 以下の3つの正規化テーブルを作成します。
   - `rooms` (収納部屋)
     - `id` UUID PRIMARY KEY
     - `name` TEXT NOT NULL
     - `user_id` UUID REFERENCES auth.users(id)
     - `created_at` TIMESTAMPTZ DEFAULT now()
   - `locations` (収納場所)
     - `id` UUID PRIMARY KEY
     - `name` TEXT NOT NULL
     - `room_id` UUID REFERENCES rooms(id)
     - `parent_id` UUID REFERENCES locations(id)
     - `user_id` UUID REFERENCES auth.users(id)
     - `created_at` TIMESTAMPTZ DEFAULT now()
   - `items` (アイテム)
     - `id` UUID PRIMARY KEY
     - `name` TEXT NOT NULL
     - `room_id` UUID REFERENCES rooms(id)
     - `location_id` UUID REFERENCES locations(id)
     - `user_id` UUID REFERENCES auth.users(id)
     - `created_at` TIMESTAMPTZ DEFAULT now()
3. `script.js` および `account.js` 内の `SUPABASE_URL` と `SUPABASE_KEY` を取得した値に置き換えます。
   ```javascript
   const SUPABASE_URL = 'https://xxxx.supabase.co';
   const SUPABASE_KEY = 'public-anon-key';
   const supabaseClient = supabase.createClient(SUPABASE_URL, SUPABASE_KEY);
   ```
4. 読み込み例:
   ```javascript
   const { data } = await supabaseClient
     .from('items')
     .select('*, rooms(name), locations(name)')
     .eq('user_id', user.id);
   ```
5. 書き込み例:
   ```javascript
   await supabaseClient
     .from('items')
     .insert({ name: 'item_name', room_id: 'room_uuid', location_id: 'location_uuid', user_id: user.id });
   ```
