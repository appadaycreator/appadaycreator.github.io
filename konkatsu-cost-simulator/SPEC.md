# 婚活費用計算【無料】マッチングアプリ・結婚相談所 総額いくら？ - 技術仕様書

## 概要

**サービス名**: Konkatsu Cost Simulator
**バージョン**: 1.4.0
**更新日**: 2026-06-13
**URL**: https://appadaycreator.com/konkatsu-cost-simulator/

婚活方法・期間・交際費用を入力して総費用を無料計算。方法別の費用比較と節約アドバイス付き。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 更新履歴

### v1.5.0（2026-06-13）

#### 追加機能

1. **目標設定・達成度トラッキング 強化版（M6フェーズ2実装）**
   - 🎯 **目標達成度ダッシュボード**：プログレスバー付きで達成率を視覚的に表示
   - 💡 **達成アクションプラン**：目標達成に必要な月額削減額・期間短縮を自動計算・提示
   - 🎛️ **目標プリセットボタン**：30万円・50万円・100万円の目標を1クリックで設定
   - ⚡ **リアルタイムフィードバック**：目標予算入力時に即座に達成可能性を判定
   - 📊 **詳細達成情報**：現在値・目標値・超過額・必要削減額を一元表示

### v1.4.0（2026-06-13）

#### 追加機能

1. **目標設定・達成度トラッキング基盤（M6実装）**
   - 💡 目標予算を設定して、達成度をリアルタイム表示
   - 🎯 計算結果と目標の乖離を「達成率 %」で可視化
   - 📊 前回の計算結果との比較表示（費用差分・増減率）
   - 💾 計算履歴を localStorage に保存（最新10件まで）
   - ✨ 継続利用促進：目標達成時は「✅ 達成」、未達時は「⚠️ 改善」と視覚的フィードバック

   - 📋 結果をクリップボードにコピー → テキスト形式で簡単共有
   - 🖨️ ブラウザ印刷機能（window.print()）→ PDF保存対応
   - 📄 テキストファイルダウンロード → 計算結果を `.txt` で保存

   - `window.lastResult` グローバル変数に結果データを保存
   - エクスポート・シェア機能で利用

### v1.2.0（2026-06-06）

#### 修正・追加

1. **入力バリデーション追加**
   - 月数: 1〜60ヶ月の範囲外は計算を止めてエラーメッセージ表示
   - デート費: 0〜100万円、ファッション費: 0〜50万円の範囲チェック
   - 赤枠エラーボックスをボタン直下に表示

2. **費用表示精度の修正（バグ修正）**
   - 1万円未満の費用（例: アプリ3ヶ月=9,000円）が「0万円」と表示されていたバグを修正
   - `formatCost()` 関数を導入: 1万円未満は「X,XXX円」、以上は「X.X万円」表示

3. **グラフの長期間対応**
   - 12ヶ月以下: 縦棒グラフ（月別表示）
   - 13〜24ヶ月: 横棒グラフ（2ヶ月集計）
   - 25ヶ月以上: 横棒グラフ（四半期集計）
   - グラフ高さをグループ数に応じて自動調整

4. **localStorage による入力値の保存・復元**
   - 計算実行時に4つの入力値を localStorage に保存
   - ページ読み込み時（DOMContentLoaded）に自動復元

5. **婚活方法別 費用比較表**
   - 結果セクションに全4方法（アプリ・パーティ・複合・相談所）の総費用比較テーブルを追加
   - デート費・ファッション費は現在の入力値で統一し、サービス費のみ各方法で計算
   - 現在選択中の方法をハイライト表示

### v1.1.0（2026-06-05）

#### 追加機能

1. **モバイル最適化の強化**
   - ボタン・入力欄の最小サイズを 48px / 44px に統一
   - パディングを増加させ、タッチ操作性を向上

2. **節約アドバイスの高度化**
   - 月別の費用効率を業界ベンチマークと比較
   - 「業界平均より 25% 低い」「効率的」などの評価を追加
   - 方法別（アプリ・パーティ・相談所・混合）に専用アドバイスを表示

3. **月別費用推移グラフ**
   - Chart.js を使用した積み上げ棒グラフを追加
   - サービス費・デート費・ファッション投資を月別に可視化
   - 初期費用と月額費用を区別して表示

4. **結婚相談所の複数社費用比較テーブル**
   - 5 大相談所（ツヴァイ・パートナーエージェント・IBJメンバーズ・オーネット・サンマリエ）の費用を一覧表示
   - 初期費用・月会費・12 ヶ月計の費用を可視化
   - 相談所選択時のみ表示

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## よくある質問（FAQ）

**Q: 婚活費用計算は無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: 結果を保存・共有できますか？**

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
