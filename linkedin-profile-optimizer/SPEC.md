# SNSプロフィール最適化診断【無料】無料Webツール - 技術仕様書

## 概要

**サービス名**: Linkedin Profile Optimizer
**バージョン**: 1.3.0
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/linkedin-profile-optimizer/

職歴・スキル・目的からLinkedIn・Wantedly等のプロフィール改善ポイントを診断。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 新機能（v1.1.0）

### P1 実装内容（フェーズ1）
- **📋 結果コピーボタン**: チェックリスト全文をクリップボードにコピー、2秒トースト表示

- **モバイルUI改善**: タッチサイズ 48px 以上、フォーム間距離拡大、チェックボックス 18px

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 業界・職種・経験年数・目的を選択（前回の選択が自動復元される）
3. 「最適化チェックリストを生成」ボタンをクリック
4. 表示されたチェックリストを確認・チェック（完成度スコアがリアルタイム更新）
5. 「📋 結果をコピー」でテキストコピー、または「🖨️ 印刷」で紙に印刷
6. SNS シェアボタン（X / LinkedIn / LINE）で共有
