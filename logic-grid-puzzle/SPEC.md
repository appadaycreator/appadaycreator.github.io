# 論理グリッドパズル - 完全無料で使えるWebツール【登録不要】 - 技術仕様書

## 概要

**サービス名**: Logic Grid Puzzle
**バージョン**: 1.1.0
**更新日**: 2026-06-07
**URL**: https://appadaycreator.com/logic-grid-puzzle/

ヒントから答えを推論する論理パズル。8問収録（やさしい〜むずかしい）。登録不要・完全無料でご利用いただけます。

## バージョン履歴

### v1.2.1 (2026-06-10)
- 【M4】グラフレスポンシブ対応：モバイル端末での2列グリッドを単列に変更
- 【M4】チャートUIの完全動作確認と最適化完了
- マイスコアダッシュボードのモバイル最適化

### v1.2.0 (2026-06-10)
- 【M4】グラフ可視化機能追加：クリア状況の円グラフ、難易度別クリア率の棒グラフ
- 【M4】マイスコアダッシュボード：クリア数・クリア率の統計表示
- 【M6】目標設定・達成度トラッキング機能追加
  - 定型目標（全8問クリア / 3日以内 / 毎日1問以上）とカスタム目標に対応
  - クリア進捗の可視化（プログレスバー）
  - クリア日時の自動記録・表示（直近5件）
- Chart.js ライブラリ導入（CDN）
- localStorage 拡張：`logic_clear_history` でクリア日時を記録

### v1.1.0 (2026-06-07)
- 問題数を3問→8問に増加（やさしい×2・ふつう×4・むずかしい×2）
- 問題8に複数ヒントを組み合わせる本格論理推論問題を追加
- 答え合わせロジックを修正（余計な○も検出するよう厳密化）
- 不正解時に各人の正誤フィードバック（✓/✗）を表示
- 正解時のアニメーション演出（カード pulse + 絵文字フロート）
- クリア進捗バー追加（X/8問クリア、クリア済み問題に✓マーク）
- 「使い方」をパズル特化の内容に更新

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## よくある質問（FAQ）

**Q: 論理グリッドパズルは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: 結果を保存・共有できますか？**

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
