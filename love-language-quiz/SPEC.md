# 愛情表現スタイル診断 - 完全無料で使えるWebツール【登録不要】 - 技術仕様書

## 概要

**サービス名**: Love Language Quiz
**バージョン**: 1.6.0
**更新日**: 2026-06-19
**URL**: https://appadaycreator.com/love-language-quiz/

5つの愛情表現（言葉・行動・贈り物・時間・ふれ合い）から自分のスタイルを診断。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## 変更履歴

### v1.4.0（2026-06-14）
- 【M6実装】目標設定・達成度トラッキング
  - 各スタイルの目標値を設定可能（スライダーで1-5段階）
  - 目標値をlocalStorageに保存（`loveLanguageGoals`）
  - 達成度ゲージを可視化（現在のスコア / 目標値）
  - 前回との比較表示（差分を色付け表示：改善時は緑、低下時は赤）
- 【M7実装】比較・シミュレーション機能
  - 結果画面に「診断結果」「比較・シミュレーション」タブを追加
  - 複数パターンを並べて比較するテーブル機能
  - スライダーで各スタイルの値を0-5の範囲で変更可能
  - パターン名を付けて保存・比較できる機能
  - localStorageに複数パターンを保存（`loveLanguageSimulations`）

### v1.3.0（2026-06-10）
- 【M3実装】データ保存・履歴機能（localStorage）
  - 診断結果をブラウザの localStorage に自動保存（最新5件まで保存）
  - 結果画面に「診断履歴」セクションを追加（前回以降の診断日時を表示）
  - ブラウザを閉じても前回の結果が復元される
- 【M4実装】グラフ・スコア・ゲージで結果を可視化（Chart.js）
  - Chart.js を統合し、5つの愛情表現タイプのスコアをレーダーチャートで可視化
  - 結果画面に 280px 高さのグラフ領域を追加
  - 各タイプのスコア分布を直感的に理解可能に

### v1.2.0（2026-06-03）
- 診断結果に「あなたへのアクション」セクションを追加（各タイプごとに3つの実践的アドバイス）
- シェアボタンをビジュアル強調（背景色ピンク・枠線・説明文追加「📱 結果をシェアして友人と比べてみよう」）

  - 通常表示に戻すボタンで復帰可能
