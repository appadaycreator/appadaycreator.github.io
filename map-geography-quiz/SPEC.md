# 世界地図・地理クイズ - 完全無料で使えるWebツール【登録不要】 - 技術仕様書

## 概要

**サービス名**: Map Geography Quiz
**バージョン**: 1.3.2
**更新日**: 2026-06-13
**URL**: https://appadaycreator.com/map-geography-quiz/

世界各国の首都・国旗・位置を学べる地理クイズゲーム。社会科学習にも活用。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## v1.3.1（2026-06-13）での改善

### 新機能
- **複数目標シミュレーション（M7実装）**: 結果画面に「複数目標シミュレーション」テーブル追加。実績に対して 3点・4点・5点という複数の目標での達成度を並べて比較表示

### 改善点

## v1.3.0（2026-06-10）での改善

### 新機能
- **スコア履歴管理（M3実装）**: localStorage を使用した最近5ゲーム分のスコア・正答率の自動保存・表示
- **グラフ・スコア可視化（M4実装）**: Chart.js を活用したスコア推移・正答率の折れ線グラフを結果画面に表示
- **印刷・エクスポート機能（M10実装）**: 結果の印刷、テキストコピー、CSV ダウンロード機能を実装
- **正答率の可視化**: ゲーム終了時に「正答数/総問数（正答率%）」を表示

### 改善点
- localStorage のスコア履歴を最大10ゲーム記録
- 4ゲーム以上の履歴があれば Chart.js でグラフ表示
- CSV ダウンロード時はブラウザのダウンロードフォルダに自動保存
- エクスポート操作後、2秒間のステータスメッセージを表示
