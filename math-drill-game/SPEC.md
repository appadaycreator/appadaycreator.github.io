算数・数学をゲーム感覚で楽しく学習！小学・中学・高校レベルに対応。タイムアタックや連続正解ストリーク機能で楽しみながら力がつく！

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. レベル（小学生・中学生・高校生）を選ぶ
2. モード（タイムアタック60秒 or じっくり30問）を選ぶ
3. 「🚀 スタート！」ボタンを押してゲームを始める
4. 表示された計算問題に答えを入力してEnterまたは「✅ 回答する」ボタンを押す
5. ゲーム終了後に結果画面でスコア・正答率・ランキングを確認する

## よくある質問（FAQ）

**Q: マスドリル - 算数・数学ドリルゲーム | 小学〜高校レベル対応は無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。
