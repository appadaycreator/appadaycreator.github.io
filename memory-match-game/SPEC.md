# メモリーマッチ | 神経衰弱ゲーム - 都道府県・動物・英単語テーマ - 技術仕様書

## 概要

**サービス名**: Memory Match Game
**バージョン**: 2.2.0
**更新日**: 2026-06-13
**URL**: https://appadaycreator.com/memory-match-game/

カードをめくってペアを合わせる記憶力ゲーム。都道府県・動物・果物・英単語・国旗の6テーマ。難易度3段階で脳トレ！

## 更新履歴

### v2.3.1 (2026-06-20)
- **V1: インフォボックスコントラスト改善**: ルール説明の青いボックス内テキスト色を `#1e40af` → `#003d82` に濃色化・フォント太さを500→600に変更
- **V2: モーダル背景透明度調整**: ルールモーダル背後が見える透明度 `bg-opacity-50` → `bg-opacity-20` に調整
- **V3: ルール説明スクロール最適化**: モーダル内の規則説明文がスクロール不要になるよう max-height を `70vh` → `85vh` に拡大、内容エリアに `flex-1 overflow-y-auto` を適用
- **M8: 難易度ボタンのツールチップ追加**: 初級(4×4/16枚)、中級(4×6/24枚)、上級(6×6/36枚)のツールチップを `title` 属性で実装

### v2.3.0 (2026-06-13)
- **M7: パターン比較シミュレーター完全実装**: リザルト画面に「テーマ・難易度選択 → シミュレート」ボタンを追加。複数パターンの条件変更で即座に予想スコアを計算・比較できる機能を実装（`runSimulation()` と `calculateSimulatedScore()` で任意の組み合わせのスコア予測を表示）

### v2.2.0 (2026-06-13)
- **M4: グラフ・スコア可視化**: Chart.js でスコア履歴をグラフ化・目標スコア表示機能を実装（`drawPerformanceChart()` で過去5回のスコア推移を線グラフで表示）
- **M6: 目標設定・達成度トラッキング**: localStorage に目標スコアを保存、達成率を自動計算・リザルト画面に表示（目標達成時に 🎯 表示）
- **M7: 比較・シミュレーション機能**: 難易度別スコア比較テーブル（`populateComparisonTable()` で初級・中級・上級の成績を並べて表示）
- **Phase 2 直接実装完了**: M4/M6/M7 の3施策すべて実装・統合確認完了

### v2.1.0 (2026-06-06)
- **バグ修正**: 新記録達成時の `highScore` 要素参照エラーを修正
- **バグ修正**: スタート画面のベストスコアが選択テーマに連動して表示されるよう修正
- **バグ修正**: Twitter Card説明文をゲーム内容に合致した内容に修正

- **機能改善**: 英単語テーマを英語カード⇔日本語カードのフリップ式学習形式に変更（`pairId` マッチング導入）

## よくある質問（FAQ）

**Q: メモリーマッチ | 神経衰弱ゲーム - 都道府県・動物・英単語テーマは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。
