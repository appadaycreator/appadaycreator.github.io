# 仕様書

## 機能概要
- 電子レンジのワット数を変換し、加熱時間を計算します。
- 計算履歴を LocalStorage に自動保存し、ページ読み込み時に表示します。履歴からよく使う変換をボタン表示します。
- 変換結果をコピーしたり、Twitter・LINE で共有できます。
- 計算結果に換算式（W × 秒 ÷ W = 秒）を表示します。
- PWA 対応のためオフラインでも動作します。

## ファイル構成
- `index.html` : メイン画面
- `main.js` : アプリケーションロジック
- `sw.js` : Service Worker
- `manifest.json` : PWA マニフェスト
- `contact.html`, `usage.html`, `privacy.html`, `terms.html`, `disclaimers.html` : 各種情報ページ

すべてのページで共通のヘッダーとフッターを使用し、フッターから各ポリシーや使い方ページへ遷移できます。

## 動作要件
- モダンブラウザ (Service Worker と LocalStorage をサポートしていること)
