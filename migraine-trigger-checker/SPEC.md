# 偏頭痛トリガー記録アプリ【無料】無料診断・アドバイス - 技術仕様書

## 概要

**サービス名**: Migraine Trigger Checker
**バージョン**: 1.1.0
**更新日**: 2026-06-04
**URL**: https://appadaycreator.com/migraine-trigger-checker/

偏頭痛の発症状況・トリガーを記録して傾向を分析。医師への相談にも役立つ記録ツール。登録不要・完全無料でご利用いただけます。

## v1.1.0 更新内容（2026-06-04）

### P1実装
- **診断結果ロジックの複数トリガー対応**: Q2（トリガー）・Q4（困ってること）を診断判定に組み込み
  - 「ストレス管理タイプ」「生活習慣改善タイプ」など複数パターンに拡張
  - 複合トリガーパターンに対応した診断メッセージを追加
- **前兆（前駆症状）判定の組み込み**: Q3「前兆」の回答が診断ロジックに反映
  - 前兆あり→「典型的な片頭痛」、なし→「群発頭痛の可能性」等の医学的分岐を追加

### P2実装
- **診断結果の印刷・共有機能拡張**: 「📄 診断レポート表示」ボタンを追加
  - 診断日時・全質問・回答・診断結果・医学的アドバイスを含むレポート表示
  - 印刷対応（レスポンシブレイアウト・医師向け資料として使用可能）
  - 免責事項を明記してセキュリティ・信頼性確保
- **モバイル UI 改善**:
  - クイズオプションボタン高さ 50px→56px、内部パディング調整で操作性向上
  - モバイル（<480px）での結果テキストフォントサイズ最適化（16px→13px）
  - ボタン最小高さ統一（48px）

## v1.2.0 更新内容（2026-06-14）

### M7実装: 比較・シミュレーション機能
- **複数パターン保存・比較機能**: 
  - 診断結果を最大3パターンまでlocalStorageに保存
  - 保存したパターンを一覧表示・選択可能

  - シミュレーション結果を別表示で確認可能
- **モード切り替えUI**:
  - 「通常診断」「比較・シミュレーション」の2つのモードを用意
  - 診断後に各パターンを柔軟に比較・分析可能
  - 複数条件を試してトリガー特定に活用

## 使い方

1. ページを開き、診断の説明を確認する
2. スタートボタンをクリックして診断を開始する
3. 表示される質問に順番に回答する（約1〜2分）
4. すべて回答すると診断結果が自動表示される
5. 結果のアドバイスを確認してSNSでシェアする

## よくある質問（FAQ）

**Q: 偏頭痛トリガー記録アプリは無料で使えますか？**

はい、完全無料でご利用いただけます。登録・ログイン・アプリのインストールも不要です。

**Q: 何回でも診断できますか？**

はい、何度でもご利用いただけます。条件を変えて繰り返し診断してみてください。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてのデバイスに対応しています。

**Q: 診断結果は保存されますか？**

結果はブラウザには保存されません。スクリーンショットやSNSシェアでご記録ください。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、個人情報・入力データのサーバー送信は行いません。
