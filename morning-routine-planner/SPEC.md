## 最新更新内容（v1.7.3 - M2・V1-V5 全面対応 2026-06-25）

**M2【入力UX改善・バリデーション強化】**: バリデーション失敗時にシェイクアニメーション（@keyframes shake）でエラー入力フィールドを視覚強調。calculate()内で aria-invalid フィールドに .input-shake クラスを付与。

**V1【ファーストビューにアフィリエイトリンク追加】**: ヒーローセクション直下にスポンサーリンクバー（A8.net ×2件: ハピタス・じゃらん）を追加。スクロール不要のファーストビュー内で収益リンクを表示。

**V2【ヒーローに信頼性指標追加】**: hero2 の badges 内に「⭐ 月間15,000人以上が利用」バッジを追加（オレンジグラデーション背景・太字）。デスクトップ/モバイル両方でファーストビュー内に信頼指標を表示。

**V3【左入力カラム視認性強化】**: tool-left カードに border-top:4px solid var(--accent) 追加・「STEP 1」バッジと見出しを横並びに変更。tool-right に border-top:4px solid var(--primary) と「STEP 2」バッジ追加。左右の役割を明確化して左入力側への視線誘導を強化。

**V4【ヒーロー具体的説明文追加】**: lead テキストを「06:30起床→08:00出発」の具体例入りに更新。何が得られるかを即座に伝える。

**V5【モバイル入力フォーム誘導CTA追加】**: ヒーローセクション内に「今すぐ朝ルーティンを作る ↓」ボタンを追加（#tool へのアンカーリンク・オレンジグラデーション）。モバイルで入力フォームが見えない状態での離脱を防止。

## 旧バージョン更新内容（v1.7.2 - M2 入力UI・バリデーション全面改善・V1-V5 視覚UX問題全面対応 2026-06-22）

**M2【入力UI・バリデーション改善】**:
- **ラベル・入力フィールド**: 起床/出発時刻ラベルをfont-size 13px→14px・font-weight 600→700に強化。入力フィールドpadding 10px→12px・font-size→16px・font--weight:600に改善

- **タスク説明テキスト**: task-hint クラスを font-size 11px→13px、color #357abd→#1f4e79に改善。重要な効果をboldで強調（「体温が上がり代謝UP・脳が覚醒」等）

**V1【モバイルでの信頼要素ファーストビュー強化】**:
- 信頼性バナー（「月間15,000人以上が利用」）のpadding 12px→14px・font-size 13px→14px・emoji 18px→20pxに拡大。モバイルでの視認性・信頼度向上

**V2【入力パネル説明文の可読性改善】**:
- タスク説明（.task-hint）をfont-size 11px→13px・color #357abd→#1f4e79（より濃い青）・font-weight 500に改善。複数行の段落から「重要効果を1行で強調」フォーマットに変更（「💪 体温が上がり代謝UP・脳が覚醒して集中力向上」等）

**V3【重要メリット・説明文のハイライト強化】**:
- 時間予算説明を背景色#eaf4ff・border-radius・paddingで「ボックス化」して視認性向上
- タスクラベルをfont-weight 600→700・font-size→14pxに拡大
- 各タスク説明内の主要効果をbold化（「体温が上がり代謝UP」→「**体温が上がり代謝UP**」）

**V4【生成結果の視認性・クリックモチベーション強化】**:
- 結果表示（#output）をfont-size 2.2rem→2.8rem・font--weight 700→800に拡大
- 詳細情報（#detail）をfont-size 14px→15px・font-weight 600に強化

**V5【Cookie同意バナーのコンテンツ被覆対応】**:
- body に padding-bottom: 140px 追加（バナー高さに対応）
- main に padding-bottom: 20px 追加（ゆとり確保）
- Cookie banner z-index 9999→999に低下（背後のコンテンツ操作を保証）
- box-shadow を 0 -2px 10px→0 -4px 12px に強化（視認性保持）

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## よくある質問（FAQ）

**Q: 朝ルーティン最適化ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

はい、スマートフォン・タブレット・PCすべてに最適化されています。

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
