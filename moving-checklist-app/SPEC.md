# 引越しToDoリスト・チェックアプリ【無料】無料Webツール - 技術仕様書

## 概要

**サービス名**: Moving Checklist App
**バージョン**: 1.6.0（2026-06-25 M2 UX強化版）
**更新日**: 2026-06-25
**URL**: https://appadaycreator.com/moving-checklist-app/

引越し前後の手続き・準備を時系列チェックリストで管理。スケジュール計算も自動化。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- Chart.js（v4）- 進捗・カウントダウン可視化
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## v1.4.1 改善点（2026-06-14 - M5確定版）

### M5: 結果に連動したアフィリエイト商品推薦（確定版）
- **動的推薦機能**: 家族構成（一人暮らし・カップル・ファミリー）＆引越し距離（近距離・中距離・遠距離）に基づいて関連商品を動的に表示
- **9パターン対応**: 3×3 マトリックス（家族構成3種×移動距離3種）で最適化された商品推薦
- **アフィリエイト統合**: 各推薦商品にはA8.netアフィリエイトリンク付属（既存px.a8.netリンク保持）
- **表示タイミング**: チェックリスト生成後、結果セクション直後に表示
- **重複排除**: フォーム再送信時に前回の推薦セクションを自動削除してから再生成
- **アクセシビリティ**: role="complementary"と aria-label 属性を追加
- **ブラウザ内動作**: 全機能はJavaScriptで実装（外部API通信なし）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## よくある質問（FAQ）

**Q: 引越しToDoリスト・チェックアプリは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: 結果を保存・共有できますか？**

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
