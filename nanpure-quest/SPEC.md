## v4.3.0 更新内容（2026-06-06）

### バグ修正
- **FAQテキスト色修正**: FAQ回答の `<dd>` テキスト色が `#d1d5db`（白背景に白文字）だったのを `#374151`（濃灰色）に修正。白背景での可読性を回復
- **モーダルのダークテーマ統一**: クリア・レベルアップ・攻略ヒント・キーボードショートカットの全4モーダルの内側コンテナ背景を `#f0f9ff`（薄水色）からダーク系グラデーション（`#1e1b4b→#2d2654`）に変更。モーダル内のテキスト（金色・薄紫・白）がRPGテーマに合致した背景で正しく読めるように修正。border-radius/paddingも適切に設定

- **保存ゲームのタイマー精度向上**: `saveGameState()` 呼び出し時点の経過時間を `savedElapsed` として保存。`startGameTimer(initialElapsed)` に引数を追加し、ゲーム再開時に前回の経過時間からタイマーを正確に継続できるように修正
- **BGMモバイルボタンの初期状態修正**: モバイルBGMボタンの初期表示を `🎵`→`🔇` に変更。localStorage 復元時も `ON/OFF` に合わせて正しいアイコンを表示

### アクセシビリティ

## v4.0.0 更新内容（2026-05-30）

### 品質改善
- **唯一解保証**: パズル生成時に `countSolutions(board, 2)` で解の数を検証。唯一解でない場合はセル削除を取り消して再試行。超難問（superhard）は処理速度優先でスキップ
- **ポーズ中の操作ロック**: `isPaused` フラグが true の場合、`inputNumber()` / `showHint()` / `undoMove()` で操作をブロックし、トースト通知を表示

### UX改善
- **誤入力の視覚フィードバック**: solution と不一致の数字を入力した際、セルに `.wrong-answer` クラス（赤背景＋シェイクアニメーション）を付与し 0.8秒後に自動除去
- **難易度別ベストタイム**: `gameStats.bestTimes[difficulty]` に難易度ごとのベストを保存。クリアモーダルに「NEW BEST！」または「この難易度のベスト: XX:XX」を表示
- **数字ボタンの残り個数バッジ**: 入力パッドの各数字ボタンに「残りN個」の小さなバッジを表示（完了時は「✓」）
- **コンフェッティ演出**: パズルクリア時に純CSS/JSの紙吹雪アニメーション（32個）を表示
- **キーボードショートカット拡充**: `M`=メモ / `H`=ヒント / `P`=ポーズ / `Esc`=モーダル閉じる / `?`=ショートカット一覧モーダル
- **ショートカット一覧モーダル**: ゲームコントロールに「⌨️ ショートカット」ボタン追加

### モバイル最適化
- **セルサイズ改善**: 480px以下でセルを `calc((100vw - 40px) / 9)` に拡大（最大 40px）
- sudokuGrid の padding を 4px に縮小し、盤面の実効サイズを拡大

## 使い方

1. 難易度を選択する（初心者〜超難問の5段階）
2. 9×9マスに1〜9を各行・列・3×3ブロックに重複なく入力
3. 「提出」ボタンで答え合わせ → 正解で敵にダメージ！
4. 敵を倒して経験値を集めレベルアップを目指す
5. 「⏸ 一時停止」でタイマーを止めて休憩できる

## よくある質問（FAQ）

**Q: 数独バトルRPGは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: スマートフォンで使えますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。

**Q: 途中で一時停止できますか？**

はい、一時停止ボタンで止めて、再開することができます。

**Q: カスタムの時間を設定できますか？**

はい、プリセット以外に任意の時間を直接入力して設定できます。

**Q: 時間終了時に通知はありますか？**

画面上でアラート表示します。ブラウザの設定によって音声が出る場合があります。
