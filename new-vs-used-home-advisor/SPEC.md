# 新築 vs 中古住宅 比較診断｜予算・条件から最適な住宅を提案 - 技術仕様書

## 概要

**サービス名**: New Vs Used Home Advisor
**バージョン**: 1.6.1
**更新日**: 2026-06-20
**URL**: https://appadaycreator.com/new-vs-used-home-advisor/

予算・条件から新築と中古の総コストを比較し最適な選択を提案。8つの質問でリフォームコスト・ローン・立地など総合的に比較する無料診断ツール。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する（各質問横の ℹ️ で補足説明を確認可能）
3. すべての質問に回答して結果を取得する
4. 表示された結果・分析・グラフを確認する
5. 必要に応じて「結果をコピー」ボタンで結果テキストをコピー、またはSNSで共有

## 新機能（v1.6）

### M7: 比較・シミュレーション機能 ✅ 実装完了
- **複数シナリオ保存**: 「📊 このシナリオを比較に追加」ボタンで診断結果を複数保存可能。保存時にカスタム名称を入力可能
- **比較テーブル**: 保存したシナリオを並べて比較する表を表示。項目別の並列比較で最適選択を支援

- **What-if シミュレーション**: 予算を上げ下げ、リフォーム意欲を変更して結果を即座に再計算
- **シミュレーションボタン**: 「📈 予算を上げる/📉 予算を下げる/🔨 リフォーム増/減」で直感的に条件変更
- **比較の永続化**: localStorage に複数シナリオを保存、ページ再読み込み後も復元
- **リセット機能**: 「🗑️ 比較リセット」で保存したシナリオをすべてクリア。確認ダイアログ付き

## 新機能（v1.5）

### M6: 目標設定・達成度トラッキング機能
- **マイ目標セクション**: 診断結果下部に「マイ目標・目標進捗」ボックスを追加
- **前回との比較**: 同じ条件で再診断すると「新築スコア」「中古スコア」「新築予算」の前回比較を表示
- **目標保存**: 「💾 目標として保存」ボタンで現在の結果を目標 localStorage に記録
- **目標リセット**: 「🗑️ 目標をリセット」ボタンで古い目標を削除可能
- **時系列追跡**: 目標保存時に timestamp を記録し、診断日時を表示

### M10: 印刷・結果エクスポート機能
- **印刷ボタン**: 「📄 印刷」ボタンで window.print() を実行、結果を PDF形式で保存
- **CSV ダウンロード**: 「📊 CSV」ボタンで診断結果・スコア・費用を CSV ファイルでダウンロード
- **テキスト エクスポート**: 「📋 テキスト」ボタンで結果をテキストファイル(.txt)でダウンロード
- **ファイル命名**: 診断日時を含むファイル名で自動保存（例: `新築vs中古診断結果_2026-06-10.csv`）
