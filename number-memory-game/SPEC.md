# 数字記憶ゲーム - 完全無料で使えるWebツール【登録不要】 - 技術仕様書

## 概要

**サービス名**: Number Memory Game
**バージョン**: 1.5.0
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/number-memory-game/

表示された数字列を記憶して入力するブレインゲーム。レベルが上がるほど長くなる。登録不要・完全無料でご利用いただけます。

## ゲーム仕様

### 基本フロー
1. 「ゲームスタート」ボタンをクリック
2. 数字がプレイ画面に表示される（レベル1は3桁、以降 +1桁ずつ）
3. 一定時間後に数字が消える
4. 入力欄に覚えた数字を入力して「答える」ボタンをクリック
5. 正解→スコア+1、レベル+1で次のラウンド開始
6. 不正解→ゲーム終了、最終スコア表示

### 難易度調整
- **表示時間**: レベルに応じて自動調整（Lv1: 2.5秒 → Lv20: 1.5秒）
- **桁数**: Lv1は3桁、以降1レベルで1桁追加
- **難易度スケール**: 適切な人間的な難易度を維持

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: スコアランキング（Top5）・連続プレイ日数（ストリーク）を保存
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## よくある質問（FAQ）

**Q: 数字記憶ゲームは無料で使えますか？**

A: はい、完全無料・登録不要でご利用いただけます。

**Q: スマートフォンで使えますか？**

A: はい、スマートフォン・タブレット・PCすべてでご利用いただけます。

**Q: ゲーム結果のデータは保存されますか？**

A: はい、ランキング（Top5）と連続プレイ日数（ストリーク）をブラウザに保存します。ブラウザのサイトデータを削除するまで記録が保持されます。

**Q: 数字が消える時間を調整できますか？**

A: いいえ、表示時間はレベルに応じて自動調整されます。難易度を段階的に上げるための設計です。

**Q: キーボードで答えを入力できますか？**

A: はい、数字を入力してEnterキーを押すと答え合わせできます。「答える」ボタンをクリックしても同じ結果が得られます。
