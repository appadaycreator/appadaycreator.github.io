子どもの年齢・性格・生活スタイルから最適な保育園・幼稚園・こども園の選び方を診断。保育料の目安と入園申込みのポイントもアドバイスします。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## v1.5.0 の改善（2026-06-13）

### M6: 目標設定・達成度トラッキング機能の実装
- **目標スコア設定パネル**: 診断結果画面に「推奨園のマッチスコア目標」設定セクションを追加
  - スライダーで 0-100 点の目標値を直感的に設定可能

- **達成度ビジュアル**: 目標パネルに以下を表示
  - 達成数 / 総診断数（例: 3/5）
  - 達成率パーセンテージ（プログレスバーで可視化）
  - 最新診断の結果と達成判定（✅達成 / 🎯未達）
- **継続利用促進**: ユーザーが複数回の診断を通じて目標達成を追跡でき、リピート利用動機を高める

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・比較表・アドバイスを確認する
5. 必要に応じてシェアボタンで SNS 共有、またはもう一度診断する

## よくある質問（FAQ）

**Q: 保育園・幼稚園選び診断 | お子さまに最適な園を無料診断 | AppADayCreatorは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: 結果を保存・共有できますか？**

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
