# PantryChef 仕様書

PantryChef は手持ちの食材からレシピを検索する PWA アプリです。主な機能は次のとおりです。

- 食材リストからの選択およびカスタム食材の追加
- 必ず使いたい食材を星マークで指定する機能
- 調理時間と調理方法による絞り込み検索
- レシピ一覧のソート、カテゴリフィルタ
- 週間献立の自動生成
- 買い物リスト、タスク管理
- オフライン対応のための Service Worker
- 使い方ガイド、プライバシーポリシー、免責事項の表示
  - 各ページ下部のフッターから上記ページへリンク

内部データは `js/recipes.js` にまとめられており、`data/recipes.json` から生成されています。
