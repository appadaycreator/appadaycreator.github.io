# 育児ストレス度チェック【無料】無料診断・アドバイス - 技術仕様書

## 概要

**サービス名**: Parenting Stress Checker
**バージョン**: 1.1.0
**更新日**: 2026-06-05
**URL**: https://appadaycreator.com/parenting-stress-checker/

子育ての悩み・睡眠不足・孤立感からストレス度を診断してセルフケア方法を提案。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、診断の説明を確認する
2. スタートボタンをクリックして診断を開始する
3. 表示される質問に順番に回答する（約1〜2分）
4. すべて回答すると診断結果が自動表示される
5. 結果のアドバイスを確認してSNSでシェアする

## よくある質問（FAQ）

**Q: 育児ストレス度チェックは無料で使えますか？**

はい、完全無料でご利用いただけます。登録・ログイン・アプリのインストールも不要です。

**Q: 何回でも診断できますか？**

はい、何度でもご利用いただけます。条件を変えて繰り返し診断してみてください。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてのデバイスに対応しています。

**Q: 診断結果は保存されますか？**

結果はブラウザには保存されません。スクリーンショットやSNSシェアでご記録ください。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、個人情報・入力データのサーバー送信は行いません。

## 変更履歴

### v1.2.0 (2026-06-10)
**M3・M4・M10改善実装**

#### M3: データ保存・履歴機能（localStorage）
- 💾 **自動保存**: 診断完了時に スコア・日時・レベルを localStorage に自動保存（最大5件）
- 📜 **履歴一覧**: 結果画面に過去の診断結果を小さいカード形式で表示
- 🔄 **個別削除**: 各履歴アイテムの「✕」ボタンで個別削除可能
- 🗑️ **一括削除**: 「履歴をすべて削除」ボタンでローカルストレージをリセット

#### M4: グラフ・スコア・ゲージで結果を可視化
- 📈 **折れ線グラフ**: 複数回の診断スコアを Canvas で折れ線グラフ表示（3回以上診断時）
- 🎯 **スコア推移**: グラフの各ポイントを色分け（現在のスコアを赤でハイライト）
- 📊 **変化インジケーター**: 診断スコアの傾向を表示（改善傾向↓ / 悪化傾向↑）
- 🔧 **外部ライブラリ不使用**: Canvas と純粋な JavaScript で実装

#### M10: 印刷・結果エクスポート機能
- 🖨️ **印刷ボタン**: `window.print()` で印刷プレビュー（ヘッダー・フッター・アフィリエイト非表示）
- 📋 **テキストコピー**: 「結果をコピー」ボタンで診断結果をクリップボードにコピー
- ✓ **コピー成功通知**: コピー実行時に画面右下にトースト通知表示

#### 技術変更
- localStorage API で診断履歴を永続化
- Canvas 要素で複数スコアの視覚化
- navigator.clipboard API でテキストコピー実装
- 既存の @media print ルールを拡張

### v1.1.0 (2026-06-05)
**P1・P2改善実装**

#### P1: UX向上
- ✨ **選択肢の色分け**: 各選択肢に段階的な色（緑→黄→オレンジ→赤）を付与し、ストレス度を視覚的に示唆
- 🎨 **ホバー効果改善**: セレクトボタンのホバー時にボーダー色と背景色が動的に変更
- 📱 **タッチフレンドリー**: min-height:48px で小型デバイスでも誤タップを削減

#### P2: 診断結果の充実
- 📊 **スコア表示**: 診断結果に 0-18 の点数をゲージバー形式で表示
- 🎯 **動的色付け**: スコアレベルに応じてタイトル色が変化（低め=緑、中程度=黄、高め=赤、非常に高い=紫）
- 📋 **セルフケア詳細化**: 各ストレスレベルごとに具体的で実践的な対策を箇条書きで表示
- ⚠️ **確認ダイアログ**: 再診断時に現在の結果を失うことを警告

#### 技術変更
- CSS カスタムプロパティ（`--opt-color`）で動的スタイリング
- `color-mix()` で色の透明度ブレンディング
- 結果オブジェクトに `score`・`color`・`detail` プロパティを追加
