# アルバイト収入・時給計算ツール【無料】簡単・無料計算 - 技術仕様書

## 概要

**サービス名**: Part Time Income Calculator
**バージョン**: 1.5.0
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/part-time-income-calculator/

時給・勤務時間・日数から月収・年収・所得税を計算。103万・106万・130万の壁もチェック。登録不要・完全無料でご利用いただけます。

## v1.5.0 変更点（2026-06-10）

### M4: グラフ・ビジュアライゼーション機能の実装
- **Chart.js 統合**: CDN経由で Chart.js (v4.4.0) を読み込み
- **収入構成グラフ（円グラフ）**: 手取り / 所得税 / 住民税 / 社会保険料の割合を可視化
  - 配色: 緑（手取り）/ 赤（所得税） / オレンジ（住民税） / インディゴ（社会保険料）
  - レスポンシブ対応: モバイル・デスクトップで見やすい表示
- **年収レベルゲージ（横棒グラフ）**: ユーザーの年収と 103万・106万・130万の「壁」を比較
  - 複数の横棒で壁の位置を視覚化
  - ユーザー年収を別色（青）で強調
- **印刷対応**: @media print でグラフも出力可能（Canvas → 印刷プレビューで表示）
- **自動レンダリング**: 計算実行時に setTimeout でグラフを動的に描画

### M10: 印刷・エクスポート機能の実装
- **🖨️ 印刷ボタン**: `window.print()` でブラウザの印刷プレビューを起動
  - 広告・シェアボタン・ナビゲーション等を非表示
  - 計算結果・グラフ・内訳は表示状態を維持
- **📥 CSVダウンロードボタン**: 計算データを CSV 形式でダウンロード
  - ファイル名: `部分時間給与計算_YYYYMMDD.csv`（例：`部分時間給与計算_20260610.csv`）
  - エンコーディング: UTF-8（Excelで文字化けなし）
  - 内容: 入力値（時給・日数・月数）＋ 計算結果（月収・年収・税金・保険料・手取り）
- **ボタン配置**: 計算結果の下部に 2 つのボタンを並列配置（モバイルではフレックスレイアウト）
- **タッチターゲット**: 44px 以上確保（モバイル環境でのタップ性向上）

### デザイン・UX改善
- 結果セクション内に視覚的な区切り（margin-top: 20px）を追加
- グラフ領域の見出し「📊 収入構成」「📈 年収レベルと各種『壁』」を明示
- ボタンカラーの選定:
  - 印刷: 紫系（#8b5cf6）
  - CSV: 青系（#06b6d4）
- レスポンシブ対応: `flex-wrap: wrap; flex: 1;` でモバイル時も見やすい

## v1.4.0 変更点（2026-06-06）

### シナリオ共有機能の完全実装
- **URLコピーボタン**: 計算結果表示時に「📋 URLをコピー」ボタンを表示
- **URLパラメータ形式**: `?h=1000&hd=5&dm=15&ym=12` 形式でシナリオを URL エンコード
- **自動復元**: 他人から受け取った URL でアクセスすると、自動的に計算結果が復元される
- **トースト通知**: コピー完了時に右下に「コピーしました！」と表示

### リセットボタンの追加
- 入力フォーム下部に「リセット」ボタンを配置（グレーカラー）
- クリック時、全ての入力をクリア＋ localStorage もクリア＋結果を非表示
- モバイルで44px × 44px以上のタッチターゲットを確保

### 計算内訳の詳細セクション（折りたたみ式）
- 結果表示時に「📊 計算内訳を見る」という詳細セクションを配置
- `<details>` タグで実装（ブラウザネイティブ）、デフォルト折りたたみ
- 展開時、以下の計算段階を視覚化：
  1. 年収の計算（時給×時間×日数×月数）
  2. 給与所得控除の金額
  3. 基礎控除（48万）
  4. 課税所得
  5. 所得税計算（課税所得×5%）
  6. 住民税計算（課税所得×10%）
  7. 社会保険料の計算式
  8. 手取りの計算（年収-税金-保険料）
- 各計算式をコード形式で視覚化

### 入力値バリデーション強化
- 時給: 100～50,000円の範囲チェック
- 勤務時間: 0.5～24時間の範囲チェック
- 勤務日数: 1～31日の範囲チェック
- 年間稼働月数: 1～12ヶ月のチェック
- 範囲外の入力時、入力フィールド下に赤いエラーメッセージを表示
- エラー時、入力フィールドのボーダーを赤色に変更

### モバイル入力体験の改善
- `inputmode="numeric"` / `inputmode="decimal"` を追加
- 各フィールドで最適なモバイルキーボード（数値キーボード）が表示される
- プレースホルダーをより具体的に（「例）1000」など）

## v1.3.0 変更点（2026-06-04）

### 入力値チェック・フィードバック強化
- 時給: 100～50,000円の範囲チェック（範囲外なら即座にエラー表示）
- 勤務時間: 0.5～24時間の範囲チェック
- 勤務日数: 1～31日の範囲チェック
- 各フィールドにエラーメッセージを動的に表示
- プレースホルダーをより具体的に（「例）1000円/時」等）

### シナリオ保存・共有機能
- **リセットボタン**: 全ての入力をクリア（localStorage も同時にクリア）
- **URLコピー機能**: 現在のシナリオを URL エンコード（querystring: `?h=1000&hd=5&dm=15&ym=12`）
- **URL パラメータ復元**: リンク経由でアクセスすると自動的に計算結果が表示される
- ページ読み込み時に URL パラメータを優先、localStorage をフォールバック

### 計算内訳の透明化
- 「計算内訳を見る」の詳細セクション（折りたたみ式）
- 給与所得控除 → 基礎控除 → 課税所得 → 税計算の段階を視覚化
- 各税種（所得税、住民税、社会保険料）の計算式を表示
- ユーザーが「なぜこの金額か」を理解できるように改善

### モバイル最適化（360px 対応）
- 360px 以下の小型画面で結果カードを 1 列表示
- タッチエリアを最小 44px × 44px に統一
- フォント・パディングの調整で読みやすさ向上

- **年間稼働月数の説明補強**: ラベル下部に「例）12ヶ月（通年）、6ヶ月（夏休み・冬休みのみ）」の例示を追加

  - 130万以上：年収の15.5%を社会保険料として計算

- **手取り計算精度向上**: 手取り = 年収 - 所得税 - 住民税 - 社会保険料（新）

## よくある質問（FAQ）

**Q: アルバイト収入・時給計算ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 計算結果は正確ですか？**

参考値としてご活用ください。重要な判断の際は専門家・公式機関にご確認ください。

**Q: 何回でも計算できますか？**

はい、回数制限なく何度でもご利用いただけます。条件を変えて比較にお使いください。

**Q: 入力したデータはどこかに保存されますか？**

いいえ。すべての計算はブラウザ内で処理され、データのサーバー送信は行いません。

**Q: スマートフォンで利用できますか？**

はい、PC・スマートフォン・タブレットすべてに対応しています。
