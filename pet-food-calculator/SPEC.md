# ペットフード給与量計算【無料】簡単・無料計算 - 技術仕様書

## 概要

**サービス名**: Pet Food Calculator
**バージョン**: 1.2.0
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/pet-food-calculator/

体重・年齢・運動量からペットの1日あたりの最適なフード給与量を計算。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ペットの種別（犬/猫）と年齢を入力する
2. ペットの体重とフードのカロリーを入力する
3. 活動レベルを選択する
4. 計算ボタンをクリックして結果を取得する
5. 計算結果と詳細な内訳（推奨給与量、食事回数など）を確認する
6. 「リセット」ボタンで初期化、または条件を変えて複数パターンを比較する

## v1.1.0 新機能（2026-06-04）

### 年齢・種別による計算精度向上
- ペット種別セレクト（犬/猫）の追加
- 年齢入力フィールドの追加
- 年齢に応じた代謝係数の自動調整：
  - 子ペット（0-1歳）: RER × 2.0
  - 成ペット（1-7歳）: RER × 1.2 + 活動係数
  - シニア（7歳以上）: RER × 1.1 + 活動係数

### 入力値の自動保存・復元
- localStorage により、入力値を自動保存
- ページリロード時に前回の値を自動復元
- 「リセット」ボタンで全値をクリア

### 入力値バリデーション強化
- 体重：0.1～100kg の範囲チェック
- 年齢：0～30歳の範囲チェック
- カロリー：50～600kcal/100g の範囲チェック
- 範囲外の入力時に警告メッセージを表示
- 無効な入力で計算ボタンが無効化される

### 詳細な給与ガイド表示
- 計算結果に「子犬/子猫期」「成犬/成猫期」「シニア期」などの段階を表示
- 推奨食事回数（子ペット3～4回、成犬/成猫2回、シニア2回）を表示
- 1食あたりの給与量を提示（実践的な分量）
- 1日の必要カロリーを併記

## よくある質問（FAQ）

**Q: ペットフード給与量計算は無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 計算結果は正確ですか？**

参考値としてご活用ください。重要な判断の際は専門家・公式機関にご確認ください。

**Q: 何回でも計算できますか？**

はい、回数制限なく何度でもご利用いただけます。条件を変えて比較にお使いください。

**Q: 入力したデータはどこかに保存されますか？**

いいえ。すべての計算はブラウザ内で処理され、データのサーバー送信は行いません。

**Q: スマートフォンで利用できますか？**

はい、PC・スマートフォン・タブレットすべてに対応しています。
