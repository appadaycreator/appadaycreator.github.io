# カメラ設定アドバイザー【無料】無料診断・アドバイス - 技術仕様書

## 概要

**サービス名**: Photography Settings Advisor
**バージョン**: 1.2.0 (Phase2実装版)
**更新日**: 2026-06-20
**URL**: https://appadaycreator.com/photography-settings-advisor/

撮影シーン・光量・被写体から最適なカメラ設定（F値・SS・ISO）を提案。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

### Phase 2実装（2026-06-20）

- **M3実装**: データ保存・履歴機能（localStorage）
  - 診断結果の自動保存（最大20件まで履歴保持）
  - 履歴一覧表示・ワンクリック復元
  - 履歴全削除ボタン（確認ダイアログ付き）

- **M4実装**: Chart.js による結果可視化
  - 推奨設定をレーダーチャートで直感的に表示
  - 各設定項目（絞り・SS・ISO）にプログレスバー・ゲージを追加
  - 0-100スケールで相対的な強度を可視化

- **M7実装**: シーン別比較・シミュレーション機能
  - 全シーン（10パターン）の推奨設定を棒グラフで一覧比較
  - 絞り・シャッター・ISOの各項目で切り替え可能
  - 水平バーチャートで複数シーンの差異を直感的に表示

- **UI改善（V1-V3対応）**
  - V1: ヒーロー領域テキストコントラスト向上（背景グラデーション強化・text-shadow改善）
  - V2: CTA（行動喚起）ボタンの優先度明確化（主CTA font-weight強化・セカンダリCTA opacity調整）
  - V3: モバイルレイアウト圧迫改善（パディング・マージン最適化・フォントサイズ調整・タッチ対象サイズ改善）

## 使い方

1. ページを開き、診断の説明を確認する
2. スタートボタンをクリックして診断を開始する
3. 表示される質問に順番に回答する（約1〜2分）
4. すべて回答すると診断結果が自動表示される
5. 結果のアドバイスを確認してSNSでシェアする

## よくある質問（FAQ）

**Q: カメラ設定アドバイザーは無料で使えますか？**

はい、完全無料でご利用いただけます。登録・ログイン・アプリのインストールも不要です。

**Q: 何回でも診断できますか？**

はい、何度でもご利用いただけます。条件を変えて繰り返し診断してみてください。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてのデバイスに対応しています。

**Q: 診断結果は保存されますか？**

はい。診断結果はブラウザのローカルストレージに自動保存されます。最大20件の診断履歴を復元できますが、ブラウザのキャッシュ削除で消去されます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、個人情報・入力データのサーバー送信は行いません。
