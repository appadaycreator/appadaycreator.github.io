# プレゼンタイマー【無料】発表時間・スライド時間管理 - 技術仕様書

## 概要

**サービス名**: Presentation Timer
**バージョン**: 1.3.0
**更新日**: 2026-06-20
**URL**: https://appadaycreator.com/presentation-timer/

プレゼンテーションの時間管理をサポートするシンプルなタイマー。スライド枚数から1枚あたりの目安時間を計算、残り時間をビジュアル表示。

## 変更履歴

- v1.3.0 (2026-06-20): M7比較・シミュレーション機能追加（複数パターン比較テーブル・タイマー適用）、V2ナビゲーションコントラスト改善、V3設定ボタン全幅化、V1Cookieバナー視認性向上

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 発表時間をプリセットボタン（3/5/10/15/20分）から選択するか任意の分・秒数を入力する
2. スライド枚数を入力すると1枚あたりの目安時間が表示される（省略可）
3. 「タイマースタート」ボタンをクリックする
4. 残り時間を円形プログレスバーと数字で確認しながら発表する（残り60秒・30秒で警告表示）
5. 一時停止/再開はボタンまたはSpaceキー、リセットはRキーで操作できる

- 印刷用スタイルシート最適化:
  - ヘッダー・フッター・ボタン非表示
  - 白背景・黒テキストで印刷適最適化

- CSV構成:
  ```

  エクスポート日時: YYYY年MM月DD日
  
  実行日時,設定時間(分),設定時間(秒),スライド枚数,1枚あたり目安(秒)
  2026/06/10 12:34,3,0,10,18
  ```

## よくある質問（FAQ）

**Q: プレゼンタイマーは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: スライド枚数から時間を計算できますか？**

はい、スライド枚数を入力すると1枚あたりの目安時間（分・秒）を自動計算して表示します。

**Q: 時間終了時に音が鳴りますか？**

画面上でアラート表示します。音声通知はブラウザのポリシー上、ユーザー操作なしでは再生できない場合があります。

**Q: プレゼン中に画面が暗くなりますか？**

ブラウザのタブをアクティブな状態にしておくことで、タイマーはバックグラウンドでも動作します。ただし画面ロックはブラウザ設定に依存します。

**Q: カスタムの時間（例：7分30秒）も設定できますか？**

はい、分・秒を個別入力して細かい時間設定が可能です。

**Q: 残り時間が少なくなると警告はありますか？**

残り60秒で黄色の点滅アニメーション、残り30秒で赤の連続点滅が表示されます。
