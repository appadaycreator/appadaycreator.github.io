# ゴミ分別・リサイクル診断【無料】無料診断・アドバイス - 技術仕様書

## 概要

**サービス名**: Recycling Advisor
**バージョン**: 1.1.0
**更新日**: 2026-06-04
**URL**: https://appadaycreator.com/recycling-advisor/

あなたのゴミ分別タイプを診断し、実践的なアドバイスを提供。自治体の違いにも対応。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## v1.2.0 の改善点（2026-06-14）

### M4: グラフ・スコア・ゲージで結果を可視化
- **スコアゲージ表示**: 診断結果を0-100%のプログレスバー（グラデーション）で可視化
- **スコア数値表示**: リサイクル意識度スコア（百分率）を見やすく表示
- **前回比較チャート**: 前回の診断との比較を差分表示（📈📉→）
- **診断履歴折れ線グラフ（Chart.js）**: 過去10回の診断スコアを折れ線グラフで可視化。Chart.js ライブラリを活用し、スコア推移をリアルタイムで表示
- **グラフデータ自動更新**: 新規診断後に履歴グラフが自動的に更新・再描画

### M14: SEO構造化データ（FAQPage）拡充
- **FAQPage スキーマの充実**: 5つのQ&A を含む FAQPage スキーマを実装
  1. 無料で使えるか
  2. スマートフォン対応
  3. 診断時間
  4. 診断結果の活用方法
  5. データ保存場所

### M6: 目標設定・達成度トラッキング
- **目標設定フォーム**: 5つのリサイクル改善目標から選択可能（プラスチック分別、フリマ活用等）
- **localStorage 保存**: 目標と設定日時を永続保存
- **達成度計算**: 目標設定以降の診断回数と目標スコア達成数を自動集計
- **進捗表示**: 診断回数、達成回数、達成率（%）を目標ステータスに表示

### M7: 比較・シミュレーション機能
- **シミュレーション機能**: 最後の質問を変更してリアルタイム結果プレビュー
- **結果タイプ切り替え**: 選択肢変更により「エコ優等生→意識向上中」等の結果変化を表示
- **スコア差分表示**: 現在の診断との点数差分を「⬆️⬇️→」で可視化
- **診断履歴表示**: 直近5回の診断結果を日付・タイプ・スコアで一覧表示
- **履歴の永続化**: 最大5件の診断履歴を localStorage に保存・表示

### 技術仕様
- **外部ライブラリ非依存**: Chart.js 等の外部CDN不使用、Pure CSS/JS による実装
- **データ永続化**: localStorage を活用した複数キー管理（recycling-results, recycling-goal等）
- **リアルタイム計算**: シミュレーション実行時に結果を即座に再計算・表示

## 使い方

1. ページを開き、診断の説明を確認する
2. スタートボタンをクリックして診断を開始する
3. 表示される質問に順番に回答する（約1〜2分）
4. すべて回答すると診断結果が自動表示される
5. 結果のアドバイスを確認してSNSでシェアする

## よくある質問（FAQ）

**Q: ゴミ分別・リサイクル診断は無料で使えますか？**

はい、完全無料でご利用いただけます。登録・ログイン・アプリのインストールも不要です。

**Q: 何回でも診断できますか？**

はい、何度でもご利用いただけます。条件を変えて繰り返し診断してみてください。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてのデバイスに対応しています。

**Q: 診断結果は保存されますか？**

結果はブラウザには保存されません。スクリーンショットやSNSシェアでご記録ください。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、個人情報・入力データのサーバー送信は行いません。
