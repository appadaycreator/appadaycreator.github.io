# リモートワーク環境・コスト診断 | テレワーク快適度と費用を無料チェック | AppADayCreator - 技術仕様書

## 概要

**サービス名**: Remote Work Advisor
**バージョン**: 1.4.0
**更新日**: 2026-06-19
**URL**: https://appadaycreator.com/remote-work-advisor/

在宅勤務・テレワークの環境と毎月のコストを7つの質問で診断。快適なリモートワーク環境構築に必要な機器・ツール・費用目安をアドバイスします。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## 更新履歴

### v1.3.0（2026-06-10）
**M4: グラフ・スコア・ゲージで結果を可視化**
- Chart.js ライブラリを導入（CDN版 4.4.0）
- 診断結果にレーダーチャートを追加（4カテゴリを視覚的に表示）
- PC環境・ネット・スペース・生産性を径線で表現
- モバイル・PCに最適化された chart サイズ（250px高）

**M10: 印刷・結果エクスポート機能**
- 「🖨️ 印刷」ボタン: window.print() で結果をPDF化可能
- 「📥 CSV」ボタン: 診断結果をCSVファイルでダウンロード
- CSVには総合スコア・カテゴリ別スコア・改善投資目安を含める
- BOM付きUTF-8で日本語文字化け対策

### v1.2.0（2026-06-03）
**P1: 利用者の声修正**
- キャリア・MBA関連の誤ったコンテンツを削除
- リモートワーク環境改善に関連した利用者の声に統一
- 鈴木さん（フリーランス）・橋本さん（リモートワーカー）・加藤さん（会社員）の声を追加

**P2: モバイル UI 最適化**
- 質問選択肢のタッチターゲットを拡大（p-3→p-4、gap-3→gap-4）
- ラジオボタンのサイズを明示的に設定（w-5 h-5）
- シェアボタンレイアウトを responsive 化（モバイル時は縦積み、sm以上で横並び）
- ボタンの最小高さを min-h-12 に統一

**P2: LocalStorage 永続化機能**
- 回答を localStorage に自動保存（'rwa_answers'・'rwa_step'）
- ページ読み込み時に自動復元 → 中断した診断を続行可能
- 「もう一度診断する」ボタンで localStorage をクリア

### v1.1.0（2026-06-03）
**P1: スコア透明化機能**
- スコア計算を4カテゴリに分解（PC環境・ネット・スペース・生産性）
- 各カテゴリ別スコア（0-25点）を結果画面に表示
- ユーザーが改善点を明確に認識可能

**P2: シェア機能強化**
- 「コピーしてシェア」ボタンで診断結果をテキスト化
- LINE・X（Twitter）に直接シェア可能
- 結果テキストに総合スコア・カテゴリ別スコア・最優先改善項目を含める
