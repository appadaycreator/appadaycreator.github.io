# 賃貸vs購入コスト比較 - 生涯コストを無料シミュレーション - 技術仕様書

## 概要

**サービス名**: Rental Vs Buy Advisor
**バージョン**: 1.6.2
**更新日**: 2026-06-21
**URL**: https://appadaycreator.com/rental-vs-buy-advisor/

年齢・年収・家族構成から賃貸と住宅購入の生涯コストを比較。ローン返済・管理費・売却想定価格を含めた総合判断を無料で診断します。

- **ストレージ**: ブラウザ localStorage（外部API通信なし）

- **クリア**: ブラウザのサイトデータ削除で初期化

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## v1.5.0 改善内容（2026-06-10）

### P1 実装（UX改善・機能追加）

1. **入力フォームへのplaceholder属性追加（M2）**
   - 6個のnumber input要素に`placeholder`属性を追加
   - 例：年齢→"35"、年収→"600万円"、家賃→"12万円"、金利→"1.5%"
   - ユーザーが期待される値の形式を一目で理解でき、入力ストレスを軽減
   - aria-label属性も追加してアクセシビリティを向上

2. **計算履歴の自動保存・復元機能（M3）**
   - 計算実行時に結果を`localStorage`の`rentalVsBuyHistory`に自動保存
   - 履歴形式：`{ timestamp, age, income, rent, price, rate, downpayRate, period, family, rentTotal, netBuyCost, verdict }`
   - 最大10件の履歴を保持（新しい順）
   - 結果セクション下に「📜 過去の計算履歴」セクションを表示
   - 履歴カードから復元ボタンをクリックで入力値を復元し、自動再計算
   - 履歴クリアボタンで確認ダイアログ付きで削除可能

3. **結果をテキスト形式でコピー（M10）**
   - 結果セクションに「📋 結果をコピー」ボタンを追加
   - 計算結果を整形テキスト形式でクリップボードにコピー
   - コピー内容：入力条件 + 計算結果 + URL
   - メール・チャット・ドキュメントへの貼り付けに対応

### 実装詳細

- **saveToHistory()**: 計算結果をlocalStorageに保存
- **displayHistory()**: ページロード時に履歴セクションを表示
- **restoreFromHistory()**: 履歴から入力値を復元して自動再計算
- **clearHistory()**: 履歴を全削除（確認ダイアログ付き）
- **copyResultText()**: 結果をテキスト形式でクリップボード出力
