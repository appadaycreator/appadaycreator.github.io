# 部屋のカラーコーディネート診断【無料】無料診断・アドバイス - 技術仕様書

## 概要

**サービス名**: Room Color Advisor
**バージョン**: 1.2.1
**更新日**: 2026-06-13
**URL**: https://appadaycreator.com/room-color-advisor/

好みの雰囲気・部屋の用途から最適な壁色・インテリアカラーを提案。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化
- **診断履歴**: 最新10件の診断結果を `room-color-history` キーで保存

## 実装済み機能（v1.2.0）

### M3: データ保存・履歴機能（localStorage）
- 診断完了時、スタイルキー・スタイル名・スコア・タイムスタンプを自動保存
- 結果ページに「診断履歴」セクションを表示
- 履歴アイテムをクリックで詳細表示
- 「履歴をクリア」ボタンで全削除可能

### M4: グラフ・ビジュアライゼーション（Chart.js）
- Chart.js ライブラリを CDN で読み込み
- 診断結果ページに横棒グラフを表示
- 4つのスタイル（ナチュラル/ノルディック/カフェ/シック）のマッチ度を%表示

### M7: 比較・シミュレーション機能（v1.2.1 検証・GSC対応）
- 「複数パターン比較」: 過去3件の診断結果を表形式で並べて比較表示（日時・スタイル・スコア）
- 「シミュレーション」: 5つの質問を再度入力して、異なるスタイルマッチ度をリアルタイムで確認
- 各モードから結果ページへ即座に戻れる「結果に戻る」ボタン
- **v1.2.1 改善**: メタディスクリプション・タイトルに「カラーシミュレーション」「内装配色」「色選びツール」キーワード追加
- **v1.2.1 改善**: FAQPage スキーマに比較・シミュレーション機能関連の Q&A を充実化

### M10: 印刷・エクスポート機能
- 「🖨️ 印刷する」: window.print()で結果をPDF化
- 「📋 テキストをコピー」: 診断結果をテキスト形式でクリップボードにコピー（navigator.clipboard API）
- 「💾 ダウンロード」: 診断結果をJSON形式でダウンロード

## 使い方

1. ページを開き、診断の説明を確認する
2. スタートボタンをクリックして診断を開始する
3. 表示される質問に順番に回答する（約1〜2分）
4. すべて回答すると診断結果が自動表示される
5. 結果のアドバイスを確認してSNSでシェアする

## よくある質問（FAQ）

**Q: 部屋のカラーコーディネート診断は無料で使えますか？**

はい、完全無料でご利用いただけます。登録・ログイン・アプリのインストールも不要です。

**Q: 何回でも診断できますか？**

はい、何度でもご利用いただけます。条件を変えて繰り返し診断してみてください。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてのデバイスに対応しています。

**Q: 診断結果は保存されますか？**

結果はブラウザには保存されません。スクリーンショットやSNSシェアでご記録ください。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、個人情報・入力データのサーバー送信は行いません。
