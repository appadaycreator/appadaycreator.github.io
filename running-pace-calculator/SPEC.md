# ランニングペース計算【無料】マラソン目標タイム・1kmペース換算 - 技術仕様書

## 概要

**サービス名**: Running Pace Calculator
**バージョン**: 1.6.0
**更新日**: 2026-06-30
**URL**: https://appadaycreator.com/running-pace-calculator/

距離・目標タイムを入力してkm/mileペースを無料計算。マラソン・ハーフ・5K対応。

## バージョン履歴

### v1.6.0 (2026-06-20) - UI・アクセシビリティ最適化（視認性・入力操作改善）
- **P1: 結果表示エリアのコントラスト強化**
  - `.result-item` 背景色を `rgba(98,0,234,0.15)` → `rgba(98,0,234,0.22)` に濃色化
  - ラベル・単位の色を `#d4b8f0` → `#e8dcff` に明るく変更
  - ペース値の text-shadow を `0 1px 4px` → `0 2px 6px rgba(0,0,0,0.6)` に強化
  - **結果**: 計算結果の数値（「10:55」など）の視認性を大幅向上、コントラスト比4.5:1以上を確保
  
- **P3: 時間入力フィールドのモバイル最適化**
  - `.time-input input` に `min-height: 44px` を追加してタップターゲット拡大
  - `.time-input span` のフォントサイズを `0.9rem` → `1rem` に拡大
  - 単位テキスト（「時間」「分」「秒」）の色を `#b39ddb` → `#d4b8f0` に明るく
  - モバイル端末（max-width:480px）でのグリッドレイアウト対応：3×2 のグリッド表示で視認性向上
  - **結果**: モバイルでの時分秒入力がしやすく、タップ誤操作を削減
  
- **P2: クッキーバナー（プライバシー通知）のUX改善**
  - バナー背景色を `#1e293b` → `#0f172a` に濃色化して視認性向上
  - パディングを `1rem 1.5rem` → `1.2rem 1.5rem` に拡大
  - ボタンのパディングを `.5rem 1.25rem` → `.6rem 1.4rem` に拡大、`min-height: 44px` を追加
  - ボタンに `flex-shrink: 0` を追加して常に見える位置に配置
  - ギャップを `.75rem` → `1rem` に拡大してレイアウト余裕を確保
  - **結果**: バナーが結果表示エリアと重ならず、「同意して閉じる」ボタンが常に操作可能に

  - 異なる目標タイムでの走行ペース差分を一目で確認可能に
- **M4: カロリー内訳チャート（ドーナツグラフ）の追加**

### v1.4.0 (2026-06-13) - M4/M8実装・GSC最適化・結果可視化強化
- **M4: SVGゲージ・チャート化による結果可視化**

  - 推定最大消費量に対する割合を進捗バーで表示
- **M8: 入力フィールドへのツールチップ・入力ガイダンス充実**

  - ホバー時にツールチップが表示され、各項目の説明が即座に確認可能
  - 各フィールド下に `input-hint` で具体的な入力例・単位を表示（例：「42.195kg（フルマラソン）」）
  - 初心者向けの入力補助が大幅に充実し、迷いなく入力可能に

  - ページタイトルに「消費カロリー」キーワードを追加
  - metaディスクリプションを「距離と目標タイムからランニングペース計算...」と明示的に書き換え、主要キーワードを盛り込み
  - og:title・og:description・twitter:* タグも同時に更新してSNS流通最適化

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

- HTML5 / CSS3 / Vanilla JavaScript

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## よくある質問（FAQ）

**Q: ランニングペース計算は無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: 結果を保存・共有できますか？**

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
