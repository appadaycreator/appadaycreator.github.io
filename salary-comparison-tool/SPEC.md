# 業界・職種別年収比較ツール【無料】無料Webツール - 技術仕様書

## 概要

**サービス名**: Salary Comparison Tool
**バージョン**: 1.0.2
**更新日**: 2026-06-21
**URL**: https://appadaycreator.com/salary-comparison-tool/

業界・職種・年齢・地域から年収の平均・中央値を比較。転職の参考に。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## UX改善 v1.0.2

- **[V1] スクロール固定CTA**: ページスクロール時に画面右下に「年収を比較する」ボタンを固定表示（結果表示時は非表示）
- **[V2] モバイルナビゲーション改善**: ナビゲーションを Flexbox で改善、フォントサイズ拡大（`.85rem` → `.9rem`）、パディング増加
- **[V3] 信頼性バッジ**: ナビゲーション下に「🔒 プライバシー保護」「📊 政府統計データ」「✅ 登録不要」の3つのバッジを表示

## 結果エクスポート機能（v1.0.1追加）

- **🖨️ 印刷**: ブラウザの印刷機能で結果を印刷
- **📋 コピー**: 結果をテキスト形式でクリップボードにコピー
- **💾 ダウンロード**: 結果をCSV形式でダウンロード（Excel等で開可能）

## よくある質問（FAQ）

**Q: 業界・職種別年収比較ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。
