## v1.8.0 (2026-05-31)

### 改善内容

#### バグ修正
- **タブボタンactiveクラスバグ修正**: タブ内の絵文字spanをクリックした場合に `event.target.closest('.tab-button')` でボタンを正確に取得するよう修正。アイコンをクリックしてもタブが正しくハイライトされるようになった
- **filterPracticeCategory二重レンダリング修正**: カテゴリフィルター切り替え時に `renderQuestionNavButtons()` と `loadPracticeQuestion()` が2回呼ばれていた問題を修正。難易度フィルターが不要な場合は1回で完結するよう最適化

#### テストモード改善
- **「採点」ボタン追加**: テストモードバナーに「✍️ 採点」ボタンを追加。未回答問題がある場合は確認ダイアログを表示してから採点
- **テストモード進捗表示**: バナーに「X/Y問回答済み」のリアルタイム進捗テキストと緑色プログレスバーを追加。回答するたびに即時更新

#### UX改善
- **過去問プログレスバー**: 問題番号ナビの下に細い進捗バー（青グラデーション）を追加。現在の問題位置を視覚化。問題を進めるごとに更新
- **ブックマーク数をフィルターボタンに表示**: ⭐ブックマーク(N)の形式でブックマーク件数をリアルタイム表示
- **弱点復習モーダルにアプリ内演習ボタン追加**: 「📝 アプリ内で苦手問題を今すぐ演習する →」ボタンを追加。クリックで過去問タブに遷移して苦手フィルターを自動適用
- **ライトテーマのモーダルフォーム視認性改善**: ライトテーマ切り替え時にモーダル内の入力フォーム・セレクトが白背景・黒文字で正しく表示されるよう修正

## 機能一覧

| 機能 | 概要 |
|------|------|
| 学習プラン自動生成 | 試験日・レベル・学習時間から序盤/中盤/直前フェーズのタスクを自動生成 |
| ダッシュボード | 学習進捗・連続記録・今日のタスク・正答率表示 |
| 過去問演習 | 35問（午前1: 10問、午前2: 10問、午後対策: 15問） |
| テストモード | 全問先に解いてまとめて採点。採点ボタン・進捗バー付き |
| カテゴリフィルター | 全問/午前1/午前2/午後/苦手/ブックマーク/難易度1〜3 |
| 問題ブックマーク | ☆/⭐でブックマーク、Fキーでトグル |
| キーボードショートカット | Enter確認/←→前後/1〜4選択/Fブックマーク |
| 合格予測 | 正答率記録から線形回帰で成長線を表示 |
| 弱点分析 | 8分野レーダーチャート・苦手問題フィルター・弱点復習モーダル |
| ポモドーロタイマー | 25/50分・カスタム時間・完了音・連続記録更新 |
| 学習カレンダー | 月単位の学習計画表示・完了マーク |
| 用語集 | 40語アコーディオン+検索 |
| オンボーディング | 初訪問時に試験日設定→プラン自動生成 |
| データ管理 | JSONエクスポート/インポート、LocalStorage保存 |
