## スコアシステム（v1.4.0）

自己紹介文の「完成度」を自動採点します。

### 採点ロジック

| 項目 | 配点 | 基準 |
|-----|------|------|
| 必須項目（氏名） | 20点 | 入力あり |
| 必須項目（職種・業種） | 20点 | 入力あり |
| オプション項目 | 最大60点 | 年数・強み・趣味・目標 各15点×4 |
| 文字数バランス | 20点 | 用途別上限の120%以内 |
| **総合** | **最大100点** | 　 |

### スコア評価ラベル

| スコア | ラベル | 意味 |
|--------|--------|------|
| 90-100 | 優秀 | 非常にバランスの取れた自己紹介文 |
| 75-89 | 良好 | 効果的な自己紹介文 |
| 60-74 | 可 | 基本的な内容は揃っている |
| 40-59 | 要改善 | 情報を追加するとより効果的 |
| 0-39 | 未完成 | 必須項目と詳細情報を追加 |

### 改善提案機能

スコアが100点未満の場合、以下の改善案を自動提示：
- 未入力のオプション項目の入力を勧める
- 文字数が上限を超えている場合は調整を勧める
- 全項目入力 & 文字数バランスが取れている場合は「完璧です！」と表示

## データ管理

- **ストレージ**: ブラウザ localStorage（キー: `self-intro-inputs`）
- **保存タイミング**: 「自己紹介文を生成する」ボタン押下時
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化
- **スコア更新**: 生成後および生成文の編集時にリアルタイム再計算

## エクスポート機能（v1.4.0）

### 印刷機能

`printResult()` ボタンで、生成した自己紹介文を印刷形式で表示します。

- window.open()で新しいウィンドウを開く
- タイトル + 用途 + 生成日時 + 本文をフォーマット
- 広告・ボタンなどは自動的に非表示（@media print）

### CSVダウンロード機能

`downloadCSV()` ボタンで、入力値と生成文を CSV形式で保存します。

- ファイル名: `自己紹介文_[Unix時刻].csv`
- 含まれる情報: 氏名・職種・経歴年数・強み・趣味・目標・用途・生成日時・生成文
- Blob API + URL.createObjectURL() で動的に CSV を生成＆ダウンロード
- 文字エンコーディング: UTF-8（Excelでの日本語表示OK）
