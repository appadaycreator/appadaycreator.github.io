# お得な買い物比較ツール【無料】無料Webツール - 技術仕様書

## 概要

**サービス名**: Shopping Comparison Tool
**バージョン**: 2.2.4（スクリーンショット解析によるUI改善版）
**更新日**: 2026-07-02
**URL**: https://appadaycreator.com/shopping-comparison-tool/

Amazon・楽天・Yahoo等の価格とポイント還元を比較して実質最安値を計算。登録不要・完全無料でご利用いただけます。

## v2.2.5 の改善内容（2026-07-02 Phase2直接実装版）

### スクリーンショット視覚分析による追加修正
- **V2 テーブル信頼性情報の最適化**: 「節約・日常費用の基礎データ」テーブルのベースフォントサイズを .95rem → 1rem に拡大。ヘッダー（th）を 1rem → 1.1rem、font-weight 700 → 800 に強化。セル padding を .5rem .75rem → .75rem 1rem に拡大。数値（strong）に font-size: 1.05rem を明示指定。行の高さを拡げて可読性を大幅向上。
- **V3 Cookie バナー遅延表示**: バナーの初期 display: none を維持。DOMContentLoaded から 3 秒後に display: flex で自動表示。localStorage キーを 'cookie-consent' → 'cookie-consent-v2' に変更して同意状態を管理。モバイルファーストビューから即座に「同意して閉じる」が見えないようにし、ページ離脱感を軽減。

## v2.2.4 の改善内容（2026-07-02）

### スクリーンショット解析による視覚的UI問題修正（Phase 2 フェーズ2実装）
- **V1 デスクトップ中段CTA強化**: hero-band の cta-primary ボタンを大型化（font-size: 20px、padding: 24px 48px、min-width: 280px、min-height: 64px）。ファーストビューから「比較を開始」のアクションを明確に見える化。
- **V2 テーブル信頼性情報の視認性向上**: 「節約・日常費用の基礎データ」テーブルのフォントサイズを .83rem → .95rem に拡大。ヘッダー行（項目・データ・出典）を font-size: 1rem、font-weight: 700 に強化。実績数値（5,000〜15,000円など）の可読性を大幅改善。
- **V3 モバイルCTA差別化強化**: Cookie バナーの初期表示を display: none に変更し、DOMContentLoaded から 3 秒後に display: flex で遅延表示。ページ離脱ボタンに見える問題を解決。z-index も 9999 → 999 に下げて背景化。「同意して閉じる」ボタンの色を #3b82f6 → #64748b グレーに変更して視認度を低下。
- **V4 各ECへのアクセスリンク明確化**: main 直下に「各ECへアクセス」セクションを追加。Amazon・楽天・Yahoo の公式サイトへの検索リンクボタン（🔍 で検索）を配置。各ECの特徴・ポイント還元率の相場が異なることをユーザーに視認させる。A8.net アフィリエイトリンク（774-775行）は保持・未変更。
- **V5 バッジ（チップス）をユーザーメリット重視に変更**: hero-band の chips 内容を「完全無料・登録不要・プライバシーファースト・スマホ対応」から「💰 年間数万円節約・⚡ 30秒で完成・✅ ポイント込み実質価格・🔄 3社一括比較」に刷新。ユーザーの実感する価値（金銭・時間・機能）を前面に押し出し、コンバージョン率向上を狙う。

## v2.2.3 の改善内容（2026-06-20）

### 視覚的UI問題修正（Phase 2 実装）
- **V1 ファーストビューCTA強化**: hero-band内の「無料で今すぐ比較する」ボタンをサイズアップ（padding: 18px 40px、font-size: 18px、min-width: 240px）。min-height: 56px で大型タッチターゲット化。訪問初期に明確な次のアクション（CTA）を視認させる。
- **V2 デスクトップ横幅活用**: hero-band-in padding-bottomを40px → 32pxに削減、.container max-width 800px → 1000pxに拡張（padding 16px → 20px）。関連サービスグリッドを固定3列から`repeat(auto-fit,minmax(280px,1fr))`のレスポンシブ対応に変更。ページレイアウトが冗長に見える問題を解決。
- **V3 チップス視認性強化**: hero-band内の.chips要素（完全無料・登録不要・プライバシーファースト・スマホ対応）をスタイル強化。border 2px solid #bfdbfe、box-shadow追加、hover時にtransform: translateY(-2px)で選択状態を可視化。タブ的UIとしての役割を強調。
- **M8 ツールチップデザイン改善**: `[style*="cursor:help"]` 要素を強化。border 1.5px、box-shadow 0 2px 8px追加、hover時に scale(1.1) & box-shadow拡大。各入力欄の「?」ツールチップを目立つUIに変更。説明テキストは既にv2.2.1で充実済み。

## v2.2.1 の改善内容（2026-06-14）

### SEO・ユーザーガイダンス強化
- **M8 入力ガイダンス大幅充実**: 各入力欄の「?」ツールチップに詳細な説明を追加。価格入力時の注意点（税抜き/税込み）、ポイント還元率の相場情報（楽天1～5%、Amazon0.5～2%）、送料の確認方法を明示。タイトル属性でも追加ヒント表示。
- **FAQPageスキーマ追加**: 5つのよくある質問にFAQPageスキーマを付与。最安値の見つけ方、ポイント還元計算、無料利用、データ保存、モバイル対応を網羅。GSC表示回数ゼロ対策。
- **メタタイトル・description強化**: 「価格比較・ポイント還元計算Webツール」と「複数EC間の送料込みコスト比較で買い物を節約」のキーワード追記。検索流入改善狙い。

## v2.2.0 の改善内容（2026-06-14）

### 新機能
- **目標設定・達成度トラッキング（M6）**: 目標節約額をlocalStorageで保存。比較結果から自動計算した節約額で達成度を表示。プログレスバーで視覚的に目標達成状況を確認可能。継続利用を促進。
- **印刷・結果エクスポート機能（M10）**: 結果をクリップボードにコピー、CSVダウンロード、ブラウザ印刷の3つのエクスポート機能を実装。ユーザーが比較結果を活用・保存しやすく。
- **入力ガイダンス充実（M8）**: 各入力欄に「?」ツールチップを配置。価格入力の注意点、ポイント還元率の相場情報、送料の初期値設定理由を明示。

## v2.1.0 の改善内容（2026-06-10）

### 新機能
- **データの自動保存・復元（M3）**: localStorage を使用した入力値の自動保存。ページ再読み込み時に前回のデータを自動復元。復元バナーで確認後、ワンクリックで前回データを復元可能。
- **詳細なガイダンス・ツールチップ（M8）**: 各入力欄に「?」アイコンを追加。ポイント還元率の相場、送料の初期値設定理由など具体的な説明をツールチップで表示。新規ユーザーが迷わずに入力できるよう改善。
- **データ保存状況の表示**: ツール上部に「入力データは自動保存されています」メッセージを表示。ローカルストレージ使用・外部送信なしを明示。

## v2.0.0 の改善内容（2026-06-03）

### 新機能
- **複数商品の同時比較**: 「商品を追加」ボタンで複数商品の入力フォームを動的に生成・管理
- **送料・税金対応**: 各店舗の送料を入力可能。税込/税抜を選択可能
- **比較結果のグラフ表示**: Chart.js で実質価格を棒グラフで可視化
- **節約額の自動計算**: 複数店舗の価格差を「最大○○円の節約」で表示

### UI/UX 改善
- モバイル最適化：3列グリッドを1列に自動切り替え
- 削除ボタン：各商品欄に「削除」ボタン追加
- デフォルト送料：楽天・Yahoo は送料 500 円でプリセット

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化
