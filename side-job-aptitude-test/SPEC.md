# 副業適性診断テスト 仕様書

## 機能概要
- 10問の質問に回答すると、ユーザーのプロフィールを解析
- 100種類の実在する副業データベースから適合度の高いTOP3を提示
- 結果はレーダーチャートで可視化
- 診断履歴はブラウザに保存され、再確認可能
- 推奨副業の「今すぐ始める」ボタンからクラウドワークスの検索ページに副業名を入力して遷移
- 言語切り替えボタン（日本語/English）

## UI
- モバイルファーストのレスポンシブデザイン
- 固定共通ヘッダーにハンバーガーメニューとPC用上部ナビを実装
- 主要コンテンツは中央寄せで表示
- ヘッダーメニューはトップ・使い方・FAQのみ表示し、文字色は白で統一

## ファイル構成
- `index.html` メインページ
- `usage.html` 使い方
- `disclaimer.html` 利用規約・免責事項
- `privacy.html` プライバシーポリシー
  (いずれもフッターリンク)
- `faq.html` よくある質問
- `jobs.js` 副業データベース

## データベース
`jobs.js` の `jobDatabase` 配列に40件の実在する副業情報を保持。必要に応じて追記可能。

## 操作フロー
1. `index.html` を開くと質問フォームが表示される。
2. 選択肢をクリックし「次の質問」ボタンまたは Enter キーで進行。
3. 10問すべて回答すると結果セクションにおすすめ副業TOP3が生成される。
4. 各副業カードの「今すぐ始める」を押すとクラウドワークス検索結果ページへ遷移。
