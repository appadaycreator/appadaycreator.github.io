# スキンケアルーティン診断【無料】自動作成ツール - 技術仕様書

## 概要

**サービス名**: Skincare Routine Builder
**バージョン**: 1.5.0
**更新日**: 2026-06-14
**URL**: https://appadaycreator.com/skincare-routine-builder/

肌タイプ・悩み・予算から最適なスキンケアの手順と商品カテゴリを診断提案。登録不要・完全無料でご利用いただけます。

### v1.5.0 新機能（2026-06-14）【M6改善】
- **目標設定・達成度トラッキング**: 月ごとの目標予算・診断回数を設定して進捗をリアルタイム表示
- **進捗サマリー表示**: 予算達成率・診断回数達成率を色付きカード表示
- **前月比較機能**: 前月との診断回数・予算の増減を自動計算・表示
- **診断実績の月別記録**: 診断実行時に予算・回数を月ごとにlocalStorageに記録
- **目標の自動保存**: 設定した目標を localStorage に保存して永続化

### v1.4.0 新機能（2026-06-10）【M4・M8改善】
- **結果のビジュアライゼーション**: Chart.js統合で朝夜のステップ数を棒グラフで表示
- **入力フォームへのツールチップ・ガイダンス**: 各入力欄に❓アイコン+説明テキストで初見ユーザーをサポート
- **予算入力の大幅UX改善**:
  - スライダー式入力（0〜30,000円）で直感的に選択可能
  - 「¥1K, ¥3K, ¥5K, ¥10K, ¥20K」のプリセットボタン実装
  - スライダーと直接入力が双方向同期
- **aria-describedby属性追加**: アクセシビリティ向上
- **予算表示の即時更新**: スライダーで選んだ金額がリアルタイム表示

### v1.3.0 新機能（2026-06-05）
- **URLパラメータ機能の完成**: `?s={肌タイプ}&c={悩み}&b={予算}` で診断結果をシェア・復元
- **診断履歴の表示機能**: 「最近の診断」セクションで過去5件を表示・復元可能
- **URLシェアボタン**: 「このURLをシェア」で現在の診断を簡単にコピー・共有
- **予算別商品ガイドの詳細化（5段階）**: 超プチプラ〜ラグジュアリーまで対応
- **ステップ表示の色彩改善**: hsl関数で朝・夜ステップの色を段階的に変化
- **アクセシビリティ強化**: 結果エリアに `aria-live="polite"` を追加

### v1.2.0 新機能（2026-06-04）
- **診断結果シェア機能**: URLパラメータで診断結果を他のユーザーと共有
- **予算別商品ガイド**: 各予算帯に応じた具体的な購入アドバイスを表示
- **診断履歴管理**: 過去5件の診断を自動保存（localStorage）
- **URLパラメータ復元**: `?s=1&c=2&b=5000` 形式で診断結果を事前復元

### v1.1.0 新機能（2026-06-03）
- **localStorage 永続化**: 入力値を自動保存・復元
- **リセット・プリント機能**: 結果エリアにボタン追加
- **予算バリデーション**: 負の数補正、100万円以上で警告
- **UX 改善**: ボタンフィードバック、ステップ表示強化、モバイル最適化
- **Affiliate 整理**: 複数ブロック統合でページ最適化

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript

## 使い方

1. 入力フォームの各項目を確認する
2. 必要な数値・条件をすべて入力する
3. 計算ボタンをクリックして結果を取得する
4. 計算結果と詳細な内訳を確認する
5. 条件を変えて複数パターンを比較する

## よくある質問（FAQ）

**Q: スキンケアルーティン診断は無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 計算結果は正確ですか？**

参考値としてご活用ください。重要な判断の際は専門家・公式機関にご確認ください。

**Q: 何回でも計算できますか？**

はい、回数制限なく何度でもご利用いただけます。条件を変えて比較にお使いください。

**Q: 入力したデータはどこかに保存されますか？**

いいえ。すべての計算はブラウザ内で処理され、データのサーバー送信は行いません。

**Q: スマートフォンで利用できますか？**

はい、PC・スマートフォン・タブレットすべてに対応しています。
