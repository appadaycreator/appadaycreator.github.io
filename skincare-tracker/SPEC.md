# 機能仕様

## v1.3.5 - 2026年6月14日（M2 Phase2 実装完了版・入力フィードバック強化）

### 改善
- **M2【満足】コア機能のUX改善（Phase2 実装完了版）**:
  - 入力フィールド成功状態に視覚的フィードバック追加：valid 状態で緑色ボーダー（#10b981）・背景淡色化
  - バリデーション失敗時に赤色ボーダー（#ef4444）を動的に適用。入力フィールドに border-red-400 クラスを付与
  - Range スライダー（肌状態・目標値）のスタイリング強化：グラデーション背景・ホバー時スケール効果・ドロップシャドウ
  - Range スライダーの thumb に hover 効果を追加：1.1x スケール・拡張シャドウで操作感向上
  - Safari/Chrome/Firefox 互換：-webkit-slider-thumb / -moz-range-thumb で統一的なスタイリング実装

## v1.3.4 - 2026年6月13日（M2 Phase2 直接実装版・リアルタイムバリデーション強化）

### 改善
- **M2【満足】コア機能のUX改善（Phase2 直接実装版）**:
  - 記録フォーム：日付・ケア種類フィールドに blur 時のリアルタイムバリデーション追加。フィールド離脱時に即座にエラーメッセージを表示
  - 製品フォーム：製品名・カテゴリフィールドに blur 時のリアルタイムバリデーション追加。入力不完全のまま次フィールドに移動する際に警告
  - 写真アップロード：ファイルサイズチェック実装。5MB 超過時にエラー通知（「写真ファイルは5MB以下にしてください」）
  - バリデーション関数を独立させて再利用性向上（validateRecordDate / validateRoutineType / validateProductName / validateProductType）

## v1.3.3 - 2026年6月13日（M2 Phase2 実装・入力UI最適化版）

### 改善
- **M2【満足】コア機能のUX改善（Phase2 直接実装版）**:
  - 記録・製品フォーム全入力要素（input/select/textarea）に focus 時のビジュアルフィードバック追加：border-color を purple-500 に、ring-2 ring-purple-200 で focus時の視認性大幅向上
  - input/select のpadding を p-2 から p-3 に拡大、タッチ操作でのヒット領域を拡張（モバイルUX向上）
  - placeholder テキストを簡潔化・実践的に（例：「SK-II朝晩、睡眠7時間」など最小限の記入例に統一）
  - memo/product-memo のカウンター表示に「文字」を追記（「0 / 500 文字」→視認性向上）
  - textarea に resize-none を追加してレイアウト崩れを防止

## v1.3.2 - 2026年6月13日（M2・M8強化版）

### 改善
- **M2【満足】コア機能のUX改善（入力UI・バリデーション強化）**:
  - 記録フォーム全項目に入力ガイダンスアイコン(?)を追加し、ホバーでツールチップ表示
  - バリデーションエラーメッセージのスタイル強化（赤背景・ボーダー・padding追加で視認性向上）
  - 入力フォームの説明文を emoji 記号で視覚的に強調（📌日付、🌅朝、🌙夜、📊評価、📸写真、💡メモ）
  - textarea のメモ欄に入力例をリアルタイム表示

- **M8【満足】ツールチップ・入力ガイダンス・ヘルプテキスト充実**:
  - 全入力フォーム（日付・ケア種類・肌状態・写真・メモ・製品名・カテゴリ）のラベル横に help icon (?) を配置
  - 各フォーム要素に aria-describedby を追加し、支援技術との連携強化
  - 肌状態スライダーに「1=乾燥・刺激強い、5=普通、10=理想的」を視覚的に明示
  - 製品カテゴリ選択肢に使用順序（①～⑤）を番号付けして表示
  - メモ欄・製品メモに具体的な入力例（『SK-IIエッセンス朝・夜、睡眠7h』など）を表示
  - placeholder テキストを改善し、実際の記入例を示唆

## v1.3.1 - 2026年6月13日（改善版）

### 改善
- **M2【満足】コア機能のUX改善（強化版）**:
  - 記録フォーム（日付・ケア種類）に入力バリデーション実装。必須項目を赤い「*必須」マークで明示
  - フォーム送信時に必須項目の空白チェック → エラーメッセージ表示（日付未入力 / ケア種類未選択）
  - 製品フォーム（製品名・カテゴリ）にも同じバリデーション適用
  - select要素に「-- 選択してください --」プレースホルダを追加し、デフォルト状態を明確化
  - textarea のメモ欄に文字数カウンター追加（リアルタイム表示）
  - input・select・textarea に focus-visible クラスを追加してフォーカス状態を可視化

- **M8【満足】ツールチップ・入力ガイダンス充実（強化版）**:
  - 全入力フォーム要素に詳細な title 属性を追加（ホバー時ツールチップ表示）
  - 記録フォーム各欄の小さい説明文を拡充：入力例・注意点・なぜその情報が重要かを記載
  - 製品フォーム：製品名に「メーカー名+製品名」の例を明示、カテゴリに「使用順序」の説明を追加
  - スライダー（肌状態・目標値）に詳細な title 属性 → スライダー値の意味（1=乾燥、10=理想的）を明確化
  - 各フォーム要素に aria-label を追加しアクセシビリティ向上
  - オプション項目を「(オプション)」として明示

## v1.1 - 2026年5月31日

### バグ修正
- **PWA theme-color統一**: lp.html / how-to-use.html / offline.html に `#8B5CF6` を追加
- **XSS脆弱性修正**: 製品リスト表示を innerHTML から DOM API (textContent) に変更

### 機能追加
- **分析画面の記録履歴リスト**: updateAnalysis に履歴描画ロジックを追加。削除ボタンも実装
- **肌状態入力のスライダー化**: input[type=range] + リアルタイム数値表示（例: "7 / 10"）

## データ管理

- 認証不要のローカルモード。データは localStorage に保存（skincare_records / skincare_products / skincare_photos）
- 端末・ブラウザをまたいだデータ引き継ぎ不可
