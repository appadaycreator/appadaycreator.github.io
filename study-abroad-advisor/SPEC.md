# 語学留学費用計算【無料】国別・期間別トータルコスト比較 - 技術仕様書

## 概要

**サービス名**: Study Abroad Advisor
**バージョン**: 1.3.0
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/study-abroad-advisor/

国・期間・学校グレードから語学留学の費用を無料シミュレーション。フィリピン・カナダ・アメリカ比較。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## バージョン 1.3.0 での改善点（2026-06-10）

### P1 実装内容
1. **M3: localStorage による入力履歴・結果保存機能** → 診断完了時に answers と計算結果を自動保存。ページリロード後も前回データが復元される。履歴一覧で過去10件まで遡れる。
2. **M4: Chart.js による費用内訳グラフ化** → 結果画面の「費用内訳」セクションにドーナッツグラフを追加。学費・滞在費・航空券の比率を視覚化。

### P2 実装内容
3. **M10: 印刷・CSVエクスポート・コピー機能** → 結果画面に「🖨️ 印刷」「📥 CSVで保存」「📋 結果をコピー」ボタンを追加。プリントフレンドリー CSS により印刷時は不要要素を非表示。

### 実装詳細
- `saveToHistory()` 関数：showResult() 完了時に answers と計算結果を localStorage に保存（KEY: 'study-abroad-history', 最大10件）
- `loadFromHistory(index)` 関数：履歴から選択したアイテムを復元し、入力フォームを復元して再計算
- Chart.js ドーナッツグラフ：tuitionTotal, livingTotal, flightTotal を3色（緑系）で可視化
- `downloadResultAsCSV()` 関数：診断結果を CSV 形式でダウンロード（ファイル名: study-abroad-advisor-YYYYMMDD.csv）
- `copyResultToClipboard()` 関数：診断結果（国・期間・費用・レベル）をテキストでクリップボード コピー
- CSS `no-print` クラス：印刷時にボタン・広告を非表示
