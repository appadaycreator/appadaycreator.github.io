使い方｜4つのステップで簡単

このアプリは、集中・記録・振り返りをひとつに。今日からすぐ始められます。

1

科目とモードを選ぶ

学習する科目と、集中25分・休憩5分などのモードを選択。カスタム時間も設定できます。

2

タイマーで集中する

スタートを押して集中。一時停止・再開も自由で、ポモドーロで効率よく進められます。

3

記録が自動で見える化

終了すると科目別にログを自動記録。グラフと達成率で積み重ねが一目でわかります。

4

振り返って共有・書き出し

『もう一度』で連続学習。CSV / JSON で書き出し、SNSへのシェアで継続を後押しします。

主な特長

集中するための仕掛けから、続けるための記録・共有まで。学習に必要な機能をひとつにまとめました。

集中音BGM（波音・雨音）

波音・雨音などの環境音を再生し、α波を引き出して深い集中をサポート。雑音をやわらげ学習に没頭できます。

試験日カウントダウン

試験日までの残り日数を表示。受験・資格までのカウントで、毎日の学習モチベーションを強化します。

タスク管理との統合

学習内容の詳細メモや優先度を管理。「何を・どこまで」やるかを整理して計画的に進められます。

ワンクリックシェア

Twitter・LINE・Google Classroom へワンクリックで共有。記録を仲間と共有して継続を後押しします。

複数デバイス同期

クラウド保存オプションで複数デバイス間の学習データを同期。PC・スマホどちらからでも続きを記録できます。

ローカル保存で安心

記録は端末内のローカルストレージに保存し、外部サーバーへ送信しません。プライバシー第一で使えます。

選ばれる理由

シンプルだから、毎日続く。登録なしのブラウザだけで、集中から振り返りまで完結します。

登録不要＆ブラウザですぐ使える

会員登録もインストールも不要。モバイル・PC共通で、ブラウザを開けばその場で学習を開始できます。

ローカル保存でプライバシー第一

記録はお使いの端末のローカルストレージに保存。外部サーバーへ送信しないので、安心して使えます。

科目別・日別・週別の3段階グラフ分析

3段階のグラフで学習量を多角的に分析。バランスや伸びを把握し、受験対策の戦略づくりに役立ちます。

さまざまなシーンで活躍

受験から社会人の学び直しまで、幅広く活用できます。

受験勉強

科目ごとの学習バランスを把握し、苦手科目に時間を配分。毎日の積み上げを可視化します。

資格取得

仕事と両立しながら学習時間を記録。目標達成率で進捗を管理し、合格までモチベーションを維持。

社会人の学び直し

スキマ時間の学習も逃さず記録。CSV / JSON で書き出して、長期の振り返りにも活用できます。

よくある質問

スマートフォンで使えますか？
はい、スマートフォン・タブレット・PCすべてでご利用いただけます。インストール不要でブラウザからすぐに使えます。
途中で一時停止できますか？
はい、一時停止ボタンで止めて、再開することができます。
カスタムの時間を設定できますか？
はい、プリセット以外に任意の時間（1〜120分）を直接入力して設定できます。
時間終了時に通知はありますか？
画面上でアラート表示します。ブラウザの設定によって音声が出る場合があります。
学習記録はどこに保存されますか？
記録はお使いのブラウザのローカルストレージに保存され、外部サーバーには送信されません。CSV / JSON で書き出してバックアップできます。
