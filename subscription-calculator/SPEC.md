# 仕様書

本アプリは、サブスクリプションサービスの料金を計算し、解約候補を提案するWebアプリです。
GitHub Pagesでホスティングされ、データはブラウザのローカルストレージに保存されます。

## 画面一覧
- トップページ: 計算機本体（サービス名から料金プランを選択可能）
- 使い方: 操作手順を説明
- 利用規約: サービス利用条件を表示
- 免責事項: 情報の正確性や責任範囲を記載
- プライバシーポリシー: 収集データと利用目的を明示
- よくある質問: 機能や利用方法のQ&A
- お問い合わせ: フォームから連絡
- 副業リンク集: 副業名とリンク一覧を掲載

すべての画面下部(フッター)からこれらのページへ遷移可能です。
リンクは相対パスで記述し、GitHub Pages上でも404にならないようにします。

## データ利用
入力したサブスクリプション情報はブラウザ内にのみ保存され、外部に送信されません。
アクセス解析にはGoogle Analyticsを利用しています。
