# テニススコア記録アプリ【無料】記録・管理アプリ - 技術仕様書

## 概要

**サービス名**: Tennis Score Tracker
**バージョン**: 1.5.0
**更新日**: 2026-06-13
**URL**: https://appadaycreator.com/tennis-score-tracker/

テニスの試合スコアをリアルタイム記録。セット・ゲーム・ポイントを管理。グラフ・統計・データ出力機能を搭載。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 旧機能（v1.4.0）

### 1. Chart.js によるグラフ・スコア・ゲージの可視化 ⭐ NEW
- グラフ表示を SVG から **Chart.js** に移行し、より直感的で美しいグラフを実現
- **累計勝率グラフ**: 直近30試合の勝敗推移を折れ線グラフで表示
- **月別勝敗グラフ**: 最近12ヶ月の勝敗数を積み上げ棒グラフで表示（緑＝勝、グレー＝敗）
- **曜日別勝率グラフ**: 曜日ごとの勝率を棒グラフで表示（50%以上は緑、未満はグレー）
- リアルタイムツールチップで詳細データを表示

### 2. 複数パターン比較・シミュレーション機能 ⭐ NEW
- 「🔮 シミュレーション」ボタンで新しいシミュレーションモーダルを開く
- 3つのパターン（A・B・C）を並べて同時入力可能
- 各パターンの「もし〜だったら」シミュレーション結果を即座に計算・表示
  - 仮想的な累計勝利数・敗北数・勝率を計算
  - パターン追加時に全体の勝率がどう変わるかを一目で比較
- 入力を変更すると即座に結果が更新される（リアルタイム再計算）
- 選択したパターンを「選択したパターンを記録」で一括記録可能

## 旧機能（v1.2.0）

### 1. 対戦相手の得意度ビジュアル表示 ⭐ NEW
- 対戦相手ごとの勝率を視覚的なバー（プログレスバー）で表示
- 得意度マーク：💚（60%以上）/ ➖（30～60%） / ⚠️（30%未満）を自動表示
- 一目で得意・不得意な相手を判定可能

### 2. 試合後の振り返りテンプレート ⭐ NEW
- 記録直後に振り返りモーダルを自動表示
- Good（良かった点）・Bad（改善点）・Next（次へのアクション）を記録
- 既存記録に対しても「振り返りを追加」リンクで後から追加可能
- 振り返りデータはエントリー内に保存・表示

### 3. グラフ表示の複数方式 ⭐ NEW
- **累計勝率グラフ**: 直近30試合の勝敗推移（従来のグラフ）
- **月別勝敗グラフ**: 最近12ヶ月の勝敗数を棒グラフで表示（緑＝勝数、灰色＝敗数）
- **曜日別勝率グラフ**: 曜日ごとの勝率を表示（50%以上は緑、未満は灰色）
- 「📊 グラフ表示」後にタブで切り替え可能

### 4. モバイルUX改善 ⭐ NEW
- フォーム入力時にEnterキーで次フィールドへ自動フォーカス移動
- セット数が「0-0」の場合に警告表示（誤入力防止）
- 削除ボタンのサイズを拡大（タッチしやすく）
- 記録追加後に記録一覧に自動スクロール

### 5. CSV出力の拡張 ⭐ UPDATED
- 従来の「日付、対戦相手、セット数、結果、スコア詳細、メモ」に加え
- 「Good・Bad・Next」の振り返りデータをカラムとして追加

## 旧機能（v1.1.0）

### 1. グラフ表示（勝敗推移可視化）
- 直近30試合の勝敗推移を折れ線グラフで表示
- 試合ごとの累計勝率を視覚的に確認可能
- 「📊 グラフ表示」ボタンで即座に表示

### 2. 削除確認ダイアログ
- 記録削除時に確認ダイアログを表示
- 誤削除を防止し、ユーザー信頼を向上

### 3. データエクスポート（CSV）
- 全記録をCSV形式でダウンロード
- 日付、対戦相手、セット数、スコア詳細、メモを含む
- 「💾 CSV出力」ボタンで実行

### 4. 対戦相手別統計
- 対戦相手ごとの勝敗を一覧表示
- 対戦回数・勝数・敗数・勝率を表示
- 「👥 対戦相手別」ボタンで表示

## 使い方

1. 記録したい項目を入力フォームに入力する
2. 「記録する」ボタンをクリックする
3. 記録一覧で過去のデータを確認できます
4. グラフ・統計・CSV出力ボタンで詳細分析が可能
5. 継続的に記録して変化を追跡する

## よくある質問（FAQ）

**Q: テニススコア記録アプリは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。
