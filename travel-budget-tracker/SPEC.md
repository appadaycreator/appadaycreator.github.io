# 旅行予算管理ツール【無料】記録・管理アプリ - 技術仕様書

## 概要

**サービス名**: Travel Budget Tracker
**バージョン**: 1.2.2
**更新日**: 2026-06-20
**URL**: https://appadaycreator.com/travel-budget-tracker/

旅行の予算配分（交通・宿泊・食事・観光）を管理。実績vs計画を比較。登録不要・完全無料でご利用いただけます。

### v1.2.2 の更新内容（UI改善 + ガイダンス充実）
- **ヘッダーナビゲーション改善（V2）**：背景をグラデーション（青→紫）に変更・文字色を白に統一・コントラスト比向上・フォントサイズ拡大（`.78rem` → `.9rem`）
- **ナビゲーション視認性向上（V3）**：フォントサイズ拡大によりメインナビゲーション（コラム・ホーム）の視認性向上
- **Cookie同意バナー改善（V1）**：「拒否」ボタンを追加・同意と拒否の選択肢を並列表示・視認性向上
- **ツールチップ・入力ガイダンス充実（M8）**：各入力フィールドに詳細な title 属性を追加・マウスホバーで具体例とコツを表示・カテゴリ・内容・金額・予算フィールドに？マークで視覚的に気付きやすく

### v1.2.1 の更新内容（M8実装）
- **ツールチップ・入力ガイダンス充実**：各入力フィールドに title 属性を追加し、マウスホバーで詳細ガイダンスを表示
- **aria-label 追加**：アクセシビリティ向上・音声読み上げ対応
- **必須項目の表示**：カテゴリ・内容・金額を必須項目として視認性向上
- **参考情報の提供**：予算フィールドに国内旅行の平均日当(3,900円)を参考値として表示

### v1.2.0 の更新内容（M4・M10実装）
- **グラフ表示機能（Chart.js）**：カテゴリ別支出をドーナッツグラフで可視化・リアルタイム更新
- **印刷機能**：ブラウザ標準印刷でサマリーのみを出力（レイアウト最適化）
- **CSVエクスポート機能**：記録をCSV形式でダウンロード（Excel/Googleスプレッドシート対応）
- **テキストコピー機能**：支出サマリーをクリップボードにコピー（SNS/メール共有対応）
- **エクスポートボタン**：サマリーセクションに専用ボタン群（無効化表示対応）

### v1.1.0 の更新内容
- グラフ表示機能（Chart.js）：カテゴリ別支出を円グラフで可視化
- CSV エクスポート機能：記録を外部ツール（Excel等）で分析可能に

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 記録したい項目を入力フォームに入力する
2. 「追加」または「記録」ボタンをクリックする
3. 記録一覧で過去のデータを確認する
4. グラフや集計で傾向を把握する
5. 継続的に記録して変化を追跡する

## よくある質問（FAQ）

**Q: 旅行予算管理ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。
