# 1日の水分摂取量計算ツール【無料】簡単・無料計算 - 技術仕様書

## 概要

**サービス名**: Water Intake Calculator
**バージョン**: 1.3.0（2026-06-10 UX改善版）
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/water-intake-calculator/

体重・運動量・気温から1日に必要な水分摂取量を計算。脱水防止アドバイス付き。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 入力フォームの各項目を確認する
2. 必要な数値・条件をすべて入力する（リアルタイム自動計算）
3. 計算結果・脱水リスク評価・時間帯別スケジュール・グラフを確認する
4. 条件を変えて複数パターンを比較する

## v1.4.0 新機能（2026-06-13）

- **M4: グラフ・スコア・ゲージで結果を可視化**: 
  - 🎯 水分管理スコア（0-100点）: 摂取量が適正範囲（1500-3000ml）かで採点・気温補正条件修正（>0で判定）
  - 📊 内訳グラフの強化: 基本量・運動補正・気温補正の内訳を横棒グラフで%と ml両方表示
  - 📈 摂取進捗ゲージ: Canvas使用した円形ゲージで推奨量達成度と健康範囲内か視覚化

## v1.3.0 新機能（2026-06-10）

- **M8: ツールチップ・入力ガイダンス充実**: 各入力項目（体重・運動量・気温）に?アイコン付きツールチップを実装。入力例や範囲を視覚的に表示
- **M10: 印刷・結果エクスポート機能**: 計算結果をコピー（📋）・テキストファイルダウンロード（💾）・印刷（🖨️）できるボタングループを追加
- **ヘルプテキスト強化**: 各入力欄に「※ 20～200kgの範囲で入力」等の補足テキストを追加

## v1.2.0 新機能（2026-06-05）

- **リアルタイム自動計算**: 入力値を変更すると即座に結果が更新される（キーアップ・選択変更時）
- **脱水リスク評価**: 計算結果に色分けバッジで4段階リスク表示（低/通常/中程度/高）
- **時間帯別スケジュール**: 朝・昼・夕方・夜に分割した飲水スケジュール提案
- **内訳グラフ**: 基本量・運動補正・気温補正の割合を横棒グラフで可視化

## よくある質問（FAQ）

**Q: 1日の水分摂取量計算ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 計算結果は正確ですか？**

参考値としてご活用ください。重要な判断の際は専門家・公式機関にご確認ください。

**Q: 何回でも計算できますか？**

はい、回数制限なく何度でもご利用いただけます。条件を変えて比較にお使いください。

**Q: 入力したデータはどこかに保存されますか？**

いいえ。すべての計算はブラウザ内で処理され、データのサーバー送信は行いません。

**Q: スマートフォンで利用できますか？**

はい、PC・スマートフォン・タブレットすべてに対応しています。
