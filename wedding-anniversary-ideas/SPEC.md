# 結婚記念日アイデアジェネレーター【無料】無料Webツール - 技術仕様書

## 概要

**サービス名**: Wedding Anniversary Ideas
**バージョン**: 1.1.0（M3/M4/M10 実装）
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/wedding-anniversary-ideas/

記念日の年数・予算・好みから最適なお祝い方法とプレゼントアイデアを提案。登録不要・完全無料でご利用いただけます。

## 新機能（v1.1.0）

### M3: データ保存・履歴機能（localStorage）
- 入力値（年数・予算・スタイル）をブラウザの localStorage に自動保存
- ページリロード時に前回の入力値を自動復元
- 「前回の結果を復元」ボタンで直近の計算結果をすぐに再表示
- ユーザーのリピート利用を促進

### M4: 結果の可視化（Chart.js版・強化版）
- **総合スコア表示**: 3項目の平均スコアを大きく表示（0〜100pt）
- **スター評価**: スコアに基づく★☆評価（5段階）を視覚化
- **レーダーチャート**: 周年度合い・予算充実度・体験スコアを多角形グラフで視覚化
- **スコア表示**: 0-100ptスケールで各項目を定量化
  - 周年度合い: 周年数 / 50年 × 100
  - 予算充実度: 低(33pt) → 中(66pt) → 高(100pt)
  - 体験スコア: おうち(40pt) → 外食(70pt) → 旅行(85pt)
- **進捗ゲージ**: 25周年なら50%、50周年なら100%を表示
- **予算レベルバー**: 低い予算（1本）→ 中程度（2本）→ 高い予算（3本）を視覚化

### M10: 印刷・エクスポート機能
- 🖨 **印刷ボタン**: window.print()で結果を印刷対応の形式で表示
- 📋 **コピーボタン**: 結果をクリップボード経由で SNS・メール・ドキュメントに貼り付け可能

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元（M3で実装）
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する
