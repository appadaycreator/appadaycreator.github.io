# 結婚式ゲストリスト管理ツール【無料】記録・管理アプリ - 技術仕様書

## 概要

**サービス名**: Wedding Guest List Manager
**バージョン**: 1.3.1
**更新日**: 2026-06-20
**URL**: https://appadaycreator.com/wedding-guest-list-manager/

招待ゲストの管理・出席確認・席次計画を一括管理。費用見積もりとも連携。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 改善履歴（v1.2.0 新機能）

### M6: 目標設定・達成度トラッキング
- **目標ゲスト数の設定**: 招待予定ゲスト数を目標値として設定
- **達成度ゲージ**: 出席確認数 / 目標値をプログレスバーで可視化
  - リアルタイム更新: ゲスト追加・削除・ステータス変更時に自動更新
  - 達成率表示: パーセンテージで出席確認の進捗を表示
  - 未確認数表示: 残り何名の確認が必要かを表示
- **前回との比較**: 前回の目標値との差分を表示（目標人数の変更を追跡）
- **データ永続化**: localStorage で目標値と履歴を保存・復元

## 改善履歴（v1.1.0 新機能）

### M4: グラフ・スコア・ゲージで結果を可視化
- Chart.js 4.4.0 導入（CDN）
- 2つの直感的なグラフを実装：
  - **出欠状況グラフ**（ドーナツ型）: 出席・未確認・欠席の比率を視覚化
  - **新郎/新婦側比較グラフ**（横棒グラフ）: ゲスト数を比較表示
- データ追加・変更時に自動更新
- レスポンシブ対応（モバイル・PC両対応）

### M8: ツールチップ・入力ガイダンス・ヘルプテキスト充実
- 全6フォーム項目に「？」ツールチップボタンを追加
  - お名前: 招待状に記載されているお名前（漢字・フルネーム推奨）
  - 続柄・関係: 友人・職場・親族など、どのような関係か
  - テーブル番号: 披露宴当日に座るテーブルの番号
  - 新郎/新婦側: 新郎側（職場・友人など）または新婦側（親族など）
  - 出欠状況: 返信状況（未確認・出席・欠席）
  - メモ: アレルギー・食事制限・席替え希望など特記事項
- 各フィールドに aria-label を追加（アクセシビリティ向上）
- placeholder 文言を詳細化・分かりやすくしたプリセットテキスト提案

### M10: 印刷・結果エクスポート機能
- **📄 印刷ボタン**: window.print() で印刷プレビュー表示（既存 @media print CSS 活用）
- **📥 CSVダウンロード**: ゲスト一覧を CSV 形式でエクスポート
  - ファイル名: `guest-list-YYYY-MM-DD.csv`
  - 列: 名前 / 続柄・関係 / テーブル / 新郎/新婦側 / 出欠状況 / メモ / 追加日
- **📋 テキストコピー**: ゲスト一覧をテキスト形式でクリップボード コピー
  - 形式: `名前 (続柄) - 出欠状況 / メモ`（1行1人）

## 使い方

1. 記録したい項目を入力フォームに入力する
2. 「追加」または「記録」ボタンをクリックする
3. 記録一覧で過去のデータを確認する

## よくある質問（FAQ）

**Q: 結婚式ゲストリスト管理ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 記録データはどこに保存されますか？**

ご使用のブラウザのローカルストレージに保存されます。他のデバイスとは共有されません。

**Q: ブラウザを閉じても記録は残りますか？**

はい、同じブラウザを使う限りデータは保持されます。

**Q: 記録を削除するにはどうすればいいですか？**

リセットボタン、またはブラウザのサイトデータをクリアすることで削除できます。

**Q: スマートフォンで利用できますか？**

はい、スマートフォン・タブレット・PCすべてでご利用いただけます。
