# ワークライフバランス採点ツール【無料】無料Webツール - 技術仕様書

## 概要

**サービス名**: Work Life Balance Scorer
**バージョン**: 1.3.0
**更新日**: 2026-06-14
**URL**: https://appadaycreator.com/work-life-balance-scorer/

勤務時間・休暇取得・趣味時間からワークライフバランスを100点満点で採点。改善提案付き。登録不要・完全無料でご利用いただけます。

## 主な機能

### M3: データ保存・履歴機能
- **localStorage**: 最大20件の計算履歴を自動保存
- **自動復元**: ページ読み込み時に前回の入力値を自動復元
- **履歴表示**: 過去の計算結果を一覧表示、クリックで復元可能
- **クリア機能**: 履歴全削除オプション

### M4: グラフ・スコア可視化
- **Chart.js**: 結果をBar Chartで可視化
- **内訳表示**: 実働・残業・自由時間・有給・睡眠の各評価と総合スコアを表示
- **リアルタイム**: 計算と同時にグラフを自動更新

### M7: 比較・シミュレーション機能
- **シナリオ保存**: 現在の入力値を「シナリオ名」付きで保存
- **シナリオ管理**: 保存したシナリオの一覧表示・読込・削除が可能
- **比較機能**: 2つのシナリオを並べて比較
- **差分表示**: スコア及び各項目の差分を可視化（📈📉で増減を表示）

### M8: ツールチップ・入力ガイダンス充実
- **Help Icons**: 各入力ラベル横に「?」アイコンを配置
- **ホバーツールチップ**: マウスホバー時に該当項目の簡潔な説明を表示
- **Title属性**: 各入力フィールドにtitle属性で詳細ガイダンスを追加
- **ベストプラクティス**: 日本平均値・推奨値・ラインなどの基準を記載

### M10: 印刷・エクスポート機能
- **印刷**: window.print()でプリントフレンドリーなレイアウトで出力
- **CSV出力**: 入力値とスコアをCSV形式でダウンロード
- **クリップボードコピー**: 結果をテキストコピー

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## よくある質問（FAQ）

**Q: ワークライフバランス採点ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: 結果を保存・共有できますか？**

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
