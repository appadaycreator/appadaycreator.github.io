# ヨガルーティン作成ツール【無料】自動作成ツール - 技術仕様書

## 概要

**サービス名**: Yoga Routine Maker
**バージョン**: 1.2.0
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/yoga-routine-maker/

目的・レベル・時間から最適なヨガポーズのルーティンを自動生成。登録不要・完全無料でご利用いただけます。

## 更新履歴（v1.1 → v1.2）

### P1: localStorage による履歴保存機能（M3）
- **機能**: 作成したルーティンを最大10個まで自動保存
- **実装**: 
  - `RoutineHistory` クラスで履歴管理
  - ルーティン作成時に自動保存
  - クリックで前回ルーティンを復元可能
  - 「すべて削除」ボタンで履歴をクリア
- **効果**: リピート利用率向上、ユーザーの手間削減

### P2: Chart.js による結果可視化（M4）
- **機能**: ルーティンの時間分布をグラフで表示
- **実装**:
  - 各ポーズのホールド時間を横棒グラフで可視化
  - 統計情報（総時間・平均・最長・最短）を表示
  - レスポンシブグラフ表示
- **効果**: ユーザーがルーティンバランスを直感的に理解

### P3: 印刷・エクスポート機能（M10）
- **機能1 - 印刷機能**:
  - 別ウィンドウで印刷プレビューを表示
  - 目的・レベル・時間・作成日時を記載
  - プリンターで直接出力可能
- **機能2 - CSVダウンロード**:
  - ルーティンをCSV形式で自動生成
  - Excelで開いて修正・共有可能
  - タイムスタンプ付きのファイル名
- **効果**: オフライン利用、共有・印刷での利便性向上

## 更新履歴（v1.0 → v1.1）

### P1: ルーティン生成アルゴリズムの改善
- **改善**: 目標時間をより正確に守る（±5分以内）
- **実装**: 各ポーズの使用回数制限（最大2回まで）、2段階シャッフル
- **効果**: ユーザーが指定した時間に合わせたルーティンが生成される

### P2: タイマーUXの改善
- **改善1**: ポーズ完了時に音声通知（Web Audio API）
- **改善2**: 次ポーズのプレビュー表示
- **改善3**: タイマーセクションを sticky 表示（スクロール時も常時表示）
- **効果**: タイマー中のユーザー体験が大幅に向上、次のポーズを事前に確認可能

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## よくある質問（FAQ）

**Q: ヨガルーティン作成ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: 結果を保存・共有できますか？**

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
